## *페이퍼클립: AI 노동을 위한 오픈소스 인간 제어 플레인* 핵심 요약

- 영상은 Paperclip의 창립자 Dotta가 Paperclip을 어떻게 활용하는지, 설치 · 조직 구성 · 고급 활용법까지 상세히 시연함
- Paperclip은 오픈소스 ‘에이전트 오케스트레이터’로, AI 에이전트로 구성된 조직도(Org Chart)를 구축하고 실제 업무를 관리·자동화할 수 있게 해줌
- 사용자는 원하는 AI 모델(예: Gemini, Claude, Codex, Hermes, Pi 등)을 선택하여 에이전트로 투입할 수 있음
- ‘제로 휴먼 컴퍼니’를 표방하며, 실제 기업의 채용·프로젝트 배정·스킬 관리·피드백·품질관리 등 현실적인 업무 흐름을 AI로 구현함
- GitHub 별 갯수가 빠르게 증가(40,000 → 50,000 이상)하며 활발한 커뮤니티 성장과 기여를 기록함
- Paperclip의 조직 구성, 프로젝트·업무 배정, 에이전트 간 커뮤니케이션 및 피드백 과정이 상세하게 데모로 소개됨
- 작업 흐름 지원(QA, 승인, 반복 피드백 등), 일정/루틴 자동화, 히스토리·비용관리 등 고차원적 에이전트 조직 관리 기능 제공
- 비개발자도 마케팅, 영업, 재무 등 다양한 업무에 적용 가능하도록 설계되어 있으나, 코딩 등 기술 지식이 있으면 더욱 효과적으로 활용 가능
- 계속해서 기능 업그레이드 중(CEO 채팅, 아티팩트 및 배포 관리, 팀 협업·다중 사용자, 클라우드 환경/에이전트 샌드박스 등)
- Paperclip은 ‘AI가 인간의 일을 대체하는 것이 아니라, 인간이 수많은 AI 노동력을 직접 관리·제어’하는 플랫폼임을 강조함

## 세부 요약 - 주제별 정리

### Paperclip은 AI 에이전트 기반 조직을 실제 기업처럼 시각화하고 관리할 수 있도록 설계됨

- Paperclip의 슬로건은 “제로 휴먼 컴퍼니를 위한 오픈소스 오케스트레이션”
- 사용자는 자신만의 기업 조직도를 만들고, CEO, CTO 등 주요 직책별로 AI 에이전트를 배치할 수 있음
- 각 에이전트는 OpenAI/Anthropic 등 다양한 AI 모델로 선택 가능
- 예시로 Paperclip 팀 자체를 Paperclip으로 관리하는 실제 조직도를 보여주며 설명
    - CEO, CTO, 주요 코더(Codex(coder), Claude(coder)), CMO, 마케팅팀, 콘텐츠 전략, 비디오 라이터 등 세분화
- 에이전트별 역할 분담이 실제 업무처럼 이루어지며, 각 단계별로 태스크가 할당됨

### 실제 실무 사례 데모: AI 에이전트 사용으로 영상 제작을 더욱 빠르고 자동화할 수 있음을 시연

- Paperclip의 GitHub 별 4만 돌파 기념 영상을 제작하는 예시로 데모 진행
- CEO 에이전트에게 ‘비디오 라이터를 고용’ 및 ‘Remotion 비디오 작성’ 지시
- Remotion(React 기반 오픈소스 영상 제작 툴)의 베스트 프랙티스 스킬 설치
- 비디오 라이터 에이전트에 브랜딩 가이드, 통계 대시보드 등 관련 자료를 맥락(컨텍스트)으로 제공
- 사용자가 원하는 컷 길이, 애니메이션 등 상세 피드백을 주고, 에이전트가 계획-초안-수정 순으로 작업
- 약 5분 만에 실제 대표 영상(애니메이션, 브랜딩, 실제 차트가 포함된 결과물) 완성
- 컨텍스트 확장 및 반복 피드백으로 ‘Paperclip 전용 Remotion 스킬’도 학습 및 고도화
- 기존 Claude Code 등에서는 여러 수작업(자료 탐색, 프롬프트 조합 등)이 필요하나, Paperclip은 모든 데이터와 가이드를 시스템 내에서 일관성 있게 관리

### 조직 내 QA, 승인 등 다중 스텝 작업흐름(Workflow)을 AI 중심으로 간단히 구현할 수 있음

- QA 에이전트가 ‘Agent Browser Skill’ 등으로 자동화된 웹 테스트, 폼 작성, 버튼 클릭 등 수행 가능
- 태스크별로 리뷰어(품질보증) 및 승인권자(결재자) 등 역할 분담 가능
- 예를 들어, 코딩 태스크가 마무리되면 QA 에이전트가 자동 리뷰 후 담당자와 반복적 피드백 가능
- 실제 현업에서 Claude Code 등 기존 툴은 각기 다른 워크플로우와 모델마다 작업 방식이 달라 혼란이 큰 문제를 Paperclip이 해결
- 다양한 AI 모델, 벤더 중립적으로 하나의 조직 내에서 필요한 에이전트 조합 및 워크플로우 생성 가능

### 반복 작업 및 일정 관리(루틴)도 루틴(Routines)으로 구조화하여 자동화 할 수 있음

- 예) ‘매일 마스터 브랜치 머지 내역으로 Discord 메시지 생성’, ‘릴리즈 체인지로그 작성’ 등의 반복 업무를 루틴으로 등록 가능
- 루틴 실행 시 입력 변수(예: 특정 브랜치명 지정)도 템플릿 형태로 지정
- 루틴과 스킬 셋(예: PR 검토용 greptile skill 등)을 연동하여 복합 자동화 가능
- 커뮤니티용 코드 리뷰 자동화 예시: PR 제출시 grep loop 기능이 동작하여 변경 사항을 검토
- 루틴·스킬·작업 단위 구조를 통해 일회성 프롬프트 복사/붙여넣기 대신 재활용 및 효율적 관리 가능

### Paperclip은 비개발자도 쉽게 활용 가능하며, 엔지니어에 국한되지 않는 다양한 업무영역 지원

- Paperclip은 단순한 ‘코딩 도구’가 아니라 기업 운영 전체를 지원하는 ‘비즈니스 오케스트레이터’임을 강조
- 마케팅, 영업, 재무관리 등 비개발자 역할의 업무에도 적용할 수 있도록 설계됨
- 단, 개발자라면 GitHub, Cursor와 수준높게 연동 가능하나, 기본적으로 비전문가도 쉽게 사용할 수 있도록 인터페이스 지원

### Paperclip에서 북마크 또는 외부 리포트 등을 활용한 확장 및 향후 액션 중심 기능 방향 소개

- 북마크한 아이디어나 외부 리포트도 가져와 Paperclip 내에서 자동 이슈 생성, 플랜 작성 등 후속 조치 제안(향후 지원 예정)
- 현재 Paperclip은 단순 보고서만 제공하지만, 앞으로는 버튼 클릭으로 바로 추가 작업(이슈, 플랜, 기능 통합 등)이 가능해질 예정

### Paperclip의 신규 조직(Company) 생성, 초기 세팅 및 템플릿 운용 방식 안내

- 새 회사를 만들 때, 프로젝트 목표·설명·조직명 등 입력하여 AI 에이전트 조직도를 자유롭게 설계
- 에이전트 추가 시 다양한 모델 선택 가능(Gemini, Claude, Codex, Hermes, Pi, Cursor 등)
- 초기에 최소한의 에이전트(예: CEO, CTO 등)만 편성해 점진적으로 조직을 확장하는 것이 추천됨
- 대규모 에이전트 템플릿(130명 조직 등)도 가져올 수 있으나, 실무 목적/업무 스타일에 맞춰 ‘맞춤형’ 조직구성이 중요

### 예산/비용 모니터링, 에이전트별 스펜드 관리 등 실제 기업 운영의 현실적 요소까지 반영

- 에이전트별·프로젝트별 한도 예산 설정 및 실 사용 금액 추적 지원
- 현재(데모 기준) Claude, Codex 등을 구독형 형태로 사용하여 월간 지출을 0으로 유지 중
- 오픈코드, 오픈라우터(OpenRouter)를 통한 저가 AI 모델 활용(예: Quinn 3.6 Plus는 무료)
- 주요 인텔리전스가 필요한 업무엔 고급 모델, 반복업무·단순업무엔 저비용/무료 모델 배분 가능

### Paperclip의 에이전트 개별화, 지속적 피드백 및 메타에이전트 운용을 통한 성능 개선 프로세스

- 각 에이전트 별로 사용자가 직접 상세한 가이드(예: 문제 발생 시 해결책 제시, 테스트 작성 시 주의점 등) 지속 관리
- 잘못된 결과가 나오면 즉시 피드백하도록 유도하여 학습 및 품질 개선
- ‘스킬 컨설턴트’라는 메타에이전트가 조직 내 다른 에이전트 진단/스킬 최적화를 도와줌
- 아직은 일부 수작업이나, 향후 버전에서 이러한 조직적 러닝과 에이전트 간 상호 피드백이 자동화될 예정

### 활발한 오픈소스 개발 현황, 기능 로드맵과 향후 발전 방향

- Paperclip은 2024년 3월 4일 오픈소스화, 약 34일 만에 50,000개 GitHub 스타 달성
- 커뮤니티의 활발한 PR(풀 리퀘스트)과 기능 강화로 빠르게 진화
- 향후 추가될 기능: CEO 채팅, 아티팩트/배포 관리, 맥시마이저 모드(프롬프트·토큰을 극대화해 자동화 몰입), 다중 사용자/팀 협업, 클라우드 및 샌드박싱 에이전트 배치, 데스크탑 앱(무료 오픈소스), 메모리 및 지식 베이스
- GitHub에서 로드맵 확인 가능하며, 30일 내 다수 신규 기능이 추가 예정(영상 시점 기준)

### Paperclip은 AI 에이전트가 인간의 업무를 완전히 대체하는 것이 아니라, 사용자가 직접 제어·감독하며 협업을 극대화할 수 있도록 하는 도구임을 강조함

- Paperclip은 “AI가 인간의 일을 빼앗는 것이 아님, 오히려 인간이 AI를 능동적으로 통제하며 대규모로 활용하는 시대”임을 강조
- 수천 명의 AI 노동력을 스스로 관리하는 'AI 조직장'의 역할을 사용자가 맡게 됨
- Paperclip으로 워크플로우를 명확히 시각화·모니터링·피드백함으로써 조직 내 품질과 일관성을 유지
- 사용자가 전문 지식 없이도 쉽고 투명하게 AI 노동력을 활용할 수 있도록, 무료로 제공
- 공식 사이트(paperclip.ing)에서 간단한 명령어(npx paperclip-ai onboard)로 즉시 시작 가능하며, 모든 사용 대상자가 쉽고 빠르게 접근할 수 있도록 배려됨

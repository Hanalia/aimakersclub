## *Claude Code로 완벽한 AI 영상 만들기: 전체 자동화 워크플로우* 핵심 요약

- 영상의 주제는 최신 AI 도구들을 연결하여 완전히 자동화된 영상(애니메이션+오디오) 생성 워크플로우를 만드는 과정을 시연 및 설명하는 것임
- HyperFrames(장면 렌더링 및 편집), Claude Code(스크립트 생성), 11 Labs(유료 음성 합성), Kokoro(무료 TTS), Archon(워크플로우 관리)의 조합을 활용함
- 모든 과정은 공개 깃허브 저장소에서 10분 이내로 간단히 설치 및 실행 가능하며, 최소한의 프롬프트 입력만으로 동작함
- Claude Code와 Archon의 결합을 통해 병렬적인 워크플로우 실행 및 세션 관리가 가능, 데이터는 SQLite 혹은 Neon Postgres에 저장됨
- 세 장의 기본 템플릿이 제공되며, 사용자는 코드 에이전트에게 자신만의 스타일/분량/구성 맞춤 템플릿 제작도 직접 프롬프트로 요청할 수 있음
- 전체 워크플로우는 웹리서치->스크립트 생성->오디오 생성->애니메이션/비주얼 합성->프리뷰->최종 영상(MP4) 렌더링으로 이어지는 단계로 구성
- Hyperframes의 프리뷰 대시보드에서 각 장면·음성·타이밍·전환 등을 실시간 확인/수정하여 완성도를 높임
- 예시 영상(Claude Code Agent View, Archon 소개 영상) 시연 및 템플릿 커스터마이징 과정도 상세히 다루어, 실사용 적용성을 보여줌
- 모든 구성요소는 오픈소스, 무료 기반이나, 11 Labs 음성 사용 시 별도의 API Key가 필요함
- 영상 종반에는 고급 템플릿 제작 및 자기만의 교육·브랜딩 목적 자동화 영상 생성법을 직접 적용하는 방법을 안내함

## 세부 요약 - 주제별 정리

### AI 기반 영상 자동 생성은 상상보다 빠르게 발전 중임을 증명함

- 불과 몇 달 전까지만 해도 LLM 기반 AI가 완전한 영상(애니메이션+음성) 생성까지는 어렵다고 생각했으나, 최근 변화가 가속화됨
- 영상 도입부에서 완전히 AI로 만든 YouTube Short(음성, 트랜지션, 애니메이션 모두 AI 생성)를 예시로 제시
- HyperFrames, Claude Code, 11 Labs(또는 Kokoro), Archon을 조합하여, 높은 수준의 자동화 영상을 실제 시연

### HyperFrames와 Claude Code 조합이 본 워크플로우의 핵심을 이룸

- HyperFrames는 Claude Code 코드 기반 애니메이션/영상을 실제로 렌더링하는 최신 에디터로, Remotion의 발전형임
- HyperFrames는 HTML 기반 템플릿으로 쉽게 커스터마이즈 및 시각 확인이 가능함
- Claude Code는 영상 스크립트(대사, 장면 전환, 메타데이터 등) 및 각종 연출 지시문을 생성함
- HyperFrames에서 실시간 프리뷰 및 장면별 타이밍/수정이 가능해, 빠른 반복 작업이 용이함

### Archon이 전체 워크플로우와 데이터 관리를 최적화함

- Archon은 오픈소스 워크플로우 오케스트레이터·하니스 빌더로, 모든 단계 실행·종속성 관리의 중추 역할
- GitHub 21,000+ Stars 기록, 한 커맨드로 에이전트 기반 개발 환경 구축이 가능
- SQLite 또는 Postgres(예: Neon) DB를 활용하여 각 영상 프로젝트의 실행 상태, 결과물을 저장 및 조회
- Archon 활용시, 여러 영상 병렬 생성도 지원되어 대규모 제작도 실용적

### 클론 저장소 설치부터 초간단 실행까지 모든 과정이 자동화됨

- 영상에서 오픈소스 저장소(깃헙) URL 제공, git clone 한 뒤 바로 사용 가능
- 간단한 프롬프트: "read the readme, set up everything so I can generate my first video"와 주제 입력만으로 세팅 완료
- 에이전트가 모든 의존성 설치, API Key 등록(11 Labs 필요 시), 디펜던시 구성까지 전자동 수행
- 무료 도구만 사용하는 옵션도 가이드하며, 전체 초기 설정에 10~15분 소요됨

### 영상 제작 워크플로우 단계를 구조화·자동화함으로써 제작 품질을 높임

- ① ID 생성(프로젝트별 격리 환경) ② 템플릿 복사 ③ 주제 리서치 및 반(反)허위 게이트 ④ 스크립트 생성(프롬프트 엔지니어링 포함) ⑤ 음성 합성(11 Labs/코코로) ⑥ 애니메이션 타이밍 계산 및 템플릿 변환(HTML) ⑦ 오디오, 전환, 레이아웃 검사 및 프리뷰 ⑧ 최종 mp4 생성 (필요시 세부 수정 반복)
- 모든 산출물(스크립트, 음성, 영상 등)이 독립 폴더 단위로 정리됨

### 템플릿 시스템으로 맞춤형 영상 유형도 쉽게 제작 가능함

- 3종류 기본 템플릿(Anthropic 등) 제공, 자신의 스타일/분량/구성 원하는 대로 에이전트 프롬프트로 맞춤 설계 가능
- 예를 들어: “더 난해한 AI 개념 설명용 템플릿 생성”, “브랜드 다이어그램, 비포/애프터 비교 등 포함 요청” 등 세부 기능까지 지정 가능
- Claude Code가 질의-응답 방식으로 템플릿 목적을 파악, 새로운 템플릿을 자가 생성·적용

### 실제 영상 제작 예시를 통해 실질적 결과물을 확인시킴

- Claude Code Agent View 데모 영상(25초) : 활용법, 주요 기능(동기화, 세션 관리, 명령어 등) 간결하게 소개
- Archon 소개 영상(30초) : 오픈소스 코드 하니스, 병렬 실행, 핵심 워크플로우, 깃허브 별 개수 등 구체적 정보로 구성
- 11 Labs에서 여러 음성(클론/기본) 테스트, 음성 자연스러움은 일부 인토네이션 한계 있으나 실사용에 무리 없음

### HyperFrames 프리뷰 대시보드로 최종 퀄리티를 정교하게 조정할 수 있음

- 영상 렌더링 전, 각 장면·음성·타이밍·트랜지션·슬라이드 구성을 실시간 프리뷰에서 검증/수정 가능
- 불필요한 전체 재렌더링 없이, 버그·음성 억양·전환 효과 등 세부 개선 후 원하는 시점에 MP4로 렌더 가능
- 텍스트, 오디오, 시각 요소의 오버플로우 등도 자동 검사 수행

### 템플릿 커스터마이징 과정을 단계별로 상세 안내함

- 새로운 개념 설명 템플릿 요청→ Claude Code가 기존 템플릿들과 비교 질의→ 원하는 구성 파악→ 완전 신규 템플릿 제작
- 제작된 템플릿은 이후 세션에서 지정해 영상 주제(예: MCP, RAG, Attention 등)에 따라 자동 적용됨
- 영상 길이·스타일·구성 등 모든 세부 설정 맞춤화 가능

### 반복적 개선 및 개인·팀 단위 활용도 강조함

- 영상 제작 후, 프리뷰 기반 반복 수정·세부 조정→ 완성본 추출까지 전 과정 자동 지원
- 예를 들어, 신기능 출시시 클로드 코드 설명 영상을 1분 이내로 자체 생성 가능
- 자가 학습/팀 설명 영상, 유튜브 숏츠, 커뮤니티 자료 등 범용적으로 활용 가능함

### 오픈소스·무료 기반이므로 자유롭게 실험·확장할 수 있음을 재차 강조함

- 대부분의 도구가 무료(오픈소스)이며, 단 11 Labs 음성 합성만 유료 플랜 필요
- 초간단 설치, 쉽고 빠른 결과 도출, 커스터마이즈 편의성 때문에 누구나 즉시 적용 가능
- Archon/HyperFrames/Claude Code 구성은 확장성이 뛰어나며, 다양한 영상 스타일 및 개인 워크플로우 구축 가능

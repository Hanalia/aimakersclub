## *Claude를 24시간 내내 코딩시키니 벌어진 일* 핵심 요약

- 영상은 인공지능 Claude Opus 4.5를 24시간 동안 멈추지 않고 코딩하도록 시킨 실험 과정을 다룸
- 실험 결과, 별도의 인간 개입 없이 claw.ai의 완전한 클론 수준의 애플리케이션이 자동으로 개발됨
- 애플리케이션에는 대화 이력 확인, 마크다운 포맷 유지, HTML 페이지 생성, 코드 작성 및 실행, 다양한 설정 UI 등 풍부한 기능이 탑재됨
- 사용자는 대화 기록을 시각적으로 살펴볼 수 있고, 각 반복 루프에서 AI가 자동 검증을 거치는 과정까지 확인 가능
- 테마 및 기본 모델 선택, 최대 토큰 수 조정 슬라이더 등 사용자 지정 기능이 구현됨
- 아직 휴먼 인 더 루프(HITL) 기능은 추가 전이지만, AI만으로 이 정도 수준의 소프트웨어 제작이 가능함이 확인됨
- 만약 24시간이 아니라 더 오랜 시간 방치했다면 완벽에 가까운 애플리케이션도 가능할 것이라는 언급도 있음
- Claude Opus 4.5와 같은 AI 모델과 자체 검증 및 반복 실행 기능(harness)의 조합이 현대적 앱 제작 환경을 크게 혁신할 수 있음을 시사함
- 실험자는 “손가락 하나 대지 않고도 이만큼 많은 기능 구현이 가능하다”고 놀라움을 표현함

## 세부 요약 - 주제별 정리

### Claude Opus 4.5를 24시간 연속 코딩에 투입하자 기능 완성도가 크게 높아짐

- 실험자는 Claude Opus 4.5라는 최신 AI 모델을 24시간 동안 단일 작업 환경에 두고 연속적으로 코딩하도록 시킴
- 독립적인 개입 없이 AI의 반복적 작업 결과만으로 프로젝트 빌드 상태를 점검
- 실험이 진행될수록 예정된 테스트의 절반만 지나도 이미 대부분의 기능이 완성됨을 확인
- claw.ai(특정 챗봇 애플리케이션)과 거의 흡사한 ‘완전히 동작 가능한 클론’이 구축됨

### AI가 자동으로 대화 이력 및 검증 루프를 관리함

- 애플리케이션은 AI가 생성한 과거 대화 데이터를 체계적으로 저장 및 시각화
- 각 기능 추가 혹은 리팩토링 과정에서 반복적으로 검증용 테스트를 실행
- “각 루프마다 결과를 검토하면서 애플리케이션 완성도가 점진적으로 향상되는 과정을 눈으로 확인할 수 있었다”고 설명

### 마크다운 포맷, HTML 생성, 코드 실행 등 다양한 기능이 장착됨

- 애플리케이션에 ‘매우 좋은 마크다운 포맷팅’이 적용되어 텍스트 표현력이 뛰어남
- 다양한 종류의 HTML 페이지를 자동으로 생성
- 내부적으로 코드 작성과 실행이 모두 웹상에서 가능
- 별도의 플러그인이나 추가 개발 없이도 복합적인 프론트엔드/백엔드 로직이 실현됨

### 사용자 환경 설정(UI) 기능이 상용 서비스 못지않게 구현됨

- 테마(디자인 스타일)를 직접 바꾸는 설정 기능이 포함됨
- 사용자가 선택할 수 있는 AI 기본 모델 교체 옵션이 추가됨
- ‘Max tokens’값(한 번에 생성할 수 있는 토큰 수)을 조정하는 슬라이더 UI도 적용됨
- UI 상에서 직접 다양한 설정을 손쉽게 변경 가능

### 휴먼 인 더 루프(HITL)는 미구현이나, 기능 완성도는 매우 높음

- 사람이 과정 중간에 직접 참여(검토·승인)하는 ‘Human in the loop’ 시스템은 추후 추가 예정으로 언급
- 그럼에도 불구하고 현 단계에서 이미 거의 완벽하게 기능하는 애플리케이션이 완성됨
- 실험자는 지금까지 “아무것도 손으로 직접 수정하지 않았음”을 강조

### AI만으로도 고도화된 앱 제작이 가능함을 실험을 통해 입증함

- 추가 테스트 절차까지 모두 이행한다면 ‘절대적으로 완벽한 애플리케이션’도 만들 수 있을 것임을 언급
- Claude Opus 4.5 수준의 AI 모델과 반복적 자동 검증 시스템(harness)을 병행하면 개발 능력과 자동화 수준이 크게 향상됨
- 실험 결과에 대해 “지금 우리가 Claude로 만들 수 있는 것의 수준이 터무니없이 대단하다”고 직접 평가

### 반복적 검증 및 개선 루프가 AI 생산성의 핵심임을 확인함

- AI의 반복적인 “검증→수정→테스트” 루프가 실제 제품 완성도와 신뢰성을 끌어올림
- 이러한 하니스(자동 검증·실행 프레임워크)가 AI가 더욱 복잡한 응용프로그램까지 자가 개선할 수 있는 기반을 제공함

### 손쉬운 자동화와 ‘무손’ 개발 경험에 놀라움을 표함

- 실험자는 "직접 뭔가를 손으로 만지지 않아도 이 정도 앱이 완성될 줄 몰랐다"며 AI의 자동화 능력에 경탄
- 영상을 통해 “하루만 더 돌렸어도 완벽에 가까운 프로그램이 됐을 것”이라고 밝힘

### Claude Opus 4.5와 하니스 조합이 앱 제작의 미래를 제시함

- 단일 AI 모델만이 아니라, 자동화된 반복 검증 환경(하니스)을 병행 구축한 것이 프로젝트 완성의 핵심 동인으로 작용
- 실험에서 드러난 결과는 향후 ‘노코드 AI개발’, ‘자가진화 앱 프로토타이핑’의 구체적 가능성을 시사
- 복잡하고 기능 풍부한 소프트웨어도 AI만으로 신속하게 제작할 수 있다는 점을 실질적으로 시연

### 결론: Claude를 통한 대규모 기능 구현 및 소프트웨어 자동화의 잠재력이 직접 확인됨

- 24시간 비개입형 AI 코딩 실험 결과, 이미 claw.ai 클론 수준의 제품이 자동 생성됨
- 저장, 레이아웃, 검증, UI, 코드 실행 등 주요 요소가 한 번에 완성됨
- Claude Opus 4.5 및 반복 하니스의 강력한 조합이 허용하는 개발 혁신을 실감할 수 있음

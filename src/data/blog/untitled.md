## *Claude에게 24시간 동안 코딩을 시켜본 결과* 핵심 요약

- 영상 제목은 "Claude에게 24시간 동안 코딩을 시켜본 결과"로, Claude Opus 4.5 AI 모델을 이용해 하루 동안 쉬지 않고 코딩 작업을 시킨 결과물을 리뷰함
- 현재까지 구현된 기능만으로도 claw.ai의 완벽에 가까운 클론이 만들어졌다고 밝힘
- 시스템은 자동 테스트 루프를 통해 각 단계별로 생성된 대화 기록과 검증 과정을 모두 저장·확인할 수 있음
- 마크다운 포맷 지원이 매우 훌륭하게 작동하며, 코드 작성 및 실행, HTML 페이지 생성까지 가능
- 테마 변경, 기본 모델 선택, 최대 토큰 슬라이더 등 다양한 설정 기능이 포함됨
- 다양한 기능들이 손 하나 까딱하지 않고 생성된 것에 대해 개발자는 매우 놀라고 있음
- 아직 사람의 개입(human in the loop) 기능 추가가 남아 있음에도 불구하고, 이미 매우 완성도 높다고 평가
- Claude Opus 4.5 및 해당 하네스를 활용하면 애플리케이션의 자동 검증 및 반복 갈고닦기가 탁월하게 이뤄질 수 있음을 강조
- 만약 이 상태로 하루 더 자율적으로 작업을 시키면 완벽한 애플리케이션이 나올 거라는 가능성도 짚음
- 이 영상을 통해 Claude의 자동화된 개발 능력과 현대 AI 도구의 강력함이 잘 드러남

## 세부 요약 - 주제별 정리

### Claude가 24시간 내에 claw.ai 클론을 생성하는 과정을 개발자가 극찬함

- 개발자는 Claude Opus 4.5를 활용해 24시간 동안 코딩을 지속적으로 수행시킴
- 현재까지 구현된 시스템을 "굉장히 인상적"이라고 평가함
- claw.ai와 거의 동일하게 작동하는 완벽한 클론이 이미 만들어졌다고 언급함
- 완성도가 매우 높아 테스트의 후반부 목적조차 알 수 없을 만큼 만족도가 큼

### 테스트 루프와 대화 이력 저장 방식이 구현됨

- 생성 과정에서 각 단계별로 테스트(검증) 루프를 돌리며, 그 내용을 모두 기록함
- 과거에 생성된 대화와 작업 기록을 모두 손쉽게 다시 확인할 수 있도록 설계됨
- 이런 점에서 객체지향적으로 기능 검증 및 과정 추적의 용이성이 부각됨

### 마크다운 지원 및 코드 실행, HTML 생성 기능 탑재

- Claude가 만들어낸 결과물은 뛰어난 마크다운(markdown) 포맷팅을 제공함
- 텍스트 뿐만 아니라 HTML 페이지 생성 기능도 통합되어 있음
- 코드 작성부터 코드 실행까지 자동화가 가능하여 개발 생산성을 극대화함

### 사용자 설정 환경(테마, 모델, 토큰 슬라이더 등)이 다양하게 구현됨

- 사용자 인터페이스 내에서 테마(밝기/어둠 등)를 쉽게 변경 가능함
- 사용할 AI 기본 모델을 설정할 수 있는 기능이 있음
- 처리할 최대 토큰 수를 직관적인 슬라이더 UI로 조절할 수 있음

### 자동화된 개발과 검증 과정에서 손수 작업이 전혀 필요하지 않음에 감탄

- "손 하나 까딱하지 않고" 이만큼의 기능을 가진 시스템을 만들 수 있었다고 언급함
- 개발자가 직접 개입하지 않아도 기능 추가와 자동 검증이 이어져 높은 효율성을 달성함
- AI에게 하루 더 작업을 맡긴다면, 더욱 완벽한 애플리케이션이 나올 수도 있다고 기대함

### 아직 사람의 직접 개입 기능은 추가해야 하지만 이미 높은 완성도가 증명됨

- human in the loop(사람의 개입) 기능을 앞으로 추가할 필요가 있다고 설명함
- 그럼에도 지금까지 구현된 결과만으로도 충분히 놀랍고 완성도가 높다고 평가함

### Claude Opus 4.5의 개발 자동화 및 반복 개선 능력 강조

- Claude Opus 4.5와 같은 하네스(harness)를 통해 애플리케이션 검증 및 반복 개선이 매우 용이해짐
- 다양한 새로운 기능 추가와 즉각적 검증이 하나의 자동화된 루프로 계속 이루어짐
- 최신 AI 도구의 기능이 소프트웨어 개발의 새로운 패러다임임을 시사함

### 추가적인 작업 시간 투입 시 완벽에 가까운 결과도 기대 가능함

- 만약 Claude에게 24시간을 추가로 부여한다면, 더욱 완벽한 결과물이 나올 것이라고 언급함
- 자동화된 반복 기능과 기능 확장성이 매우 뛰어남을 강조함

### Claude와 하네스(앱 실행 및 검증 프레임워크)를 통한 AI 활용의 무한 가능성 확인

- Claude의 강력한 실행력과 자체 검증 기능을 통해 사람이 개입하지 않고도 고도의 앱 개발이 가능함을 보여줌
- 하네스가 각 기능과 개선 과정을 체계적으로 관리·검증하는 기반이 됨

### AI 기반 개발툴의 혁신성이 실제 프로젝트 적용을 통해 확인됨

- 개발자는 Claude Opus 4.5 기반 도구의 혁신성과 생산성을 직접 체험했음을 강조함
- claw.ai 같은 AI 앱도 AI가 인간 없이 개발하는 시대의 도래 가능성을 시사함

## *Claude에게 24시간 내내 코딩을 강제한 실험 결과* 핵심 요약

- 영상 제목은 ‘Claude에게 24시간 내내 코딩을 강제한 실험 결과’로 번역됨.
- 실험자는 Claude Opus 4.5를 사용해 24시간 동안 자동 코딩을 진행함.
- 결과적으로 claw.ai의 거의 완벽한 복제본을, 코드 한 줄 직접 쓰지 않고 개발할 수 있었다고 언급함.
- 각 반복(loop)마다 Claude가 생성한 이전 대화 기록 전체를 확인할 수 있으며, 해당 검증 과정도 추적 가능함.
- 생성된 결과물에는 마크다운 형식의 문서, 다양한 HTML 페이지, 코드 작성 및 실행 기능 등 다양한 요소가 포함됨.
- 설정 메뉴에서는 테마, 기본 모델, 최대 토큰 수 조절 슬라이더 등이 제공되어 사용자 맞춤화가 용이함.
- 추후 ‘human-in-the-loop(사람 개입)’ 요소 추가 필요성을 언급하지만, 현재 기능만으로도 매우 인상적이라고 평가함.
- Claude Opus 4.5와 검증을 위한 자동화 프레임워크(harness)의 조합이 반복적 실험과 검증을 수월하게 만듦을 강조함.
- 추가로 하루만 더 돌리면 완벽한 앱이 탄생할 수 있다고 가능성을 시사함.
- 최근 AI 툴로 개발 가능한 범위와 수준이 비약적으로 향상됨을 체감하며, 실험의 가치와 결과를 구체적으로 설명함.

## 세부 요약 - 주제별 정리

### Claude Opus 4.5를 통한 자동화 코딩 실험의 진행 상황이 매우 인상적임

- 실험자는 Claude Opus 4.5를 이용해 24시간 동안 논스톱(auto) 코딩을 시행함.
- ‘현재까지 구축한 모든 것들이 상당히 인상적’이라며 중간 평가를 내림.
- 마지막 테스트 반의 목적마저 모를 정도로 이미 모든 기능이 잘 구현됨을 언급.
- claw.ai의 완전한 복제판에 가깝게 동작한다고 느꼈음을 밝힘.

### 대화 기록과 검증 과정을 모두 투명하게 확인할 수 있음

- 코딩 실험의 각 반복(loop)마다 Claude가 생성한 대화와 결과물을 쉽게 확인 가능.
- 이전 대화와 명령, 코드 작성 및 검증 과정을 모두 추적할 수 있도록 설계됨.
- 이를 통해 실험자가 전체 개발 프로세스를 상세히 검증할 수 있었음.

### 다양한 기능과 세련된 마크다운, HTML 페이지, 실행 코드를 자동 생성함

- 결과물에는 포맷이 우수한 마크다운 문서들이 포함됨.
- 다양한 HTML 페이지를 자동으로 생성할 수 있었음.
- 코드 작성뿐만 아니라, 자동으로 코드를 실행하는 기능도 구현함.
- 생성된 기능들의 완성도가 높아 추가 수정 없이 사용할 수 있을 수준임.

### 사용자는 설정 창에서 다양한 특성을 직접 조정할 수 있음

- 생성된 인터페이스에서는 테마(theme) 및 기본 사용 모델을 직접 변경 가능함.
- 최대 토큰 수(max tokens)를 조절할 수 있는 슬라이더 기능이 구현됨.
- 사용자 경험을 세밀하게 설정할 수 있는 다양한 환경설정 옵션이 마련됨.

### 아직 사람의 개입(human-in-the-loop) 요소는 추가 예정임

- 실험자는 향후 ‘human-in-the-loop’ 기능(사람의 검수, 개입 등)을 추가해서 더 완벽을 기할 계획임을 밝힘.
- 현재 자동화만으로도 기능 구현이 충분히 뛰어나다고 평가함.

### Claude Opus 4.5의 자동화 및 반복 검증력에 극찬을 보냄

- 하루만 더 실험을 연장하면 완전히 완벽한 앱이 될 수 있을 것이라고 긍정적으로 전망함.
- Claude Opus 4.5와 검증&반복용 자동화 프레임워크(harness)의 조합에 큰 가능성을 느낌.
- “이제 이런 수준의 애플리케이션까지도 완전히 자동화로 구현 가능한 세상이 오고 있다”는 느낌을 전달.

### AI의 프레임워크와 반복 검증 메커니즘이 개발의 새로운 청사진을 제시함

- 코드 작성, 실행, 검증, 환경 변경 등 전체 사이클이 자동화되어 효율성이 극대화됨.
- 수많은 반복(loops)을 거치며, 이전 결과와 대화 내용도 모두 DB처럼 활용 가능함.
- AI가 단순 반복이 아닌 실제 의미 있는 기능 구현과 오류 검증까지 담당, 개발 과정에서 발생할 수 있는 문제도 자체적으로 처리함.

### 실험자 본인의 직접 작업 최소화가 가능해짐

- 실험자는 “직접 손 하나 안 대고 여기까지 왔다”는 점을 강조함.
- 대부분의 과정이 자동화 덕분에 실험자가 개입할 필요가 없었다고 밝힘.
- 개발 생산성 및 효율성을 극적으로 끌어올릴 수 있음을 실증함.

### claw.ai와 같은 완성형 애플리케이션 복제 및 기능 추가가 AI로도 충분히 가능함을 시사함

- claw.ai와 유사한 구조와 기능들을 Claude Opus 4.5로 거의 완벽하게 복제함.
- HTML, 마크다운, 설정, 코드 실행 등 주요 기능을 모두 문제 없이 생성하며 실질적 복제임을 강조.
- 앞으로 AI의 자동화 개발 능력 확장의 잠재력이 매우 크다고 시사함.

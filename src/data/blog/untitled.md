## *클로드 코드로 더 안전한 오픈클로(OpenClaw) 대체품을 직접 만들었다* 핵심 요약

- 영상은 현존하는 인기 오픈소스 AI 비서 '오픈클로(OpenClaw)'가 얼마나 급속도로 흥행했는지 소개하며 시작함
- 오픈클로는 개인 맞춤형 AI 에이전트로, 사용자에 대해 배우는 기억 기능 및 선호도 추적 기능 등이 강점임
- GitHub 오픈클로 저장소는 185,000개라는 이례적인 수의 스타를 받았으며, 이는 n8n 등 기존 인기 오픈소스 프로젝트도 능가함
- 하지만 오픈클로는 에이전트에게 많은 권한을 부여하고 코드베이스 자체에도 구조적 결함이 있다는 점에서 심각한 보안 문제가 지적되고 있음
- 이러한 문제와 더불어 개인화된 맞춤 기능이 필요하다는 이유로 영상 제작자는 스스로 오픈클로의 대체품을 제작하게 됨
- 제작자는 오픈클로의 핵심적인 강점(개인화, 기억 축적 등)을 며칠 만에 Python 코드와 마크다운 몇 천 줄로 직접 구현하는 데 성공했다고 밝힘
- 영상을 통해 시청자에게 직접 오픈클로의 매력을 참고하여 자신만의 안전하고 맞춤화된 AI 비서를 만드는 과정을 소개하겠다고 안내함
- 오픈소스 AI 프로젝트의 노하우를 적극적으로 참고하되, 직접 만들어보며 완벽히 이해하는 경험이 중요하다고 강조함

## 세부 요약 - 주제별 정리

### 오픈클로 프로젝트의 폭발적인 인기와 영향력은 전례 없는 수준으로 성장함

- 오픈클로(OpenClaw)는 현재 가장 화제가 되는 개인용 AI 비서 프로젝트임
- 사용자로부터 크게 호평받고 있으며, '진짜 내 마음을 이해하는 첫 AI 에이전트 같다'는 평가가 많음
- 시간이 지날수록 사용자가 진행 중인 업무와 개별 취향을 점점 더 잘 학습하고 기억함
- 완전히 오픈소스 기반으로, 누구나 자유롭게 소스를 활용 가능함
- GitHub 저장소 기준, 185,000스타를 돌파했는데, 이는 다른 유명 오픈소스 프로젝트인 n8n을 능가하는 수치임
- 영상 제작자인 Cole Medin은 본인도 GitHub에서 이런 사례를 본 적이 없다고 강조함

### 오픈클로의 급성장 이면에는 심각한 보안 문제가 존재함

- 오픈클로는 사용자에 대한 높은 이해와 맞춤형 기능을 위해 에이전트에게 많은 권한과 제어력을 부여함
- 동시에 코드베이스 자체에도 잠재적·구조적 보안 결함이 존재함
- 영상 제작자는 이러한 보안 이슈가 잘 알려져 있음을 시사함
- 보안 문제로 인해 사용자가 실제 업무나 중요한 정보 처리를 맡기기에는 리스크가 크다고 판단함

### 영상 제작자는 오픈클로의 아이디어를 바탕으로 직접 대체품을 제작함

- 오픈클로의 핵심 아이디어와 기능은 매우 유용하다는 점에 동의함
- 그러나 보안 문제와 동시에 '나에게 더 맞는, 커스텀 가능한 비서'가 필요했음
- 이 두 가지 동기로 인해 영상 제작자는 자신만의 오픈클로 버전을 만들기로 결심함

### 오픈클로의 주요 기능적 매력은 며칠 만에 재현 가능하다는 점이 강조됨

- 오픈클로의 핵심 장점(기억, 사용자 선호도의 학습, 개인화 등)은 큰 노력을 들이지 않고도 복제 가능함을 발견함
- 영상 제작자는 Python과 markdown으로 된 수천 줄의 코드만으로 본질적인 기능을 직접 구현함
- '단 며칠, 몇천 줄의 코드면 충분했다'고 언급함
- 시청자 역시 충분히 쉬운 작업이므로, 직접 시도할 것을 권장함

### 영상은 시청자에게 직접 AI 비서를 만들어볼 것을 적극적으로 제안함

- 오픈클로와 같은 혁신적인 오픈소스 AI 프로젝트의 아이디어를 학습하는 것이 중요함
- 하지만 남이 만든 도구를 그대로 사용하는 것보다는, 실제로 자신이 구현해보면서 원리를 이해하는 것이 가치 있음을 강조함
- 자신의 필요와 취향에 맞게 맞춤화 및 보완이 가능한 자가 제작의 장점을 언급함
- '진짜 내가 통제하고 이해하는 AI 비서를 갖는 경험'의 중요성을 역설함

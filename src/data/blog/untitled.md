## *이 무료 도구로 Claude 코드가 10배 더 좋아진다* 핵심 요약

- 영상 제목은 “이 무료 도구로 Claude 코드가 10배 더 좋아진다”이며, Claude 코드 사용자의 코드 리뷰 작업을 획기적으로 개선하는 도구를 소개함
- 제작자는 Claude 코드의 강력함을 인정하지만, 여러 인스턴스의 병렬 실행 결과물을 검토하는 데 어려움이 컸음을 밝힘
- 가장 큰 문제는 AI가 생성한 코드의 모든 결과물을 일괄적으로 리뷰하는 과정이 번거롭고 시간이 많이 든다는 점임
- 이를 개선하기 위해 제작자는 무료 도구 “Kinugi(키누기)”를 적극적으로 사용하게 되었고, 핵심은 Cloud Code를 대체하지 않고 그 위에 시각적 인터페이스 레이어를 제공함
- Kinugi는 자신이 동시에 여러 Claude 코드 인스턴스를 실행할 때, 모든 에이전트 세션의 상태, 변경 내역, 리뷰 필요 여부를 한눈에 시각적으로 볼 수 있게 해줌
- 에이전트가 구현 플랜을 세우면 즉시 검토 및 실시간 댓글로 피드백할 수 있으며, 최종 리뷰 단계에선 풀 리퀘스트(Pull Request) 방식의 일괄 검토가 가능함
- AI가 생성한 코드 리뷰 종류도 다양하게 적용 가능하고, 특히 Sonar 팀이 개발해서 보안 코드 리뷰 기능 등도 내장됨
- Kinugi는 현재 macOS에서만 사용 가능하고 매우 실험적인 버전이지만 무료로 제공되며, Sonar 측의 제안으로 제작자가 직접 사용 후기와 추천 후기를 공유함

## 세부 요약 - 주제별 정리

### Claude 코드의 강력함과 동시에 발생하는 주요 불편함을 경험함

- Claude 코드는 개발 작업에서 매우 강력하게 작동하며, 제작자는 모든 개발에 Claude 코드를 적극적으로 활용하고 있음
- 하지만 실제로 Claude 코드를 많이 쓸수록 몇 가지 ‘갭(gap)’들, 즉 단점이 느껴졌다고 밝힘
- 그중 가장 큰 문제점은 AI가 출력하는 코드 결과물을 한 번에 모두 리뷰(검토)해야 하는 번거로움임
- 특히 복수의 Cloud Code 인스턴스를 병렬로 실행하는 경우, 각각의 결과물을 일일이 확인하는 데 많은 시간이 소요됨

### 기존 Claude 코드의 리뷰 및 검증 과정이 비효율적임을 설명함

- 실시간으로 Claude가 파일, 코드, 기능 등의 여러 부분을 빠르게 수정/생성하지만, 사람이 이 결과를 확인하는 과정이 불편함
- 실제로 각 Cloud Code 로그와 생성된 파일들을 개별적으로 열어보아야 하기 때문에, 병렬 프로세스 수가 많을수록 업무 부하가 크게 증가함
- AI 코드라도 반드시 사람이 최종적으로 검토(reviews)해야 한다는 점을 강조함

### Kinugi라는 무료 도구가 Cloud Code의 리뷰 과정을 획기적으로 개선함

- 최근에 ‘Kinugi(키누기)’라는 무료 도구를 알게 되어 매우 만족스럽게 사용하고 있다고 소개함
- Kinugi는 기존 Cloud Code를 대체하는 것이 아니라, 그 위에서 동작하는 ‘시각화 레이어’임을 강조
- 즉, 기존 Claude 코드의 사용 방식을 그대로 유지하면서 추가적인 편의 기능을 얻을 수 있음
- Kinugi는 “agentic development environment”, 즉 여러 AI 에이전트 세션을 시각적으로 한눈에 관리하는 도구임

### Kinugi를 활용하면 여러 Claude 코드 세션의 상태와 변경 내용, 리뷰 필요 여부를 한눈에 파악할 수 있음

- Kinugi에서는 모든 병렬 Claude 코드 에이전트 세션을 한 화면에 시각적으로 펼쳐놓고 관리할 수 있음
- 각 세션마다 어떤 항목이 리뷰가 필요한지, 현재 실행 중인 것은 무엇인지 직관적으로 확인 가능함
- 에이전트가 작업 계획(plan of attack)을 생성하면, 즉각적으로 해당 플랜을 검토하고 댓글로 피드백 제공 가능
- 사람들이 Claude 코드의 각 결과물을 일일이 로그에서 찾지 않고, 모든 변경사항을 종합해서 “풀 리퀘스트” 형식으로 한 번에 리뷰할 수 있음
- 이는 작업 흐름의 효율성을 크게 높여주며, 리뷰 및 피드백 과정을 간소화함

### Kinugi 플랫폼이 제공하는 여러 장점 및 특이 기능이 있음

- 가장 큰 장점은 모든 병렬 세션의 현황과 리뷰 상태, 진행 상황을 한 곳에서 시각적으로 모니터링할 수 있다는 점임
- 프로그래밍 작업 중 ‘어떤 세션이 완료되었는지’, ‘어떤 코드가 리뷰 대기 중인지’ 명확하게 파악 가능함
- 코멘트(댓글)를 바로 달 수 있어, AI 에이전트가 실시간으로 해당 피드백을 바로 반영함
- 최종 단계에서는 풀 리퀘스트처럼 변경된 코드 묶음을 일괄적으로 보고 승인(혹은 재요청) 가능함

### Kinugi는 AI의 자동 코드 생성물에 대해 다양한 유형의 코드 리뷰를 적용할 수 있게 설계됨

- 전통적 코드 리뷰뿐만 아니라, 보안 코드 리뷰(security review) 기능도 내장
- Kinugi가 제공하는 리뷰 옵션은 AI가 만든 코드라도 ‘품질 유지’, ‘보안’, ‘표준 준수’ 등이 가능하도록 돕는 역할을 함
- Sonar팀에서 개발했기 때문에, 코드 품질 및 보안에 특화된 리뷰 기능과 경험이 반영됨

### Sonar팀이 개발한 Kinugi의 기술적, 실험적 특징들을 설명함

- Kinugi는 Sonar팀에서 제작했으며, Sonar는 코드 품질과 보안 분야에서 잘 알려진 기업임
- Sonar에서 Kinugi를 macOS 전용으로 개발하였고, ‘아주 실험적인’(very experimental) 상태임을 명시함
- 이 때문에 Kinugi는 현재 무료로 제공되고 있으며, macOS 환경에서만 직접 체험해볼 수 있음

### 제작자가 Sonar의 제안으로 Kinugi를 직접 사용하고 영상을 제작함

- Sonar팀에서 제작자에게 직접 Kinugi를 체험해 보라고 연락했고, 이에 곧바로 관심을 갖고 도구를 사용해 봄
- Kinugi를 매우 높이 평가하며, ‘정말 멋진 툴’이라며 시청자에게 소개함
- 영상 마지막에 Sonar팀에 감사 인사를 전하고, 소개 링크를 영상 설명란에 남긴다고 안내함

### 결론적으로 Kinugi는 Claude 코드의 생산성, 리뷰 편의성, 보안성이 모두 향상되는 혁신적인 툴임

- 기존 Claude 코드 사용자가 느끼는 리뷰 불편함, 복수 세션 관리의 번거로움 등 근본적인 문제를 효율적으로 해결함
- 시각적 에이전트 관리, 풀 리퀘스트 리뷰, 보안 검토 등 차별화된 기능을 macOS 환경에서 무료로 체험 가능함
- AI 기반 개발과정을 한층 더 실무적으로 다루고 싶은 개발자들에게 Kinugi의 사용을 적극적으로 권장함

### Kinugi 사용 시 유의사항 및 향후 활용 방향에 대한 안내가 있음

- 아직은 macOS만 지원하며, 실험적 단계의 소프트웨어로 잦은 변화가 예상됨
- 공식 웹사이트를 통해 다운로드 및 추가 정보 확인 가능함
- 생산성 향상에 관심이 있다면 직접 사용해볼 가치가 있음을 밝힘

## *Claude에게 24시간 동안 멈추지 않고 코딩을 맡겼더니 벌어진 일* 핵심 요약

- 영상 제작자는 Claude Opus 4.5를 이용하여 24시간 동안 자동으로 애플리케이션을 개발하도록 실험함
- Claude가 claw.ai와 유사한 클론 애플리케이션을 별도의 인간 개입 없이 거의 완벽하게 구현했음을 밝힘
- 개발 과정에서 자동 생성된 대화 내역과 검증 반복 과정(루프)을 모두 확인할 수 있다는 점을 언급함
- 애플리케이션은 마크다운 포맷팅 지원 등 매우 깔끔한 UI를 갖추고 있음
- HTML 페이지 생성, 코드 작성 및 실행 등 고급 기능까지 자동으로 구현됨
- 설정(세팅)에서 테마 및 기본 모델 변경, 최대 토큰 수 조절 슬라이더 등 다양한 옵션 제공
- 기능이 매우 풍부하며, 아직 'human in the loop'(사람의 검토 및 개입) 기능만 추가 예정이라고 설명함
- 사용자 개입 없이도 이 정도까지 맞춤형 앱을 자동으로 생성할 수 있다는 점에 큰 놀라움을 표함
- 만약 Claude를 더 오랜 시간 실행하면 더욱 완벽한 애플리케이션이 나올 것이라고 예상함
- Claude Opus 4.5와 테스트 및 반복을 관리하는 하네스(harness)가 있으면 검증과 반복 개발이 매우 용이해짐을 시연함

## 세부 요약 - 주제별 정리

### Claude는 24시간 동안 자동으로 claw.ai의 완성도 높은 클론을 제작함

- 영상 제작자는 Claude를 사용해 별도 수동 조작 없이 24시간 코딩을 실행함
- 목표는 claw.ai와 유사한 기능의 클론 애플리케이션을 자동으로 완성하는 것이었음
- 결과적으로 인간이 전혀 개입하지 않았음에도, 거의 완벽하게 동작하는 앱이 완성됨
- 이 과정에서 Claude가 테스트 케이스를 대량으로 반복 실행하며 자체적으로 검증함

### 자동 생성된 대화와 검증 과정은 투명하게 추적 및 확인 가능함

- 생성된 대화 내역(컨버세이션 히스토리)을 모두 확인할 수 있는 기능이 앱에 포함됨
- 각 루프(반복)마다 어떤 것이 검증되어 진행됐는지 명확히 볼 수 있음
- 이러한 추적 기능은 디버깅, 품질 관리, 개발 신뢰성 향상에 크게 기여함

### 마크다운 포맷팅, HTML 페이지 생성 등 다양한 고급 기능이 자동 탑재됨

- Claude가 자동 생성한 앱은 마크다운 문서 형식을 자연스럽게 지원
- 다양한 HTML 페이지 생성 기능이 기본적으로 들어가 있음
- 나아가 코드 작성과 실행까지 Claude가 직접 구현하여 개발자의 손길이 전혀 필요 없었음

### 애플리케이션은 다양한 커스터마이징 설정을 제공함

- 앱 내에 테마(색상 등 디자인) 변경 기능이 탑재됨
- 기본 AI 모델 변경이 가능해 다양한 용도에 맞게 적용할 수 있음
- 최대 토큰 수를 슬라이더로 간편하게 조절할 수 있는 UI도 자동으로 구현됨
- 사용자가 개입하지 않았음에도 매우 풍부한 기능 구성을 갖춤

### 아직 사람의 검토(‘human in the loop’) 기능만 추가가 남았음을 밝힘

- 영상 제작자는 마지막으로 ‘human in the loop’ 기능만 추가하면 될 것이라고 설명함
- 이 기능은 Claude가 자동으로 개발한 결과물 중 일부에 대해 인간 개발자가 점검, 수정하는 메커니즘
- 즉, 이미 대부분의 기능이 완성된 상태임을 강조함

### Claude Opus 4.5와 하네스(harness) 조합은 반복 검증 및 개발 자동화에 매우 효과적임을 시연함

- Claude Opus 4.5는 고도화된 자동화 및 반복 개발이 가능한 AI임을 실험으로 증명함
- 하네스(harness)란, 테스트 케이스 검증 및 반복 실행을 지원하는 관리 프레임워크를 의미
- 이 조합으로 일련의 기능을 자동 검증·개선하는 전체적 개발 파이프라인이 구현됨
- AI가 코드는 물론, 테스트 및 릴리즈까지 통합적으로 처리하는 모습을 보여줌

### 사용자의 추가 개입 없이도 복잡한 앱을 거의 완성할 수 있다는 점이 놀라움을 자아냄

- 영상 제작자는 자신의 직접적인 노력이 거의 들어가지 않았음에 놀라워함
- 사람 없이도 완성도 높은 애플리케이션이 개발되는 현실을 강조함
- “만약 하루만 더 계속 실행하면 완벽한 앱이 만들어질 것”이라며 AI 개발의 강력함을 강조함

### Claude의 자동화 능력으로 소프트웨어 개발의 한계를 넘어섰음을 보여줌

- 단순한 코드 생성이 아니라, 실제 작동 가능한 복잡한 서비스 전체가 자동화로 개발됨
- 개발, 테스트, UI 구성, 설정, 코드 실행 등 여러 단계가 AI 자동화로 통합됨
- 이러한 통합적 자동화 능력이 기존 개발 방식의 한계를 넘어섬을 시연함

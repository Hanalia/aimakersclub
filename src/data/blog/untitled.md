## *이 무료 툴로 Claude 코드 품질이 10배 좋아진다* 핵심 요약

- 영상에서는 Claude Code의 뛰어난 효율성에 대해 강조하면서, 실제 코드 개발 전 과정에 Claude Code를 적극 활용 중이라고 밝힘
- Claude Code 사용 시 가장 큰 단점은 결과물 리뷰 과정이 번거롭고, 병렬로 여러 인스턴스를 동시에 운영할 때 관리의 어려움이 있다는 점임
- 이 문제를 해결하기 위해 소개되는 툴은 "Kinugi(키누기)"로, Sonar 팀이 개발하였으며 완전 무료로 제공됨
- Kinugi는 Claude Code 위에 시각적 레이어로 덧씌워진 ‘에이전틱 개발 환경’을 제공하여, 모든 병렬 세션을 한 번에 관리·리뷰 가능하게 함
- 코딩 에이전트들이 생성한 변화, 실행중인 세션, 리뷰가 필요한 부분 등을 한 화면에서 즉시 확인할 수 있음
- Claude Code의 작업 플랜(Plan of Attack)에 대해 실시간 코멘트 및 피드백이 가능하며, 이후 구현 및 리뷰가 완료되면 이를 “풀리퀘스트” 형태로 검토 가능
- 코드 리뷰 과정에서 보안(Security) 리뷰 옵션도 제공하는데, 이는 Sonar 팀의 특성이 잘 반영된 부분임
- Kinugi는 현재 MacOS에서만 사용 가능하며, 실험적 상태이기 때문에 무료이나, 그럼에도 매우 강력한 기능을 제공함
- Sonar 팀이 직접 제작자에게 연락하여 체험을 요청하였고, 이에 영상 제작자는 적극적으로 해당 툴을 추천함
- 영상 마지막에는 Kinugi 다운로드 링크 안내와 Sonar 팀에 대한 감사 표시가 포함되어 있음

## 세부 요약 - 주제별 정리

### Claude Code는 강력한 개발 도구이지만 출력 리뷰 과정에서 한계가 드러남

- Claude Code는 실제 개발에 광범위하게 활용될 정도로 강력한 도구임
- 영상 제작자는 현재 모든 개발 과정을 Claude Code에 의존할 정도로 자주 사용 중임
- 그러나 Claude Code를 오래, 많이 쓸수록 한 가지 큰 단점을 경험하게 됨
- 바로 각각의 Claude 코드 출력물(아웃풋)을 일일이 검토하는 과정이 번거롭고, 시간 소모적임
- 특히 병렬로 여러 Claude Code 인스턴스를 동시에 운영하는 경우, 어떤 세션에서 어떤 변경이 발생했는지 한눈에 파악하기가 어려움

### Kinugi는 Claude Code의 출력을 시각적으로 한눈에 볼 수 있도록 지원함

- Kinugi(키누기)는 Sonar 팀이 개발한 무료 툴임
- Claude Code를 대체하는 것이 아니라 그 위에 ‘시각적(Visual) 레이어’를 덧씌우는 방식임
- 이를 통해 기존 Claude Code 인스턴스(에이전트)들을 그대로 운영하면서, 추가적으로 Kinugi가 시각적으로 병렬 세션을 한 화면에 보여줌
- “에이전틱 개발 환경(agentic development environment)”이라는 용어로 Kinugi의 컨셉을 소개
- 사용자 입장에서는 모든 Claude Code 인스턴스, 병렬 세션, 실시간 변경 내역 등을 시각적으로 직관적으로 살피고 관리할 수 있게 됨

### Kinugi는 병렬 세션 전반을 통합적으로 관리하며 리뷰 부담을 대폭 경감함

- 사용자는 Kinugi를 통해 어떤 세션이 본인의 리뷰를 기다리고 있는지, 어떤 세션이 아직 실행 중인지를 쉽게 파악가능
- Claude Code가 생성하는 변화 내역 및 “Plan of Attack”도 한눈에 검토 가능
- “Plan of Attack” 내용에 실시간으로 코멘트를 달아 피드백을 줄 수 있으며, Claude Code는 즉각 그 내용을 반영
- 구현이 완료된 후에는 다시 Kinugi 화면에서 ‘풀리퀘스트(Pull Request)’ 형태로 코드를 리뷰할 수 있음
- 기존엔 각 세션의 로그와 파일을 따로따로 뒤져야 했지만, Kinugi 덕분에 훨씬 통합적이고 간편한 경험을 제공

### Sonar 팀의 전문성이 Kinugi에 녹아들어, 코드 품질 및 보안 검증이 강화됨

- Kinugi에는 일반 코드 리뷰외에도 ‘보안(Security) 리뷰’ 기능이 내장되어 있음
- Sonar 팀이 원래 보안·코드 품질 분야에서 잘 알려진 팀임을 반영하는 기능임
- AI가 생성한 모든 코드에 대해 쉽게 추가적인 보안점검을 실행할 수 있어 실사용자에게 장점이 큼
- 영상 제작자 본인 역시 이 보안 리뷰 기능을 크게 만족하며 강조함

### Kinugi는 MacOS에서만 사용 가능하고 아직 실험적이나 상당한 잠재력을 보여줌

- Kinugi는 현재 MacOS에서만 사용 가능하다는 점을 Sonar 측에서 명확히 고지함
- 동시에 공식적으로는 실험적(experimental) 상태이지만, 이미 매우 강력한 성능을 제공
- 이 점 때문에 현재는 무료로 배포하고 있음

### 영상 제작자는 Kinugi 체험 이후 강력 추천 의사를 밝힘

- Sonar 팀이 직접 영상 제작자에게 연락해 Kinugi를 체험해 보고 공유해 달라고 요청함
- 실제로 사용해 본 결과, 영상 제작자는 “정말 멋진 도구”라며 적극 추천하고 있음
- 마지막 부분에서 Kinugi 다운로드 링크 안내와 Sonar 팀에 대한 감사 메시지가 포함됨

## *AI 전쟁이 다시 시작됐다! Sonnet 4.5, DeepSeek V3.2, GLM-4.6 등장* 핵심 요약

- 올 여름에는 그다지 눈에 띄는 새로운 대형 언어모델(LLM)이 출시되지 않았으나, 최근 들어 주요 신작 모델들이 한꺼번에 공개되고 있음
- Anthropic가 어제 공개한 Claude Sonnet 4.5는 최신 AI 코딩 기능에서 상당한 우위를 보이며, 사용 경험상 Opus 4.1이나 GPT5 코드엑스보다 빠름
- Sonnet 4.5는 약간 더 성능이 뛰어나면서도 속도 면에서 크게 앞서 있다는 평가
- DeepSeek 3.2는 기존 모델에 비해 훨씬 빠르고 경량화되었으며, 여전히 강력한 성능을 유지함
- DeepSeek 3.2는 Sonnet 4.5에 비해 수십 배 저렴한 비용으로 사용할 수 있음
- 중국에서도 GLM 4.6이 새롭게 출시되었으며, Sonnet 4.5 이후 하루 만에 발표됨
- 벤치마크 비교 결과, GLM 4.6이 Sonnet 4.5를 상회하는 성능을 여러 영역에서 기록함
- 최근 LLM 시장이 다시 치열해지고, 일련의 신제품 발표로 인해 매우 흥미로우면서 격동적인 시기가 도래했음을 보여줌

## 세부 요약 - 주제별 정리

### AI 대전이 다시 시작되며 신제품 LLM들이 한꺼번에 쏟아지고 있음

- 여름 시즌 동안에는 눈길을 끄는 대형 언어모델 출시가 거의 없었음
- 최근 시점에서 여러 새로운 LLM이 동시다발적으로 발표되고 있음
- "AI Wars are back"이라는 말로 현재 분위기를 설명

### Anthropic의 Sonnet 4.5는 뛰어난 AI 코딩 성능과 빠른 속도를 제공함

- Anthropic이 Claude Sonnet 4.5를 어제(영상 기준) 공식적으로 출시함
- Sonnet 4.5는 AI 코딩 성능에서 "아마도 현재 최고의" 모델로 평가됨
- Cole Medin(제작자 본인)의 테스트 결과, Opus 4.1과 GPT5 코드엑스 대비 약간 더 뛰어남
- 성능뿐 아니라 처리 속도 측면에서 눈에 띄게 개선됨
- 실제 사용 경험에서 Sonnet 4.5가 훨씬 더 빠름을 직접 언급

### DeepSeek 3.2는 저렴하면서도 빠르고 경량화된 LLM으로 주목받음

- DeepSeek의 새 버전 3.2도 동시에 출시됨
- 이번 버전은 기존 모델보다 훨씬 빠르고, 용량이 가벼워진 점이 특징
- LLM 분야에서 "lightweight"와 "powerful"을 동시에 달성했다는 점을 강조
- DeepSeek 3.2의 사용 비용은 Sonnet 4.5 대비 "도저히 비교 안 될 만큼" 저렴함(수십 배 차이)
- 저비용-고성능 LLM의 새로운 선택지로 부상

### 중국발 GLM 4.6, 출시 하루 만에 Sonnet 4.5와 벤치마크에서 경쟁함

- 중국에서 GLM 4.6이 출시되었으며, Anthropic Sonnet 4.5 발표 직후 바로 발표된 점을 언급
- "Anthropic이 모델 발표하면, 중국도 곧바로 신제품 출시"라는 현상을 지적
- GLM 4.6은 등장 직후부터 Sonnet 4.5와 벤치마크 경쟁이 이루어짐
- 여러 벤치마크 지표에서 GLM 4.6이 Sonnet 4.5를 앞서는 결과가 확인됨
- "벤치마크가 모든 것을 보여주지는 않지만" 현재 GLM 4.6의 우위는 주목할 만하다고 언급

### 다수의 신형 LLM이 등장하며 시장 경쟁과 혁신이 가속화됨

- 갑작스럽게 여러 최첨단 LLM이 동시 공개되며, 업계 전반에서 경쟁이 치열해짐
- Sonnet 4.5, DeepSeek 3.2, GLM 4.6 등 각기 강점이 뚜렷한 신제품이 단기간에 쏟아져 나옴
- 다양한 가격, 성능, 전문성을 갖춘 모델들이 본격 경쟁에 돌입
- 벤치마크 지표, 실제 테스트 결과 등 실질적 비교가 등장하고 있음
- 이러한 흐름은 LLM 기술 발전과 시장 혁신 속도가 여전히 매우 빠르다는 신호로 해석됨

### 현재 LLM 분야는 전례없이 흥미롭고 역동적인 국면에 진입함

- 제작자는 "매우 흥미롭고 멋진 시기"라며 LLM 시장 분위기를 요약
- 연이은 신제품 발표와 벤치마크 경쟁이 LLM 분야에 혁신의 물결을 일으키고 있음을 강조
- 단순 하드웨어·성능 경쟁을 뛰어넘어, 실제 활용성과 가격 경쟁력까지 포괄하는 시장이 형성되고 있음을 시사
- LLM 트렌드가 단기간에 바뀌며, 사용자의 모델 선택 폭이 넓어짐을 재차 언급

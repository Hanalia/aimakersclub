## *Anthropic, Claude Sonnet 5를 출시하며 중급 모델에서 Opus 4.8에 거의 근접한 성능과 저렴한 가격을 달성함* 핵심 요약

- Anthropic이 새로운 Claude Sonnet 5 모델을 출시함
- 이번 출시의 핵심은 속도가 아니라, 중급 모델인 Sonnet 5가 SweBench Verified 코딩 벤치마크에서 85.2점을 기록해 Opus 4.8의 성능에 거의 근접함
- Sonnet 5의 가격은 Opus 4.8 대비 훨씬 저렴해 가격 경쟁력이 매우 높음
- Sonnet 5는 이전 Sonnet 대비 에이전트 성능이 크게 향상되어, 저장소 지정·목표 제시만으로도 스스로 계획, 터미널 명령 실행, 복잡한 다단계 작업까지 완료할 수 있음
- Sonnet 4.6 대비 각종 벤치마크에서 큰 도약을 기록: SweBench Verified 85, Terminal Bench 67→80, Browse Comp 76→86 등
- Frontier Code 벤치마크에서 Sonnet 5는 38점을 기록, 4.6의 15점에서 2배 이상 성장
- 이전 Sonnet 모델에서 Sonnet 5로 마이그레이션은 단순히 모델 ID만 변경하면 되어, 추가 작업이 거의 없음
- Opus 4.8이 일부 벤치마크에서는 여전히 앞서 있지만, Sonnet 5는 거의 근접하는 수준에 성능-가격비를 보임
- Sonnet 5의 코딩 기능은 무료, 프로, 맥스, 팀, 엔터프라이즈, Claude 코드, 플랫폼 전체에서 모두 사용 가능
- 8월 31일까지 100만 토큰당 입력 $2, 출력 $10의 가격으로 제공되어 Opus급 코딩을 Sonnet 가격으로 제공
- 영상의 마무리는 Sonnet 5가 세대적 도약인지 더 저렴한 Opus에 불과한지 시청자 의견을 요청하며 끝남

## 세부 요약 - 주제별 정리

### Sonnet 5는 전작 대비 대규모 성능 점프를 달성하며, 중급 모델 위치에서 Opus 4.8 성능에 근접함

- Anthropic이 출시한 Sonnet 5는 기존 중급 모델과 달리 SweBench Verified 벤치마크에서 85.2점을 취득함
- 이 점수는 Opus 4.8과 매우 유사하며, 높은 성능을 시사함
- 가격대는 Opus와 달리 훨씬 저렴해 가성비에서 큰 강점
- Sonnet 5는 속도보다 뛰어난 성능 향상이 핵심임이 강조됨

### Sonnet 5는 이전 버전 대비 에이전트적(Agentic) 성능과 자율 작업 처리 능력이 대폭 향상됨

- 기존 Sonnet 모델은 복잡한 다단계 작업을 중간에 포기하는 경향이 있었음
- Sonnet 5는 저장소(Repository)를 지정하고 목표(Goal)를 주면, 자체적으로 계획(Planning)부터 터미널 명령 실행, 작업 완수까지 독립적으로 처리 가능
- 구체적인 예시로, 사용자가 단순히 목표만 제시하면 Sonnet 5가 연이어 여러 단계를 알아서 실행함

### 각종 벤치마크에서 Sonnet 5는 Sonnet 4.6 대비 두드러진 수치 개선을 기록함

- SweBench Verified: Sonnet 4.6 대비 Sonnet 5가 85점대로 대폭 상승
- Terminal Bench: Sonnet 4.6이 67점, Sonnet 5가 80점으로 두드러진 향상
- Browse Comp: Sonnet 4.6이 76점, Sonnet 5가 86점으로 상승
- Frontier Code 벤치마크에서 Sonnet 4.6의 15점에서 5는 38점으로 2배 이상 뛰어올라 "가장 큰 개선폭"으로 언급됨

### Sonnet 5로의 마이그레이션은 매우 간단하며, 개발 비용-노력이 추가되지 않음

- Sonnet 4.6에서 Sonnet 5로의 이전은 코드에서 "모델 ID" 한 글자만 바꿔주면 가능함
- 추가 코드 수정이나 환경 변경이 거의 필요 없음
- 이는 개발자 및 팀에 큰 이점을 제공함

### Opus 4.8이 여전히 일부 벤치마크(코딩 등)에서 앞섬에도 불구하고, Sonnet 5는 대부분의 실사용 환경에서 이에 육박함

- Opus 4.8이 한두 벤치마크에서 Sonnet 5를 앞서는 점은 인정됨
- 하지만 주요 코딩 능력은 Opus에 "아주 근접"한 수준에 도달
- Sonnet 5가 제공하는 성능이 실질적으로 Opus 대신 사용해도 큰 차이가 없을 정도로 확인됨

### Sonnet 5의 향상된 코딩 성능은 Anthropic 전 라인업에 폭넓게 제공됨

- Sonnet 5의 코딩 기능은 무료(free), 프로(pro), 맥스(max), 팀(team), 엔터프라이즈(enterprise), Claude 코드, 플랫폼 등 Anthropic의 거의 모든 체계에서 제공 예정
- 별도의 추가 제품이나 서비스 구매 없이 동일한 성능을 다양한 요금제에서 누릴 수 있음

### Sonnet 5는 2024년 8월 31일까지 파격적인 가격에 제공됨

- 100만 토큰 기준 입력 $2, 출력 $10의 가격 정책을 적용
- 기존 Opus 성능에 가까운 코딩 품질을 Sonnet 가격으로 이용 가능
- 이 특별가는 8월 말까지 한시적으로 제공됨

### 영상을 제작한 Cole Medin은 Sonnet 5의 진정한 세대적 도약 여부에 대해 시청자 의견을 구하며 마무리함

- Sonnet 5가 엄청난 발전인지, 혹은 그냥 저렴한 Opus에 불과한지는 사용자 평가에 맡긴다는 메시지로 종료
- 별도의 해석, 평가 없이 사실 위주로 정보를 전달함

### Dynamis라는 회사에서 Sonnet 5 활용 에이전트 코딩 워크숍을 주기적으로 운영함을 안내함

- 본 영상 진행자는 에이전트 코딩 관련 실전 워크숍을 Dynamis 내부에서 주간 운영 중임을 언급
- 실제 Sonnet 5 같은 최신 모델 코딩 적용 방법을 배울 수 있음을 안내함

### Sonnet 5 출시에 따른 산업현장·개발현장의 변화 가능성 시사

- Opus와 유사한 성능의 미드레인지 가격 모델 등장으로, 대규모 서비스 개발·배포에서 비용 효율적 코딩 AI 선택지가 등장
- 별도의 제품이나 추가 통합 없이, 기존 API 사용자가 모델 ID만 교체하면 새 성능을 곧장 적용 가능
- 코딩 AI 경쟁 구도에서 Anthropic Sonnet 5가 확보한 가격-성능 이점 강조됨

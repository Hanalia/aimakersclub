## *내 클로드 코드가 내 컴퓨터 건드리지 않고 GitHub 이슈를 자동 수정함(무료 툴!)* 핵심 요약

- Cole Medin은 자신의 컴퓨터에서 코드 한 줄도 쓰지 않고 GitHub Pull Request를 생성했다고 밝힘
- AI 코딩 에이전트가 원격 컴퓨터에서 이슈 자동 처리: 사용자는 GitHub Issue에 'coder' 태그만 붙임
- 모든 처리는 코더(coder) 플랫폼의 무료 오픈소스 툴 "Workspaces"를 이용해 클라우드에서 실행됨
- Workspaces는 AI 코딩 에이전트 세션을 대규모로 원격 관리·오케스트레이션해주는 인프라 역할을 담당
- 워크스페이스 인프라는 개인 컴퓨터 대신 VPS(가상 사설 서버)에 Self-hosting으로 구축 가능
- GitHub Issue에 'coder' 태그를 추가하면, Workspaces가 자동으로 작업을 할당·실행
- 각 이슈별로 독립된 작업 환경이 생성되고, 클라우드 코드가 원격으로 문제를 처리하여 PR까지 자동 생성
- 작업 진행 상황은 Issue에 자동 코멘트와 추적 링크로 기록됨
- 이 구조 덕분에 여러 이슈를 동시에 병렬로 작업 가능하며, 팀 단위 공동 활용도 가능
- 사용자는 오로지 태깅만 하면 되고, 실제 코드 작성·이슈 해결·PR 생성은 모두 클라우드에서 자동화됨
- 최종적으로 GitHub에는 자동으로 Pull Request가 제출되어 리뷰 및 병합만 남음
- Workspaces는 무료이므로 누구나 바로 시도해볼 수 있다는 점을 강조함

## 세부 요약 - 주제별 정리

### 클라우드 AI 에이전트가 내 컴퓨터 접속 없이 GitHub 이슈를 자동 처리함

- Cole Medin은 영상에서 자신의 로컬 환경에서 아무 작업 없이 GitHub Pull Request를 생성했다고 밝힘
- 모든 프로세스가 완전히 자동화되어 있었으며, 사용자는 직접 코드 한 줄도 작성하지 않음
- 원격 클라우드 컴퓨터에서 AI 코딩 에이전트가 이슈를 읽고, 수정 및 PR을 제출함

### GitHub 이슈에 태그만 추가하면 원격 코드 자동 수행

- 이슈에 ‘coder’라는 태그만 부여함으로써 전체 자동화 프로세스가 시작됨
- 별도의 설정이나 추가 입력 없이, 태그가 자동 트리거로 작동하여 AI가 이슈를 처리

### Workspaces 툴이 AI 코딩 에이전트 인프라 역할을 수행함

- ‘Workspaces by the platform coder’라는 무료 오픈소스 툴을 활용함
- 이 툴은 자체적으로 코딩 에이전트가 아니라, 여러 코딩 에이전트 세션을 관리하는 오케스트레이션 레이어임
- 사용자는 Workspaces를 VPS(가상 사설 서버) 등에 Self-hosting하여 인프라를 구축함
- 이 인프라를 통해 remote하게 AI 에이전트를 관리 및 확장 운영할 수 있음

### 여러 코딩 에이전트 세션을 병렬로 운영 가능함

- Workspaces 인프라는 각 GitHub 이슈마다 독립된 환경을 만들 수 있음
- 덕분에 여러 건의 이슈 처리 작업이 동시에 병렬로 진행 가능하며, 팀 단위로 협업도 가능
- 예를 들어, 수십 건의 GitHub 이슈를 동시에 처리하여 생산성을 극대화할 수 있음

### 이슈 처리 상황이 GitHub 이슈 코멘트와 링크로 실시간 추적 가능함

- Workspaces에서 태그를 추가하고 나면, 자동으로 해당 이슈에 작업 진행상황 링크가 코멘트로 남음
- 사용자와 팀은 해당 링크를 통해 클라우드 작업의 구체적 상태를 실시간으로 확인할 수 있음

### 전체 처리 과정은 완전히 원격에서 자동화됨

- Cloud code가 원격 환경에 자동으로 스핀업되고, 이슈 분석·수정 코드 작성·PR까지 모두 처리함
- 사용자는 아무것도 직접 실행하거나 환경을 설정할 필요 없이, 최소한의 인터렉션(태그 추가)만 필요

### 작업 완료 후 자동으로 Pull Request 생성 및 리뷰에 노출됨

- 작업이 완료되면 자동으로 PR이 생성되어 GitHub에 제출됨
- 사용자는 PR을 검토하고 병합만 하면 됨으로써 전체 개발 프로세스가 대폭 단순화됨

### 팀 단위로 Workspaces 인프라를 공유하며, 효율적 협업이 가능함

- Workspaces 인프라는 특정 개인이 아니라 팀 단위로도 공유 및 활용 가능
- 모든 작업 환경이 격리되어 있어, 여러 사람이 동시에 별도 이슈를 무리 없이 다룰 수 있음

### 무료 툴로 누구나 즉시 시도할 수 있음을 강조함

- Workspaces는 무료이자 오픈소스이므로, 비용 부담 없이 직접 구축·테스트 가능함
- 영상 하단 설명란에 관련 링크를 안내함

### 로컬 환경이 아닌 외부 인프라에서 코드/보안/확장성 측면 혜택을 누림

- 자체 PC나 노트북에 코딩 에이전트를 띄우지 않아도 되므로 리소스 부담 감소
- 각 작업이 격리되어 있어, 보안이나 오류 전파 위험도 낮음
- 인프라 리소스 조절, 병렬 확장 등 대규모 운영 편의성을 제공함

## *MCP 서버의 심각한 문제점과 그 해결책* 핵심 요약

- 영상 제목은 "MCP 서버의 심각한 문제점과 그 해결책"으로 해석됨
- MCP 프로토콜이 인기 상승과 함께 치명적인 약점을 드러내고 있음을 강조
- 주요 문제는 토큰 소비량 과다와 컨텍스트 소실(context rot) 현상임을 지적
- 영상은 MCP의 실질적 문제점을 예시를 통해 상세히 설명
- 표준적인 5개의 MCP AI 코딩 서버가 각각 AI 에이전트에게 기능 설명만 해도 수천 개의 토큰을 소모한다고 언급
- 여러 MCP 서버를 동시에 사용하면 토큰 비용이 폭증하여 실제 세션에서 비효율적임을 경험적으로 설명
- 이 문제의 해법으로 최근 Anthropic이 공개한 '에이전트 스킬' 기능을 소개
- '에이전트 스킬'은 AI가 API 엔드포인트 조작, 사용 방법 자동 생성, 즉석 코드 작성 등 다양한 스크립트 생성 및 실행을 지원
- 새로운 접근법이 MCP 문제를 근본적으로 해결함을 강력하게 주장
- 마지막 질문으로 “이제 우리가 알던 MCP는 끝인가?”라는 문제 제기를 통해 영상 마무리

## 세부 요약 - 주제별 정리

### MCP 프로토콜은 인기와 함께 치명적인 단점을 드러내고 있음

- MCP(Multi-Component Protocol)는 최근 인기도와 관심이 크게 증가한 프로토콜임
- 그러나 이 프로토콜에는 치명적인 문제점이 내재되어 있음
- 그 중에서도 ‘토큰 소비’와 ‘컨텍스트 소실(context rot)’이 최근 가장 큰 화두임을 강조
- 영상의 목표는 MCP의 구체적인 문제와 유력한 해결책을 다루는 데 있음

### 토큰 소비 증가와 컨텍스트 소실이 MCP의 가장 큰 문제임

- MCP에서 여러 서버(에이전트)를 운영할 때 각 서버는 AI 에이전트에게 자신의 기능, 역할, 스펙을 전달해야 함
- 이 정보 전달에만 서버 하나당 수천 개의 토큰이 소모됨을 지적
- 여러 MCP 서버를 한 세션에서 동시에 활용할 경우 토큰이 기하급수적으로 늘어나는 심각한 문제 발생
- 이 과정에서 컨텍스트 한계 내에 정보를 다 담지 못하기 때문에 점점 과거 정보가 소실(conext rot)됨

### 실제 5개의 표준 MCP 서버 사용 사례로 비효율성을 강조함

- 코딩 지원을 위한 MCP 서버 5개를 실제로 오랜 기간 사용해왔음을 언급
- 그러나 같은 세션에서 5개 서버를 동시 사용하면 AI에게 각 서버의 기능 설명만 해도 어마어마한 토큰이 필요
- “절대로, 정말 절대로!” 이들 서버를 한 세션에서 모두 사용하지 않는다고 강조
- 이로 인해 한계 상황에 빠지는 실질적 예시로 사용자 경험을 전달

### 문제의 핵심은 ‘토큰 비용’과 ‘컨텍스트 부하’로 요약됨

- 여러 서버의 기능과 설명을 컨텍스트에 넣어야 할 때 토큰 한도가 쉽게 소진됨
- 이는 AI의 상황 인식력 저하와 명령 정확도 감소, 비용 증가로 이어짐
- 이 현상이 MCP의 구조적 문제로 고착화되어 있다는 점을 다시 한 번 강조

### Anthropic의 ‘에이전트 스킬’이 MCP 문제의 결정적 해결책임을 소개함

- 최근 Anthropic에서 ‘Agent Skills(에이전트 스킬)’ 기능을 출시했다고 소개
- 이 기능이 MCP의 토큰 및 컨텍스트 문제를 원천적으로 해결한다고 평가
- ‘에이전트 스킬’은 AI가 필요한 순간에 API 엔드포인트와 상호작용할 스크립트를 직접 생성함
- 사용자는 AI가 자동으로 작업 명령, 코드 생성, 사용법 명령 등을 작성하도록 할 수 있음
- 코드 실행(code execution)을 매우 효율적으로 구현하게 해주는 혁신적 기능

### ‘에이전트 스킬’ 방식은 코드 자동 생성·실행까지 지원함을 강조

- 기존에는 각 서버가 정적인 설명만 전달했다면,
- 이제는 AI가 동적으로 API, 명령, 스크립트 등을 필요 시 생성/수정/실행할 수 있음
- “코드 실행의 절정(code execution at its finest)”이라는 표현으로 성능을 극찬
- 스킬 기반 방식이 서버 관리와 에이전트 활용의 패러다임을 바꿀 수 있음을 시사

### 이번 변화가 ‘우리가 알던 MCP의 종말’이 될 수 있다는 질문으로 마무리함

- “이것이 우리가 알던 MCP의 끝인가? (Is this the end of MCP as we know it?)”라는 질문을 제기
- Anthropic의 신기능 도입이 MCP 이용 방식 전반에 근본적 변화를 줄 수 있다는 점을 은연중에 강조
- 영상 내용의 핵심 메시지를 이 종합적 질문에 담아 마무리

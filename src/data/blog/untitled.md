## *이 무료 도구로 Claude 코드가 10배 더 좋아진다* 핵심 요약

- 영상은 개발자가 Claude Code 사용 중 마주치는 코드 리뷰의 비효율성과 병렬 세션 관리 어려움을 지적함
- Claude Code는 매우 강력하고 본인의 모든 개발에 사용될 정도로 만족도가 높지만, 결과물 리뷰가 번거롭다는 점을 단점으로 꼽음
- 특히 여러 개의 Cloud Code 인스턴스를 동시에 실행 시, 각각의 세션의 상태와 코드 변경사항을 추적하고 리뷰하는 것이 불편함
- 이러한 불편함을 해결해주는 무료 도구로 Sonar 팀이 개발한 “Kinugi(기누기)”를 소개
- Kinugi는 Claude Code를 대체하는 도구가 아니라, 그 위에 시각화된 계층(visual layer)을 추가하여 모든 에이전트 세션을 한눈에 보여주는 역할을 함
- Kinugi에서는 실행 중인 세션, 리뷰가 필요한 부분, AI가 생성한 작업 계획 등을 실시간으로 확인, 주석 및 피드백 제공이 가능
- 구현 완료 후엔 전체 변경사항을 마치 '풀 리퀘스트' 보듯 한 화면에서 손쉽게 리뷰할 수 있음
- Kinugi에서는 AI 생성 코드에 일반 리뷰뿐만 아니라, Sonar답게 보안 리뷰(Security Review) 기능도 추가 적용할 수 있음
- 현재 Kinugi는 Mac OS 한정, 매우 실험적 단계이기 때문에 무료로 제공됨
- 영상 제작자는 Sonar의 요청을 계기로 Kinugi를 체험하고 적극적으로 추천하며, 링크 공유로 시청자에게 사용을 권유함

## 세부 요약 - 주제별 정리

### Claude Code는 개발 생산성을 크게 높이지만 코드 리뷰에서 한계를 드러냄

- Claude Code는 AI기반 코딩 지원 도구로, 영상 제작자가 개발의 모든 과정에 사용할 만큼 강력함
- 다양한 변경 작업과 코드 생성을 빠르고 효율적으로 수행하는 ‘AI 코드 에이전트’를 배치 가능
- 하지만 Claude Code로 생성된 결과물(review된 코드)을 사람이 일일이 검토해야 한다는 ‘후처리’ 부담이 남음
- 병렬적으로 여러 Claude Code 인스턴스를 동시에 운영하면, 각각의 산출물을 따로 관리·리뷰해야 해 비효율적임

### 병렬 세션을 한눈에 관리·리뷰하는 것이 기존 Cloud Code에서는 불가능했음

- 영상 제작자의 실제 사례로, 다수의 Cloud Code 세션을 복수로 항상 실행하며 작업 진행
- 각각의 인스턴스 결과물이 Cloud Code 로그와 파일 산출물로 흩어져 있어, 수동으로 확인하고 리뷰하는 과정이 번거로움
- 이로 인해 진행 상황 파악, 어떤 세션에 리뷰가 필요하고 완결되었는지 구분하는 것이 어렵고, 시간 소요가 큼

### Kinugi는 Claude Code 위에 시각적 인터페이스를 더해 병렬 작업을 한 번에 관리할 수 있게 함

- 무료 도구 ‘Kinugi(기누기)’는 Claude Code의 워크플로우를 그대로 사용하면서, 그 위에 시각적 관리 기능을 얹음
- Claude Code를 대체하거나 편의성만 추가하는 것이 아니라, ‘Agentic Development Environment’(에이전트 기반 개발 환경)라는 개념으로 동작
- 실행 중이거나 완료된 Claude Code 세션을 한 화면에서 동시에 관찰할 수 있음
- 각 세션의 상태, AI가 제안한 행동 계획, 산출물 등에 즉시 접근, 시각적으로 확인 가능

### AI가 생성한 코드 변경 사항을 집약적으로 리뷰하고, 실시간 피드백 기능도 제공함

- Kinugi를 사용하면, 모든 병렬 세션에서 어떤 코드 변경이 일어났는지 집약적으로 파악 가능
- 세션별로 어떤 부분에 리뷰가 필요한지, 무엇이 아직 실행 중인지 바로 표시됨
- AI가 생성한 '플랜 오브 어택(plan of attack: 작업계획)' 단계에 직접 댓글과 피드백을 남길 수 있으며, AI가 실시간으로 이를 반영하도록 함
- 실제 코드 구현 및 반복 작업 이후, 리뷰 항목은 마치 Git의 '풀 리퀘스트'처럼 전체 변경사항이 하나의 화면에 깔끔히 정리되어 리뷰할 수 있음

### 번거로운 로그 확인 및 개별 파일 리뷰 과정을 ‘풀 리퀘스트 중심’ 업무로 대체해 효율이 크게 상승함

- 기존 Claude Code만 사용할 때는 로그(log) 파일과 산출물 파일을 모두 개별적으로 확인하며 리뷰해야 했음
- Kinugi는 전체 변경 이력 및 리뷰 포인트를 일목요연하게 정리된 ‘풀 리퀘스트’ 형식으로 제공함
- 개발자는 각각의 세션, 변경점 등에 쉽고 빠르게 접근하며 종합적으로 검토 가능함

### Sonar 개발팀이 만든 Kinugi는 AI 코드 리뷰에 ‘보안 검토’까지 덧붙이는 차별성을 갖춤

- Kinugi에는 일반 코드 리뷰뿐만 아니라, AI 산출 코드에 대해 보안(Security) 리뷰 필터가 별도 적용됨
- 이는 코드 품질 관리를 중시하는 Sonar(SonarQube, SonarCloud 등) 팀의 노하우가 반영된 기능임
- 보안 리뷰 기능을 통해, AI가 만든 코드의 잠재적 리스크(취약점 등)까지 자동 점검이 가능해짐

### Kinugi는 매우 초기 단계(Mac OS 전용, 실험 버전)이나, 무료로 공개되어 있음

- 현재 Kinugi는 Mac OS에서만 사용 가능하며, 매우 ‘experimental’(실험적) 단계임을 명시
- 무료로 공개되어 있어 누구나 시도할 수 있다는 장점이 설명됨 (유료화 전이거나 시장 반응, 안정성 검증 단계)

### Sonar의 공식 제안으로 제작자가 직접 체험 후 영상 공개를 결정함

- Sonar팀 측에서 영상 제작자에게 Kinugi 활용 제안을 먼저 전달
- 제작자가 실제로 도구를 사용해보고, 실질적인 효용성을 느껴 짧은 영상 제작을 수락함
- 영상 말미에서 제작자는 “진심으로 멋진 도구”라 표현하며, 적극적으로 시청자에게 시도 권장
- 영상 설명란에 Kinugi 무료 다운로드 링크를 함께 공유함

### Kinugi는 기존 Claude Code 워크플로우를 해치지 않고, 시각화·협업·보안 기능을 향상시킴

- Claude Code의 본래 기능(코드 자동 생성, AI 에이전트 병렬 처리 등)을 그대로 유지함
- Kinugi는 상단에 시각화, 변경 관리, 리뷰 도구, 협업(리뷰/댓글 반영) 인터페이스만 추가 제공함
- 개발 생산성은 꾸준히 유지, 병렬 워크플로우의 난이도는 크게 줄어들어 써볼 가치가 충분함

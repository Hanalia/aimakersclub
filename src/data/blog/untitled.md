## *클로드에게 24시간 동안 코딩을 시켜본 결과* 핵심 요약

- 영상에서는 인공지능 모델인 Claude Opus 4.5를 24시간 연속으로 코딩하게 한 실험 결과를 소개함
- Claude는 claw.ai의 완벽한 복제 서비스에 가까운 수준의 기능을 스스로 구현함
- AI는 다양한 과거 대화 내역을 저장하고 반복적으로 검증 및 수정하는 루프를 탑재함
- Markdown 포맷팅, HTML 페이지 생성, 코드 작성 및 실행 같은 고급 기능까지 자체적으로 구현됨
- 테마, 기본 AI 모델, 최대 토큰 수 슬라이더 등 세밀한 설정 기능도 지원함
- 아직 ‘human-in-the-loop’(사람의 최종 검증)이 추가되어야 하지만 현재 상태도 매우 완성도가 높음
- 실험자는 직접 손을 대지 않고도 이 정도로 풍부한 기능을 구현한 결과에 큰 놀라움을 표현함
- 이런 도구와 자동 검증·반복 시스템이 결합될 때 AI로 구축 가능한 어플리케이션의 수준이 매우 높아짐을 사례로 제시
- 만약 AI에게 하루 더 시간을 준다면 완벽한 응용 프로그램이 될 수도 있었을 것이라고 평가함

## 세부 요약 - 주제별 정리

### 클로드 오퍼스 4.5를 24시간 동안 코딩에 투입하자 완성도 높은 응용프로그램이 구축됨

- 실험자는 첨단 인공지능 모델 ‘Claude Opus 4.5’에게 24시간 연속으로 코딩을 하게 함
- 결과물은 claw.ai의 기능을 완벽하게 복제한 수준에 가까운 소프트웨어로, 실질적으로 완벽히 동작함
- 최종 단계의 테스트가 필요 없는 수준으로 보여 실험자가 놀라움을 느낌
- 인공지능의 실시간 반복 개선, 검증 기능이 전과 비교할 수 없을 정도로 향상됨을 체험함

### AI는 대화 기록, 코드 검증 루프 등 고차원 자동화 과정을 학습하고 구현함

- 클로드는 생성된 모든 이전 대화 및 작업 기록을 스스로 관리하고 저장함
- 각 루프(반복 단계)마다 결과를 스스로 검증하고, 문제가 발견되면 즉시 수정하여 다시 반복함
- 코드 검증 및 수정을 완전 자동화하여, 사람 개입 없이 품질을 높임
- 실험 전반에 걸쳐 높은 수준의 자기 점검 및 품질 관리 기능을 보임

### Markdown 포맷, HTML 생성, 코드 실행까지 다양하고 고급스러운 기능을 스스로 구현함

- 클로드는 복잡한 Markdown 포맷팅으로 결과물을 미려하게 출력함
- 다양한 HTML 페이지를 스스로 생성하고 구성할 수 있음
- 소스코드 작성, 실제 코드 실행까지 전 과정을 자동화함
- 사용자의 별도 조작 없이 여러 결과물 포맷을 자유롭게 지원함

### 테마, 모델 변경, 토큰 수 설정 등 UI/UX 세팅이 포함된 관리 기능도 자동 구현됨

- 인공지능이 직접 ‘settings’ 메뉴를 개발하여, 테마 변경이 가능함
- 사용자가 프롬프트를 통해 기본 AI 모델을 다르게 선택할 수 있도록 옵션을 추가함
- 최대 생성 토큰 수를 실시간으로 조절하는 슬라이더를 포함함
- 결과적으로 아주 풍부하고 정교한 기능의 UI를 갖춤

### 인간 검증자를 추가하면 완성도가 더 올라갈 여지가 있지만, 이미 자율적 구현 역량이 매우 높음

- 향후 시스템 내에 ‘human in the loop’(사람의 최종 검토) 단계만 추가하면 된다는 의견 제시
- 그럼에도 현재 모습만으로도 인공지능이 혼자서 많은 것을 구축했음을 확인함
- 사용자가 엔지니어링에 손을 대지 않고도 상당한 수준의 제품 개발이 가능함을 강조함

### 하루 더 시간을 주면 더욱 완벽한 어플리케이션이 생성될 수 있었을 것임을 시사함

- 실험자는 “만약 더 오랜 시간을 투입했다면 정말 완벽한 애플리케이션이 되었을 것”이라고 평가함
- 지금도 이미 상상 이상으로 다양한 기능이 잘 구현되어 있다고 느낌

### 클로드 오퍼스 4.5와 같은 AI의 검증 및 반복 시스템이 향후 앱 개발 방식을 혁신함을 보여줌

- 실험 사례는 첨단 AI(Claude Opus 4.5)와 반복적 테스트·검증 툴(“harness”)이 결합되면 자동화 수준이 매우 높아진다는 것을 입증함
- 완전한 자율 반복(Validation, Iteration) 과정을 통해 빠르고 정확하게 앱을 진화시킬 수 있음
- 인공지능과 소프트웨어 품질관리 도구의 결합이 실제 제품 개발의 생산성을 급격히 높임을 보여줌

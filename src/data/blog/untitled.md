## *Claude에게 24시간 동안 쉬지 않고 코딩을 시킨 결과* 핵심 요약

- 진행자는 Claude AI를 활용해 24시간 동안 자동화된 코딩을 시도함
- 최종적으로 claw.ai를 거의 완벽하게 복제한, 기능적인 애플리케이션을 만들었다고 평가
- 테스트 과정에서 생성된 모든 과거 대화 이력을 손쉽게 확인할 수 있는 구조를 구현
- Markdown 형식의 문서화 기능과 다양한 HTML 페이지 생성 기능이 추가됨
- 실제 코드 작성과 실행이 모두 Claude를 통해 가능했음을 언급
- 테마, 기본 모델, 최대 토큰 수 조절용 슬라이더 등 설정 메뉴도 구현함
- "이 애플리케이션은 정말 기능이 많다(so so featurerich)"고 강조
- 아직 'human in the loop' 등의 추가 기능 도입 여지가 남았음을 언급
- 사용자는 손으로 코딩하지 않고도 엄청난 결과물을 얻었다는 점에 놀라움을 표함
- Claude Opus 4.5와 같은 최신 AI가 자동 검증·반복을 지원하는 프레임워크(테스트 하네스)와 결합될 때, 기대 이상의 수준 높은 애플리케이션 개발이 가능함을 체험함

## 세부 요약 - 주제별 정리

### Claude를 24시간 연속으로 작업시키자 완성도 높은 claw.ai 복제본이 구현됨

- 진행자는 Claude AI에게 24시간 동안 쉬지 않고 코딩을 시켰다고 밝힘
- 그 결과, "완전히 기능적인 claw.ai 클론"을 갖추게 됐다고 평가
- 진행자는 "지난 테스트의 절반이 뭘 위한 건지조차 모르겠다"며, 이미 충분히 수준 높은 결과물임을 강조
- claw.ai의 핵심 기능들이 잘 구현되어 있음을 언급함

### 과거 대화 내역 및 검증 루프 추적 기능이 완벽하게 구현됨

- Claude가 기능별로 검증(loop)하는 과정에서 생성된 모든 과거 대화(conversations)를 한눈에 확인 가능
- 각 검증 단계, 반복 과정에서의 내역을 투명하게 추적할 수 있음
- 개발 및 검증, 개선 과정에 있어 기록과 추적이 매우 용이해졌음을 알 수 있음

### 뛰어난 Markdown 문서화 및 HTML 페이지 생성이 가능해짐

- 애플리케이션은 "정말 멋진(markdown) 문서화" 기능을 갖춤
- 개발된 코드나 설명 내용을 Markdown 형식으로 보기 좋게 포맷할 수 있음
- 다양한 HTML 페이지를 동적으로 생성하는 기능도 Claude가 자동 구현

### Claude가 직접 코드 작성 및 실행까지 지원함

- Claude를 통해 "코드를 작성하고 실제로 실행"하는 과정까지 일괄적으로 처리함
- 사용자는 직접 코드를 입력하거나 수정하지 않고도 원하는 기능이 구현됨
- 수동 개입이 없는 '자동화 코딩'의 탈인간적 효율성을 강조함

### 사용자 편의를 위한 다양한 설정 메뉴가 구현됨

- 테마 선택, 기본 적용할 모델 지정 등 다양한 설정(세팅) 기능이 탑재됨
- 최대 토큰 수를 조절하는 슬라이더도 추가함
- 사용자 인터페이스(User Setting) 측면에서도 디테일을 잘 살림

### “정말 기능 풍부(so so featurerich)”하다고 극찬함

- 진행자는 "so so featurerich"라는 표현을 반복적으로 사용
- 방대한 수의 기능과 디테일, UI, 활용성을 모두 갖춘 결과물임을 강조
- 본인이 직접 손으로 개발하지 않고도 이런 수준 높은 결과물이 나왔다는 점에서 놀라움을 표함

### 향후 추가하고 싶은 ‘human in the loop’ 기능을 언급함

- 지금도 완성도가 높으나, 이후 ‘human in the loop’(사람의 검증/개입 기능)도 추가하고 싶다고 밝힘
- 현재는 거의 자동화로 순환되지만, 인간의 타이밍/판단이 개입될 영역도 존재한다고 시사

### Claude의 지속적인 반복(Iteration)과 검증(Validation)이 품질을 높임

- Claude는 준비된 프레임워크(하네스)에 맞춰, 다양한 루프와 검증, 반복 과정을 거침
- 각 반복에서 논리적 오류, 기능 부족 점을 빠르게 수정하며 진화함
- 이 구조 덕분에 아주 빠른 자동 피드백과 고도화가 가능해짐

### AI의 발전 정도와 개발 환경의 혁신을 실감하게 됨

- "진짜 우리가 Claude Opus 4.5로 만들 수 있는 게 미쳤다(crazy)"며, 최신 AI의 생산성에 감탄
- 이번 시도로, ‘AI+자동화 테스트 프레임워크’만으로도 수준 높은 응용프로그램 제작이 현실화됐음을 확인
- 앞으로 하루만 더 돌리면 "완벽에 가까운 앱"이 될 수 있겠다고 언급하며, AI 발전 가능성에 대한 기대를 내비침

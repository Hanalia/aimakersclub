## *Claude에게 24시간 동안 쉬지 않고 코딩을 맡긴 결과* 핵심 요약

- 영상에서는 Claude Opus 4.5 AI에게 24시간 내내 자동화된 코딩을 맡겨, 실제로 어떤 결과물이 만들어졌는지 시연함
- 실험 결과 기존 claw.ai 서비스를 완전히 복제한 듯한, 전체적으로 완성도 높은 클론 앱을 제작하는 데 성공함
- 개발 과정에서 AI가 자체적으로 수많은 테스트와 반복 점검(loops)을 수행하며 코드를 검증함
- 챗 기록(과거 대화 기록) 탐색, 검증 과정 확인, 마크다운 포맷팅 등 다양한 기능이 자연스럽게 구현됨
- HTML 페이지 생성, 코드 작성 및 실행까지 자동화가 가능하며, 결과물에 다양한 고급 기능(테마 변경, 기본 모델 설정, 토큰 슬라이더 등)까지 탑재됨
- 본인은 손을 거의 대지 않고도 이처럼 다양한 기능성을 갖춘 앱이 만들어진 점이 매우 인상적이라고 평가함
- 아직 "human in the loop"(사람의 개입)를 추가하고자 하는 의지는 있으나, 이미 현재 결과물만으로도 놀랍다고 언급
- Claude와 같은 첨단 모델과 자동화된 검증·반복 시스템의 결합이 실제 애플리케이션 개발의 수준을 비약적으로 높인다는 점을 시사함

## 세부 요약 - 주제별 정리

### Claude Opus 4.5를 이용한 완성도 높은 claw.ai 클론 앱 제작에 성공함

- 24시간 동안 Claude Opus 4.5 AI에게 코딩을 맡기자 claw.ai의 모든 주요 기능을 모방한 앱이 완성됨
- 개발자는 "무엇을 위해 남은 테스트들이 존재하는지 모를 정도"로 이미 완전히 작동하는 앱이라고 평가함
- 실제로 claw.ai의 기능적 복제가 이루어져, 완성도가 매우 높았음을 강조함

### 자동화된 테스트와 반복 루프를 통해 기능을 신속히 검증‧구현함

- Claude가 개발하는 동안 수차례에 걸쳐 자동 테스트와 반복 검증(loops) 절차를 거침
- 각 루프마다 과거 대화 내역, 코드 실행 결과 등을 자동으로 기록하고 관리함
- 사용자는 이러한 과정을 한눈에 파악할 수 있도록 설계함

### 대화 기록 탐색, 검증, 마크다운 포맷 등 상세 기능이 자연스럽게 갖춰짐

- 앱에서는 과거 챗 기록을 쉽게 조회할 수 있도록 구현됨
- AI가 기능 검증 과정에서 생성된 대화를 전부 저장, 필요 시 확인이 가능함
- 마크다운(markdown)으로 결과물과 코드가 깔끔하게 포맷팅되어 보여짐

### HTML 페이지 생성 및 코드의 작성/실행이 자동화됨

- Claude가 직접 다양한 HTML 페이지를 생성하는 기능을 구현함
- 사용자 요청에 맞는 코드를 즉시 작성하고, 곧바로 실행까지 가능케 만듦
- 결과적으로 AI가 코딩, 테스트, 실행, 결과확인을 모두 자동화함

### 고급 사용자 인터페이스 옵션(테마, 모델 선택, 토큰 수 조절 등)까지 반영됨

- 앱에는 테마(색상 등) 변경 기능을 비롯하여, 기본 사용할 AI 모델 선택 기능이 탑재됨
- 토큰 수를 지정하는 슬라이더도 제공되어, 사용자가 직접 최대 토큰 수를 조정 가능함
- 사용자 편의성과 커스터마이즈가 모두 반영된 인터페이스로 완성됨

### "Human in the loop"는 아직 미구현이나 추가 여지가 있음을 언급함

- 개발자는 앞으로 사람이 개입할 수 있는 "human in the loop" 시스템을 도입하고 싶다고 밝힘
- 하지만 현재 상태만으로도 AI 혼자 만든 결과물이 기대 이상임을 언급

### 개발자가 직접적으로 코딩에 관여하지 않고도 이 결과를 얻었음을 강조함

- 본인은 코드 작성에 손 하나 대지 않았다고 반복 강조함
- 순수하게 Claude의 AI 엔진과 자동화 시스템만으로 이정도의 완성도를 달성함

### 자동화된 harness(실행·검증 도구)와 Claude의 결합이 대규모 개발 자동화를 구현함

- 개발 과정 내내 애플리케이션 검증과 개선을 자동 반복하는 harness(자동화 도구)를 활용함
- Claude와 harness의 결합으로 기능 검증, 코드 고도화, 반복 개선이 자체적으로 이루어짐
- 이같은 방식이 현존 AI 개발 도구의 강력함과 가능성을 여실히 보여줌

### 하루만 더 돌릴 경우 더욱 완벽한 애플리케이션 제작이 기대됨

- 만약 Claude에게 하루만 더 시간을 주면 모든 기능을 완전히 구현한 완벽한 앱이 될 수도 있다고 평가함
- 현재 수준만으로도 AI 자동화의 위력을 실감할 수 있으며, 추가 확장성도 높음을 언급함

### Claude Opus 4.5가 실질적인 폭넓은 애플리케이션 제작 도구임을 시사함

- Claude Opus 4.5의 활용만으로 기능이 방대한 애플리케이션을 신속히 제작 가능함을 실험으로 확인함
- 실험적 개발이지만 실제 사용 가능한 클론 앱 개발에 성공함으로써 상용화 수준임을 보여줌

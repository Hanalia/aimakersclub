## *Claude Code로 아름다운 다이어그램을 만드는 전체 워크플로우* 핵심 요약

- 영상 제목은 "Claude Code로 아름다운 다이어그램 만들기(전체 워크플로우)"로, 코딩 에이전트(특히 Claude Code)를 활용해 시각적 다이어그램을 효율적으로 생성하는 과정을 다룸
- 코딩 에이전트들은 기본적으로 비주얼을 잘 다루지 못하며, 시각적으로 설명하려면 많은 프롬프트 및 세팅이 필요함
- 제작자는 자신의 전체 워크플로우를 ‘Excalra Diagram Skill’이라는 형태로 패키지화하여, 코드 에이전트에 시각적 논증 능력을 부여함
- 이 워크플로우는 Claude Code가 Excal(Excalidraw 형태) 다이어그램용 JSON을 생성하고, 이를 실제로 렌더링하며, PNG 스크린샷을 통해 결과를 자체 검증하고 반복적으로 개선함
- 코드 에이전트에 ‘Excalra Diagram Skill’을 적용하면 다이어그램의 구성, 색상, 브랜드 일치성 등 커스터마이징도 가능함
- 사용자는 코딩 에이전트에게 Excalra 다이어그램 스킬 셋업을 직접 요청할 수 있고, 에이전트는 README를 읽고 대부분의 셋업 과정을 스스로 진행함
- 워크플로우의 핵심은 ‘시각적으로 논증하는 방법’을 코딩 에이전트에 가르치는 강력한 프롬프트(명령어)임
- 기존 자동 생성 다이어그램은 단순 블록 나열이 되기 쉽지만, 이 스킬은 구조적 설명 및 라벨링을 중시함
- Excalra Diagram Skill을 활용하면 PDF, 코드베이스 등 다양한 입력에서 맞춤형 다이어그램을 빠르게 만들 수 있음
- 이 방식은 매주 수 시간씩 절약될 만큼 생산성을 크게 향상시키며, 누구나 리포지토리를 클로닝해 즉시 활용 가능함

## 세부 요약 - 주제별 정리

### 코딩 에이전트는 기본적으로 시각적 설명에 약점을 보임

- Claude Code 등 코딩 에이전트는 텍스트 중심이어서 시각적으로 정보를 설명하거나 다이어그램을 그리는 데 한계가 있음
- 단순히 “다이어그램 그려줘”와 같은 지시만으로는 원하는 결과가 잘 나오지 않음
- 시각적 설명 과정은 많은 추가 프롬프트와 스킬 셋팅이 필요함을 강조함
- 사용자는 타인에게 설명할 때나, 자신이 이해를 돕기 위해 비주얼을 원하지만, 기본적으로 에이전트는 이러한 요청을 잘 처리하지 못함

### Excal 다이어그램 생성 및 검증을 위한 전용 워크플로우 패키지화

- 제작자는 자신의 전체 다이어그램 생성/관리 워크플로우를 하나의 스킬로 패키지화함
- 이 스킬(Excalra Diagram Skill)은 Claude Code 뿐 아니라 다른 코딩 에이전트에도 적용 가능함
- 워크플로우 내에서 가장 중요한 요소는 ‘시각적으로 논증할 수 있는 능력(ability to argue visually)’을 에이전트에 부여하는 것임

### 다이어그램 생성용 JSON 파일 및 Excalidraw 연동 구조

- Excalra Diagram Skill의 핵심 명령어 세트(primary instruction set)는 Claude Code가 Excal 다이어그램(Excalidraw 형식)용 JSON 파일을 생성하는 데 집중되어 있음
- 에이전트가 생성한 JSON은 Excal을 통해 시각적 이미지를 만들어냄
- Excalidraw의 특징을 적극적으로 활용하며, 코드와 비주얼 간의 연계를 구현함
- 실제 데모 예시에서는 Claude Code가 JSON을 만들어 다이어그램을 그리고, 이 다이어그램이 Excal에서 렌더링되는 과정을 볼 수 있음

### 자동화된 다이어그램 결과의 시각적 검증 및 반복 개선

- 이 워크플로우의 차별점은 ‘시각적 결과의 자동 검증(self-validation)’ 과정임
- Claude Code가 다이어그램을 그리고, 결과물을 PNG로 스크린샷을 찍은 후, 이미지 자체를 평가하여 오류 또는 불완전한 부분을 스스로 찾아냄
- 필요시 이러한 결함들을 기반으로 다이어그램을 반복적으로 수정·개선함
- 이 과정 덕분에 “insane results”, 즉 매우 우수한 시각적 완성도를 경험할 수 있다고 언급됨

### 설정 및 셋업의 간소화로 누구나 쉽게 사용 가능함

- Excalra Diagram Skill은 셋업이 매우 쉽도록 설계됨
- 사용자는 Claude Code 등 코딩 에이전트에게 “Excalra 다이어그램 스킬을 설치하고 준비해 달라”는 일반적인 요청만 하면 됨
- 에이전트는 README 문서를 참고해 대부분의 설치, 설정, 환경 구축(“manual steps”)을 스스로 처리함
- 최종적으로 사용자는 별다른 추가 작업 없이 필요한 다이어그램 제작 요청을 바로 할 수 있음

### 맞춤형 다이어그램 제작(브랜드, 색상 등)도 자유롭게 가능

- 스킬을 이용하면 사용자의 브랜딩(특정 기업/개인 스타일 반영), 선호 색상, 구조 등 맞춤 설정이 가능함
- 제작자는 다이어그램 커스터마이징이 “가장 중요한 기능” 중 하나라고 강조함
- 예시에서는 단순 다이어그램 생성뿐 아니라, 사용자가 직접 좋아하는 색상이나 스타일로 결과물을 만드는 법도 다룸

### 코드 에이전트의 프롬프트 디자인은 ‘시각적 논증’ 교육에 집중함

- skill.mmd 내 강력한 프롬프트 세트를 통해 “시각적으로 논증하는 법(How to argue visually)”을 에이전트에 효과적으로 주입함
- 단순한 블록(사각형 나열) 다이어그램 생성에서 벗어나, 구조의 의미, 라벨링, 개념의 관계 설명까지 포괄적으로 지시함
- “Blocky”하게 그리는 기본 다이어그램(excal diagrams without this skill)은 지양
- 목적은 구조적이고 논리적으로 설계된 다이어그램 자동생성임

### 다양한 활용(예: PDF, 코드베이스 등) 및 직접 테스트 권장

- Excalra Diagram Skill은 PDF 문서, 코드베이스 등 다양한 대상에 적용 가능함
- 영상 속에서 “지금 바로 PDF, 코드베이스 등에 적용해보라”는 직접적 권장 멘트 제시
- 리포지토리를 클론(clone)하고 스킬을 연동하는 것만으로 곧바로 사용 가능함을 강조

### 업무 생산성 향상 및 시간 절약의 구체적 효과

- 본인의 경험에 의하면 기존 방식보다 다이어그램 제작 시간이 매주 여러 시간씩 절감됨
- 자동 다이어그램 생성을 빠르게 반복하며, 항상 시작점을 마련해주는 역할로 매우 유용함
- 반복 수작업이 줄어듦에 따라 본인뿐만 아니라 모두가 큰 시간 절약이 가능함을 언급

### 리포지토리 공개 및 즉각적 적용 가능성을 안내함

- Excalra Diagram Skill 리포지토리는 누구나 클론할 수 있도록 오픈되어 있음
- 자세한 사용법과 프롬프트 세트는 README 등 문서에서 확인 가능
- 스킬 적용·셋업 과정은 높은 자동화 수준으로, 개발 지식이 많지 않은 사용자도 쉽게 접근할 수 있음

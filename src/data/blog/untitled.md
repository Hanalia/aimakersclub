## *Claude에게 24시간 동안 쉬지 않고 코딩을 시킨 결과* 핵심 요약

- 영상은 Claude Opus 4.5 인공지능에게 24시간 동안 휴식 없이 코딩을 시켜 웹 애플리케이션을 개발하는 과정을 보여줌
- 기존의 claw.ai를 완전히 복제한 듯한 수준의 기능을 구현하는 데 성공했다고 설명함
- 개발 과정 중 생긴 대화 기록, 각 검증 단계별 루프 결과 등도 확인 가능하여 추적성이 우수함
- 앱은 마크다운 포맷이 잘 적용되어 있고, 다양한 HTML 페이지도 생성할 수 있으며, 코드 작성과 실행 기능도 갖춤
- 설정 메뉴에서 테마, 기본 모델, 최대 토큰 수를 조절할 수 있는 슬라이더 등 고급 기능이 포함됨
- AI가 만든 소프트웨어임에도 인간의 개입 없이 매우 많은 기능을 신속하게 구축하는 데 놀라움을 표현함
- 프로젝트에 ‘Human-in-the-loop’(사람의 검증) 기능을 추가할 계획도 언급됨
- Claude Opus 4.5와 같은 AI와 자체 검증-반복(harness) 체계를 활용하면, 완성도 높은 앱을 거의 자동으로 만들 수 있음을 시사함
- 만약 24시간이 아니라 하루만 더 주어진다면 완벽한 수준의 애플리케이션도 만들 수 있겠다고 평가함
- 영상은 최신 AI의 자동화된 개발 능력과 소프트웨어 생산성 향상을 실제 사례로 증명함

## 세부 요약 - 주제별 정리

### Claude Opus 4.5로 기존 claw.ai의 모든 기능을 성공적으로 복제함을 강조함

- 24시간 AI 코딩 실험의 결과, 개발자는 “이미 완전히 작동하는 claw.ai의 복제본(clone)을 갖고 있는 것 같다”고 언급함
- claw.ai는 인기 있는 AI 대화 앱 중 하나이며, 기존 서비스 수준의 기능 복제가 이번 실험의 주요 목표였음
- 성공적으로 복제하며 기존 제품 못지않은 사용성과 기능성을 갖추게 됨
- 영상 내내 해당 복제의 수준과 완성도에 대한 높은 만족감을 반복적으로 표현

### AI가 생성한 모든 개발 기록과 검증 과정을 손쉽게 추적할 수 있게 됨

- Claude가 여러 번의 루프를 거치며 기능을 코드로 구현할 때, 그 과정에서 생성된 과거 대화 내용과 이력을 모두 확인 가능
- 각 기능별 자동 검증 루프마다 어떤 대화와 코드 생성이 있었는지 일목요연하게 정리돼 있음
- 개발 진행 상황 및 AI 의사결정 과정을 투명하게 파악할 수 있도록 설계됨

### 마크다운 포맷, HTML 페이지, 코드 실행 등 고급 기능이 자연스럽게 포함됨

- 앱의 인터페이스와 출력 결과에 “아주 훌륭한 마크다운 포맷팅”이 적용됨
- 다양한 HTML 페이지 생성이 가능하며, 이를 통해 사용자 경험의 폭이 넓어짐
- 코드 작성 및 실행 기능까지 탑재되어 IDE(통합 개발 환경) 형태의 활용도 가능해짐

### 사용자 설정 옵션이 풍부해 실용성과 유연성을 동시에 확보함

- UI 내 ‘설정(Settings)’ 영역에서 다양한 기능 조정이 가능
- 테마 변경, 기본 AI 모델 선택 기능이 포함됨
- 최대 토큰 수 조절용 슬라이더 등 상세한 리소스 제어 옵션까지 갖춤
- 이러한 고급 설정이 자동으로 구현된 점을 AI의 큰 강점으로 부각함

### 인간의 개입이 없어도, AI만으로도 충분히 강력한 애플리케이션을 빠르게 구축함

- “아무것도 손을 대지 않고도 이렇게까지 많은 기능이 구현된 것이 정말 멋지다”고 직접 언급
- AI 코드 생성이 실무 자동화를 획기적으로 가속화할 수 있음을 실제로 입증
- 완전 자동 AI 코드 기반 개발의 미래 가능성을 시사

### 향후 ‘Human-in-the-loop’ 기능으로 AI와 사람의 협업을 더할 계획임을 시사함

- 앞으로 “사람 개입(human in the loop)” 기능을 추가해 인간의 검증 및 교정 단계를 넣는 확장 의도 밝힘
- 현재 수준도 이미 인상적이지만, 사람의 점진적 관여로 완성도를 더욱 높일 계획임을 시사함

### 하루를 더 주었을 때 더 완벽한 앱 제작이 가능함을 자신 있게 언급함

- “만약 이 이상의 시간이 주어진다면 정말 완벽에 가까운 애플리케이션이 완성될 것”이라고 평가
- Claude Opus 4.5의 잠재력과 이미 구현된 성능에 대해 확신을 드러냄

### Claude Opus 4.5와 자동화된 반복 검증 체계(harness)의 결합이 혁신적 성과를 가능하게 함

- Claude Opus 4.5와 자체 검증-반복 구조(하니스 하네스 harness)의 유기적 결합이 효율적이고 빠른 개발을 가능하게 함
- AI가 만든 결과물을 harness가 자동 검증하고, 다시 개선하는 반복 구조로, 앱의 완성도가 빠르게 올라감
- 이런 워크플로우가 차세대 소프트웨어 개발 패러다임의 잠재력을 보여주는 사례임

### 실험 결과가 현대 AI의 소프트웨어 진화 및 자동화 개발 능력을 증명함

- 24시간 미만의 AI 코딩으로, 제품 수준의 소프트웨어 복제와 고급 인터페이스까지 단숨에 구현할 수 있음을 시연
- 사람 개입 없이 자동화 반복과 AI의 창의성이 결합돼 소프트웨어 생산성이 획기적으로 상승할 수 있음
- 동영상 내내 “정말 대단하다”, “crazy(미쳤다)”, “so feature-rich(기능이 너무 많다)”는 감탄사가 반복적으로 나타남

### 영상은 AI 주도 소프트웨어 개발의 혁신성과 구체적 결과물의 수준을 직접적으로 보여줌

- 단기 실험이지만 실제로 누구나 동영상을 통해 결과물과 과정을 확인 가능
- AI로 만든 웹앱의 실제 기능, 코드 자동 생성‧실행, 설정 패널 완성 등이 구체적으로 언급됨
- 최신 대규모 언어모델 기반 개발 자동화의 현실적인 성취를 명확히 제시함

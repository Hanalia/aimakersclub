## *AI 전쟁이 다시 시작된다! Sonnet 4.5, DeepSeek V3.2, GLM-4.6 출시* 핵심 요약

- 최근 AI 전쟁이 재점화되었으며, 여름 동안은 새로운 대형 언어 모델(LLM) 출시에 정체가 있었으나 지금은 다양한 신제품이 한꺼번에 출시되고 있음
- 어제 Anthropic에서 Claude Sonnet 4.5를 출시했으며, 이 모델은 현재 "코딩 분야의 새로운 AI 왕" 후보로 꼽힘
- Sonnet 4.5는 Opus 4.1 및 GPT-5 Codecs보다 소폭 우수하지만, 특히 속도 면에서 확연한 강점을 보임
- Cole Medin 본인의 테스트 결과, Sonnet 4.5는 코드 생성 속도가 매우 빠름
- DeepSeek의 새로운 버전 3.2 역시 공개되어, 경량화된 구조와 강력함, 그리고 저렴한 비용이 특징임
- DeepSeek 3.2는 Sonnet 4.5에 비해 수십 배 저렴하며, 속도와 효율성에서도 뛰어남
- 중국에서는 GLM 4.6이 곧바로 공개되었으며, Sonnet 4.5 출시 직후 대응 제품으로 주목받음
- GLM 4.6은 벤치마크 테스트에서 Sonnet 4.5를 능가하는 성적을 보이며, 여러 항목에서 우위를 점함
- 벤치마크 결과만으로 모든 성능을 단정할 순 없지만, 현재 LLM 업계가 극도로 흥미롭고 빠르게 변화하고 있음을 시사함

## 세부 요약 - 주제별 정리

### AI 전쟁이 여름의 정체기를 지나 다시 본격화되었음

- 지난 여름 동안에는 전례 없이 주목받을 만한 새로운 LLM(대형 언어 모델)이 거의 출시되지 않았음
- 그러나 최근 단기간에 여러 모델이 동시다발적으로 공개되어 AI 시장의 경쟁이 재점화됨
- AI 모델 간에 한 제품이 출시되면 곧바로 다른 제품이 대응하여 출시되는 패턴이 반복되고 있음

### Claude Sonnet 4.5의 등장은 코딩 분야에서 강력한 경쟁자를 의미함

- Anthropic가 어제(영상 기준) Claude Sonnet 4.5를 출시함
- Claude Sonnet 4.5는 "AI 코딩의 새로운 왕"으로 불릴 정도로 성능이 주목받고 있음
- 성능 평가는 기존 Opus 4.1, GPT-5 Codecs보다 "약간" 우수하다는 점이 언급됨
- Cole Medin 자체 테스트 기준, Sonnet 4.5의 실행 속도가 눈에 띄게 빠름
- 실사용 관점에서 속도가 매우 큰 장점으로 부각됨

### DeepSeek V3.2는 성능-가격-속도 삼박자를 갖춘 경량 모델로 등장함

- DeepSeek의 최신 버전 3.2도 새롭게 공개됨
- 이전 버전보다 훨씬 빠르고 경량화된 구조임에도, 여전히 뛰어난 성능 유지
- 특징적으로 Sonnet 4.5 대비 수십 배 저렴한 가격대를 제공함
- DeepSeek 3.2는 저비용, 고효율, 강력한 모델을 원하는 사용자들에게 차별화된 선택지임

### 중국의 GLM 4.6은 출시 직후부터 글로벌 경쟁 구도를 형성함

- 중국에서 GLM 4.6이 곧바로 공개되어, Anthropic의 Sonnet 4.5 발표 하루 만에 대응함
- 업계의 빠른 반응 속도와 글로벌 경쟁 분위기를 반영함
- GLM 4.6은 바로 공개된 벤치마크에서 Sonnet 4.5를 겨냥해 비교되고 있음
- 여러 벤치마크 테스트 항목에서 Sonnet 4.5보다 앞선 성능 결과를 보임

### GLM 4.6은 벤치마크에서 두각을 드러냈으나, 실사용 평가는 추가적 검토가 필요함

- Cole Medin은 "벤치마크가 모든 것을 말해주지는 않는다"는 점을 강조함
- 그럼에도 GLM 4.6이 Sonnet 4.5를 앞서는 영역이 많음(“winning most of the benchmarks”)
- 실제 사용성, 적용 사례 등은 향후 추가 평가가 필수적임

### 신규 LLM 경쟁은 기술 혁신과 사용자의 선택 기회를 폭발적으로 확대함

- 최근 짧은 기간 동안 다양한 모델이 신속하게 공개되어 사용자가 선택할 수 있는 폭이 넓어짐
- 각 모델은 성능, 속도, 가격, 효율 측면에서 차별화된 강점을 내세움
- 업계 전반에 걸쳐 기술 혁신 경쟁이 다시 한 번 본격화되고 있음을 보여줌

### 현재 LLM 시장은 극도의 흥분과 불확실성이 공존하는 시기임

- Cole Medin은 “아주 흥미롭고 신나는 시기”라 언급하며, 예측이 어려운 격변기를 강조함
- 벤치마크, 출시 속도, 모델 기능 등에서 기록적인 변화가 이어지고 있음
- 앞으로도 새로운 LLM의 등장과 경쟁이 지속적으로 전개될 것으로 전망됨

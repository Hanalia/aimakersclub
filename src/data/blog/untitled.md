## *Claude Code로 아름다운 다이어그램 만들기: 전체 워크플로우* 핵심 요약

- 클로드 코드(Claude Code)와 같은 코딩 에이전트는 시각적 설명이 미흡하여, 충분한 프롬프트(명령)와 적절한 기능 부여 없이는 시각적 업무에 한계가 있음
- 이를 개선하기 위해, 제작자는 자체 워크플로우 전반을 하나의 ‘스킬’로 패키징하여 코딩 에이전트가 실용적이고 아름다운 다이어그램을 손쉽게 만들 수 있도록 구성함
- 주요 방식은 코딩 에이전트가 Excal(Excalidraw, Excalra) 다이어그램용 JSON 파일을 생성하고, 이를 바탕으로 시각적으로 주장(argue visually)하는 능력을 강화하는 것임
- 워크플로우의 핵심은 자동화된 ‘시각적 자기 검증(self validation)’ 단계로, 클로드 코드가 다이어그램 생성 후 PNG로 렌더링 및 스크린샷을 찍고, 이를 검토해 불완전한 부분을 반복 수정함
- 이 과정을 통해 기존보다 훨씬 완성도 높은, 실용적이고 ‘개성 있는’ 다이어그램을 빠르게 제작 가능함
- 사용자는 코딩 에이전트에게 Excalra 다이어그램 스킬을 셋업하도록 간단히 요청만 하면 되고, 자동으로 설치 과정이 안내됨
- 스킬의 중심 철학은 구조와 라벨링 등 시각적 요소를 통한 논리적 설명에 있으며, 기존의 단순 ‘블록 나열'에서 벗어나 다이어그램만으로도 개념을 명확히 전달하는 데 초점을 둠
- 이 워크플로우는 PDF, 코드베이스 등 다양한 대상에 바로 적용할 수 있으며, 실무에서 매주 수 시간 이상 절약 가능
- 각자 브랜드 컬러, 선호 스타일 등 맞춤화된 다이어그램 제작이 용이함
- 해당 스킬 레포지터리를 클론해 누구나 손쉽게 시작할 수 있음

## 세부 요약 - 주제별 정리

### 코딩 에이전트의 시각적 한계를 극복하기 위한 필요성이 제기됨

- 클로드 코드 등 최신 코딩 에이전트는 뛰어난 텍스트 분석과 코딩 능력을 보유하지만, 시각 자료 설명이나 시각적 설계는 기본적으로 약함
- 사용자와의 소통이나 개념 설명에 있어 시각적 도구의 부재로 비효율 발생
- 충분한 프롬프트, 복잡한 지시 없이는 원하는 수준의 시각적 결과 얻기 어려움
- 이러한 문제를 해결하고자 제작자가 직접 시각화 관련 워크플로우를 개발하게 됨

### 전체 다이어그램 제작 워크플로우가 Excalra 스킬로 패키징됨

- 본 워크플로우 전반을 별도 ‘스킬’로 묶어 Claude Code 등 어떤 코딩 에이전트에도 적용 가능하게 설계
- 이 스킬을 통해 코딩 에이전트가 Excal 다이어그램 같은 시각 자료 제작을 자동으로 처리하게 됨
- 수동 작업을 최소화하고, 매우 복잡하거나 실용적인 다이어그램도 쉽게 생성
- 사용자는 단순 요청만으로, 에이전트가 알아서 각 단계별 처리를 수행

### Excalra 다이어그램을 생성하는 기본 프로세스가 소개됨

- 코딩 에이전트는 Excal(Excalidraw, Excalra) 다이어그램을 위한 JSON 포맷 파일을 자동 생성함
- ‘시각적으로 주장하기(argument visually)’라는 주요 명령어 집합에 따라 동작
- 구조화된 JSON 데이터 기반으로 다양한 구성, 도형, 텍스트 등이 자동 배치됨
- 실제 인공지능이 논리와 논거를 시각적으로 설계할 수 있도록 프롬프트가 정교하게 조정됨

### 다이어그램의 ‘자동 자기 검증' 단계로 완성도를 크게 높임

- 다이어그램이 생성되면, Claude Code가 이를 즉시 시각적으로 렌더링하여 PNG 이미지로 저장
- 이미지(스샷)를 인공지능이 직접 검토, 미비점이나 개선 필요 부분을 감지
- 반복적으로 다이어그램 구조를 수정/보완하는 ‘iterate’ 과정을 통해 완성도 향상
- 기존처럼 사람이 직접 눈으로 확인하고 수정할 필요 없이, 에이전트가 셀프 체크를 수행함

### Excalra 설치 및 활용은 코딩 에이전트의 자동화된 설명으로 간편함

- 사용자는 코딩 에이전트에게 ‘엑스칼라 다이어그램 스킬’을 설치하도록 한 마디만 요청하면 됨
- 에이전트가 README 내용을 읽고, 매뉴얼의 모든 설치 절차를 단계별로 자동 실행
- 사용자가 별도의 튜토리얼이나 수동 설치와정을 거치지 않아도 됨
- 설치 후, 바로 다양한 다이어그램 요청이 가능하며, 이전 데모에서 선보인 예시와 같이 손쉽게 조작 가능

### 개인 브랜드 또는 선호 색상까지 커스터마이징이 용이함

- 생성되는 모든 다이어그램의 컬러, 폰트, 스타일 등 맞춤 조정이 가능
- 브랜드 정체성 혹은 개인이 선호하는 디자인 요소를 쉽게 적용할 수 있음
- 프롬프트 및 스킬 내부 logic을 통해 사용자 정의 색상표, 스타일 통일성이 자동 적용됨

### ‘블록 나열’이 아닌 논리적 구조 설계를 에이전트가 배움

- 기존 Excal 다이어그램 자동 생성은 주로 ‘많은 사각형(블록)’ 나열 결과를 냄
- 스킬의 핵심 프롬프트 철학(skill.mmd 파일)은 ‘시각적 논증’을 에이전트에게 교육
- 단순히 도형 배열이 아니라, 각 구조와 라벨링, 관계성 등 전체 개념이 그림 만으로 설명되도록 설계됨
- 실제 지식 전달이나 시스템 설명, 프로세스 도식화 등에서 압도적 효율 구현

### PDF, 코드베이스 등 다양한 실무 컨텍스트에 즉시 적용 가능성을 강조함

- Excalra 다이어그램 스킬은 일반 문서(PDF), 코드베이스 등 실무의 여러 상황에 바로 활용 가능
- 복잡한 코드 구조, 데이터 흐름 등도 다이어그램으로 직관적으로 시각화 가능
- 영상 데모 예시처럼, 문서나 코드 내용 한 줄 프롬프트로 다양한 차트/도식 자동 생성

### 시간 절약 및 워크플로우 효율화 효과가 매우 큼

- 반복적이거나 복잡한 도식 작업에 주당 여러 시간 단축 효과
- 초안부터 완성까지 사람이 개입하는 단계를 최소화
- 창의적인 다이어그램 초안을 빠르게 얻고, 세부 조정만 직접 수행

### 누구나 스킬 레포지터리를 클론/도입해 바로 사용 가능함

- 공개 깃허브 레포지터리가 제공되어 누구든 바로 복제, 설치 가능
- 별도 고급 지식 없이, 프롬프트만 사용해 워크플로우에 바로 적용
- 영상 제작자는 ‘시작해 보라’며 즉각적 활용을 적극 권장함

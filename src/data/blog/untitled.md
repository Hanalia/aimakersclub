## *내 Claude 코드가 내 컴퓨터를 거치지 않고 GitHub 이슈를 자동으로 해결하는 방법 (무료 도구!)* 핵심 요약

- 영상에서는 사용자의 직접적인 코드 작성 없이 AI 코딩 에이전트가 원격 환경에서 GitHub 이슈를 해결하는 과정을 소개함
- 클라우드 기반 도구인 **Workspaces by coder**(무료, 오픈소스)를 활용해 전체 프로세스가 사용자의 로컬 컴퓨터가 아닌 원격 인스턴스(VPS)에서 진행됨
- 사용자는 GitHub 이슈에 'coder' 태그만 추가하면 자동으로 코딩 에이전트가 해당 이슈를 처리하도록 트리거함
- Workspaces는 코딩 에이전트를 원격, 대규모, 병렬로 오케스트레이션할 수 있는 인프라를 제공함
- 실제 코드 작성, 환경 세팅, 이슈 해결, 풀 리퀘스트 생성까지의 모든 작업이 자동화되어 있음
- 각 코딩 세션은 격리된 환경에서 독립적으로 실행되어 여러 이슈를 동시에 처리 가능함
- 작업 진행 상황은 GitHub 이슈에 실시간 링크로 코멘트되어 추적 가능함
- 최종적으로 GitHub에서 자동으로 생성된 풀 리퀘스트를 확인 및 머지 가능
- 팀 단위에서도 인프라를 공유하여 병렬적으로 협업 가능함
- 해당 도구는 무료이며 직접 사용해볼 것을 권장함(영상 설명란에 링크 제공)

## 세부 요약 - 주제별 정리

### AI 코딩 에이전트가 사용자의 개입 없이 GitHub 이슈 해결을 전담함

- 사용자는 단 한 줄의 코드도 직접 입력하지 않음
- 모든 작업이 코딩 에이전트의 자체 컴퓨터(원격 인프라)에서 이루어짐
- GitHub 이슈를 개설하고 'coder' 태그만 추가하면 나머지 처리 과정이 자동화됨

### 무료 오픈소스 도구 Workspaces by coder로 모든 자동화의 기반을 구축함

- Workspaces는 Coder 플랫폼에서 제공하는 무료, 오픈소스 소프트웨어임
- AI 코딩 지원의 상위 오케스트레이션 레이어를 원격 구축하는 역할
- 도구의 배포 및 관리가 쉽고 원격 인프라 상에서 구동 가능함
- 팀 단위의 인프라 공유 및 확장성 보장

### 원격 인프라에서 코딩 관련 모든 세션이 병렬로 관리됨

- 사용자는 자신의 PC가 아닌 원격 VPS(가상 사설 서버)에 Workspaces를 셀프 호스팅
- 각 코딩 에이전트 세션이 완전히 격리된 환경에서 실행되어 문제점 간섭 없음
- 동시에 여러 GitHub 이슈를 병렬로 처리할 수 있음(예: 10개 이상도 가능)

### 'coder' 태그만으로 자동 트리거돼 간편하게 코딩 업무를 맡길 수 있음

- GitHub 이슈에 'coder' 태그를 붙이는 것만으로 처리 프로세스가 시작됨
- 추가적인 수동 작업 없이 전체 파이프라인이 자동으로 활성화됨
- 태그 적용 직후 Workspaces에서 새 작업(task)이 생성되는 것을 실시간으로 확인 가능

### 작업 프로세스가 실시간으로 추적 가능하도록 연동됨

- Workspaces의 태스크 목록에서 이슈 관련 새로운 작업이 등록되는 과정을 볼 수 있음
- 작업이 초기화되면 GitHub 이슈에 코멘트로 진행상황 추적 링크가 자동 등록됨
- 사용자는 추가 조작 없이 링크를 통해 현재 상황을 관찰할 수 있음

### 코딩 에이전트가 환경 생성부터 이슈 해결, PR 생성까지 모든 과정을 수행함

- 원격 환경에서 해당 이슈 해결을 위한 개발 환경을 자동으로 스핀업
- 코딩 에이전트(예: Claude code)가 실제 코드 작성 및 문제 해결 작업을 진행
- 작업이 완료되면 GitHub에 자동으로 풀 리퀘스트(PR)가 생성됨

### 최종 코딩 결과를 GitHub에서 바로 리뷰 및 머지할 수 있음

- Workspaces UI 내에서 각 작업의 완료 여부를 쉽게 확인 가능
- 자동 생성된 PR을 GitHub에서 직접 리뷰 및 머지 가능
- 개발 프로세스가 완전히 자동화 및 원격화되어 있음

### 여러 코딩 에이전트 세션을 팀 단위로도 병렬 운영하며 협업 효율을 극대화함

- 팀원 전체가 인프라를 공유하여 다양한 이슈를 동시에 처리 가능
- 각 세션은 격리되어 협업 중 발생 가능한 충돌/간섭을 방지함
- 병렬성 덕분에 대량의 이슈 처리 및 자동화가 용이함

### Workspaces는 '코딩 에이전트 인프라'로서 스케일과 관리의 장점을 제공함

- Workspaces 자체는 코딩 에이전트가 아니며, 코딩 에이전트의 실행 환경 및 병렬 오케스트레이션을 담당
- 원격 인프라/클라우드 환경에 쉽게 배포 및 관리 가능
- 확장성과 유연성으로 다양한 규모의 팀/개별 개발자에 적용 가능

### 영상 제작자는 무료 체험을 적극 권장하며 사용법 관련 정보도 함께 안내함

- Workspaces 도구는 무료이므로 누구나 쉽게 접할 수 있음
- 영상 설명란에 공식 정보 및 사용 링크 제공
- 효율적인 코드 자동화 및 팀 개발에 관심 있는 이들에게 적합한 도구임

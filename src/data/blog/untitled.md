## *Claude에게 24시간 내내 휴식 없이 코딩을 시켰더니 어떤 일이 일어났나* 핵심 요약

- 영상에서는 AI 모델인 Claude를 24시간 동안 쉬지 않고 코딩하도록 한 실험 결과를 소개함
- 실험 결과, claw.ai의 완전한 클론(복제판) 수준의 소프트웨어가 만들어짐
- 생성된 프로그램은 과거 대화 내역을 모두 저장하고, 검증 및 반복 루프를 거치며 탄탄한 결과물을 형성함
- 결과물에는 Markdown 포맷 지원, HTML 페이지 생성, 코드 작성 및 실행 등의 다양한 기능 포함
- 테마 변경, 기본 모델 선택, 최대 토큰 수 슬라이더 등 사용자 설정 또한 지원
- 기능적으로 매우 풍부(feaure-rich)하여 상당한 완성도를 자랑함
- 아직 ‘human in the loop'(사람의 검수 추가) 기능은 넣지 못했으나, 이미 자동화 수준이 매우 높다고 느낌
- Claude Opus 4.5와 같은 최신 AI와, 이를 반복 검증하는 프레임워크를 결합할 때 놀라운 성과가 가능함을 강조
- 남은 기능도 하루만 더 주면 완벽한 애플리케이션이 될 것 같다는 기대감을 밝힘
- 전체적으로 "손을 대지 않고" AI가 생산한 결과에 대해 놀라움을 표현

## 세부 요약 - 주제별 정리

### Claude로 하루 만에 claw.ai 수준의 클론을 개발하는 데 성공함

- AI 모델인 Claude에게 24시간 내내 자동으로 코딩을 맡기는 실험을 진행함
- 실험의 목적은 Claude가 어느 수준까지 애플리케이션을 자율적으로 개발할 수 있는지 확인하는 것이었음
- 결과적으로 claw.ai를 완전히 복제한 듯한, 전 기능이 모두 구현된 프로그램을 뽑아냄
- 실험자는 이미 중간 이후의 테스트가 왜 필요한지 의문을 가질 만큼 결과물이 완벽에 가까웠다고 평가함

### AI가 생성한 대화 내역과 검증 루프가 소프트웨어 완성도를 높임

- 개발 과정에서 Claude가 생성한 모든 대화 내역을 추적하고 저장함
- 생성된 대화 내역을 반복적으로 활용하며, 각 단계에서 자체적인 검증 루프를 거침
- 이러한 자동화된 검증 과정이 소프트웨어의 신뢰성 및 완성도를 크게 높였음

### Markdown 포맷 지원 등 다양한 문서화 기능이 구현됨

- 결과 프로그램에서는 Markdown으로 잘 포맷된 문서화가 제공됨
- 코드, 결과 설명 등 각종 출력이 깔끔하게 구성되어 있어 사용성 측면에서 매우 우수했음
- 생성된 문서들은 리뷰 및 재사용에 용이하도록 체계적으로 정리됨

### HTML 페이지 생성과 코드 작성·실행까지 자동으로 구현함

- AI가 직접 HTML 페이지를 다양한 방식으로 생성하는 기능을 구현함
- 프런트엔드, 백엔드 구분 없이 다양한 코드도 자동으로 작성
- 작성된 코드를 바로 실행할 수 있는 환경 구성까지 자동화하였음

### 사용자는 테마, 모델, 토큰 수 등 환경을 자유롭게 설정 가능함

- 결과물에서는 사용자가 직접 테마(색상 등 UI 스타일)를 변경할 수 있게 함
- 어떤 AI 모델을 기본(default)으로 선택할지 지정 가능함
- 토큰(max tokens) 갯수도 슬라이더로 손쉽게 조절하는 등 풍부한 사용자 환경 지원

### 전체적으로 매우 많은 기능과 높은 완성도를 자랑함

- 실험자는 "지금까지 만들어 놓은 것이 정말 인상적"이라고 평가함
- 직접 코딩에 관여하지 않고도 이렇게 다양한 기능 구현이 가능했던 점을 강하게 강조함
- 자동 생성된 애플리케이션이 긴 시간 수작업으로 만든 서비스와 비견될 만큼 완성도가 높다고 강조함

### 아직 'human in the loop' 기능은 미완성이지만 자동화 수준에 감탄함

- 남은 기능으로 ‘human in the loop’, 즉 사람의 검수·보완은 아직 구현하지 못함
- 그럼에도 불구하고 나머지 핵심 기능을 거의 완벽히 자동화한 점에 대해 감명을 밝힘

### Claude Opus 4.5와 검증 및 반복 프레임워크 결합의 혁신성 언급

- Claude Opus 4.5 모델의 능력과, 이를 지원하는 ‘harness’(반복 검증 환경)의 결합이 큰 성과를 냄
- 해당 프레임워크를 통해 AI가 스스로 기능을 검증하고 빠르게 개선해 갈 수 있었음
- 이러한 구조가 향후 애플리케이션 자동 생성의 혁신적 모델이 될 수 있음을 암시함

### 하루만 더 실험하면 완벽한 앱이 나올 가능성을 언급함

- 실험자는 남은 기능까지 다루려면 하루만 더 Claude에게 작업 시간(24시간!)을 준다면 완벽한 애플리케이션이 나올 것 같다고 전망함
- Claude와 자동화 환경이 결합할 경우 소프트웨어 개발 방식 자체가 크게 변화될 수 있다는 점이 이번 실험에서 입증됨

### 직접적인 사람의 개입 없이 AI가 개발을 주도한 점에 극도의 놀라움을 표현함

- 영상 마지막 부분에서 “직접 손 하나 까딱하지 않고 이런 결과를 얻었다”며 강한 충격을 드러냄
- AI의 독립적 문제 해결 능력과 완성도 높은 결과물 생산 역량에 거듭 놀라워함

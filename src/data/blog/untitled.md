## *클로드 코드를 10배 더 효율적으로 만들어 주는 무료 툴* 핵심 요약

- 클로드(Claude) 코드의 강력함과 생산성은 인정하지만, AI 생성 코드의 리뷰 과정에서 불편함이 존재함을 지적
- 클로드 코드의 여러 인스턴스를 병렬로 실행할 때, 각각의 출력물을 일일이 검토하는 과정이 특히 번거로워짐
- 이 문제 해결을 위해 최근 무료 툴 ‘킨우기(Kinugi)’를 적극적으로 활용하고 있음
- 킨우기는 클로드 코드 자체를 대체하지 않고, 그 위에 시각화된 인터페이스를 제공하는 ‘에이전틱 개발 환경’임
- 킨우기를 통해 모든 병렬 실행 세션의 상태(리뷰 필요, 실행 중 등)를 한 눈에 파악하며 코드 변경 내용을 직관적으로 검토 가능
- 에이전트가 생성한 ‘플랜(Plan of attack)’도 실시간 코멘트 및 피드백으로 반영할 수 있음
- 리뷰 완료 후 ‘풀 리퀘스트(Pull Request)’와 유사한 형태로 결과물을 받아볼 수 있어 기존 코드 로그와 파일별 확인보다 효율적임
- 킨우기에는 AI 코드에 추가적인 코드리뷰(예: 보안 검토) 기능이 내장되어 있으며, Sonar팀에서 개발했고 MacOS에서만 실행 가능하며 현재 실험적임
- 무료로 제공되며, 기존 클라우드 코드 기반의 워크플로우를 그대로 유지하면서 관리와 생산성이 크게 향상됨
- Sonar에서 직접 체험 요청을 받아 본 툴로, 링크가 설명란에 안내됨

## 세부 요약 - 주제별 정리

### 클로드 코드는 강력하지만 리팩토링 및 코드 리뷰에 한계가 있음

- 클로드 코드(Claude Code)는 AI 기반 개발 도구로, 개발 과정 전체를 크게 향상시키는 역량을 보여줌
- 강력한 자동화 기능 덕분에 영상 제작자는 모든 개발 과정에서 해당 도구를 활용한다고 언급
- 그러나 클로드 코드가 생성하는 방대한 양의 출력물을 한 번에 모두 검토하는 데 시간과 노력이 크게 소요됨
- 특히 여러 개의 클로드 코드 인스턴스를 병렬로 실행할 때 각 코드 세션을 번갈아 확인해야 하는 번거로움이 큼
- 이런 리뷰 부담이 실제 사용자 경험의 주요 방해 요소로 작용함을 강조

### 병렬 세션의 코드 리뷰가 불편하며 작업 효율을 떨어뜨림

- 영상 제작자는 평소 여러 클로드 코드 인스턴스를 동시에 실행한다고 밝힘
- 각각의 AI 결과물을 일일이 로그와 파일로 확인해야 해 작업 흐름이 자주 중단됨
- AI가 생성한 코드라도 최종적으로 반드시 사람이 검토하는 과정이 필수라고 강조함
- 대량의 출력물을 효율적으로 관리할 방법이 절실히 필요함을 언급

### 킨우기는 클로드 코드 위에 시각화 계층을 추가하여 문제를 해결함

- 킨우기(Kinugi)는 클로드 코드와 통합되는 에이전틱 개발 환경(Agentic Development Environment)임
- 자체적으로 클로드 코드를 대체하는 것이 아니라, 그 위에 ‘시각적 레이어’를 더함
- 모든 병렬 실행 중인 클로드 인스턴스를 한 화면에서 관리하고, 각 코드 변화 및 리뷰 대상 현황을 한눈에 파악하게 해줌
- 사용자는 모든 코드 에이전트의 실시간 상태, 세션별 진행 상황, 리뷰 요청 등을 세밀하게 추적할 수 있음

### 에이전트가 만드는 ‘플랜’을 손쉽게 확인하고 지시할 수 있음

- 클로드 코드가 생성한 ‘플랜(Plan of attack)’을 한 화면에서 확인 가능
- 사용자는 각 ‘플랜’에 코멘트(댓글)를 달아 피드백을 바로 전달하며, 수정 및 보완을 실시간으로 반영함
- 이러한 협업 및 피드백 과정이 병렬적이고 직관적으로 이루어짐

### 리뷰가 끝난 코드는 ‘풀 리퀘스트’와 유사하게 깔끔하게 확인할 수 있음

- 킨우기는 리뷰 완료된 코드를 ‘풀 리퀘스트(Pull Request)’와 유사한 형식으로 한번에 모아 보여줌
- 전통적인 방식처럼 각각의 클라우드 코드 로그 및 파일을 별도로 파고들 필요가 없음
- 모든 리뷰 및 피드백 과정을 한 곳에 통합하여 전체 흐름을 효율적으로 관리 가능

### 킨우기에는 보안 리뷰 등 추가 코드리뷰 기능이 내장됨

- 킨우기에는 AI가 만든 코드에 추가적으로 코드 리뷰를 적용할 수 있는 새로운 기능들이 포함됨
- 예시로 ‘보안 검토(Security Review)’를 언급, 이는 영상 제작자가 직접 강조함
- 이러한 기능이 내장되어 있는 이유로 킨우기의 개발사가 ‘Sonar’임을 밝힘

### 킨우기는 MacOS에서만 사용 가능하며 현재 실험적 단계에 있음

- Sonar 측에서 공식적으로 킨우기는 현재 Mac OS에서만 지원되며 매우 실험적인 프로젝트라고 언급
- 이런 이유로 현재 무료로 사용 가능함을 안내함
- 실험적임에도 불구하고 다양한 강력한 기능 덕분에 생산성과 관리 효율이 이미 크게 향상됨

### 킨우기로 클라우드 코드 기반의 워크플로우를 그대로 유지하며 관리 효율을 극대화할 수 있음

- 기존의 클라우드 코드 중심 작업 방식(에이전트 기반 워크플로우)에 변화 없이, 킨우기를 추가로 통해 효율성이 배가됨
- 별도의 더 복잡한 도구나 대체 과정을 추가할 필요 없이, 시각적 관리와 리뷰 자동화 계층만 추가함

### Sonar에서 직접 체험 요청을 받고 영상에서 실제 경험을 공유함

- 킨우기는 영상 제작자가 Sonar로부터 직접 체험 요청을 받아 사용하게 되었음을 밝힘
- 실제 사용 후 ‘정말 매력적이라고 느낀다’, ‘상당히 멋지다’는 긍정적 피드백을 영상 내에서 언급
- 킨우기 다운로드 링크는 유튜브 영상 설명란에서 확인할 수 있다고 안내함

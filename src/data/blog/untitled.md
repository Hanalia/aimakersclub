## *Claude에게 24시간 동안 쉬지 않고 코딩을 시켰더니 벌어진 일* 핵심 요약

- 영상은 AI 언어모델 Claude(Opus 4.5)를 이용해 24시간 동안 자동으로 응용 프로그램을 개발한 실험 결과를 다룸
- 놀랍게도 기존 claw.ai 서비스의 완벽한 복제본과 유사할 정도로 기능이 거의 완성된 상태에 도달함
- 생성 과정을 남긴 과거 대화 로그를 통해 각 개발 반복 단계와 검증과정이 투명하게 확인 가능함
- 최종 결과물은 풍부한 마크다운(Markdown) 포매팅, 다양한 HTML 페이지 생성, 코드 작성·실행 등 복수의 고급 기능을 포함함
- 사용자 설정에서 테마, 기본 사용 모델, 토큰 수 조절 슬라이더 등 상세한 옵션도 제공되도록 구현됨
- 프로젝트는 아직 ‘Human in the Loop’(사람의 개입) 기능이 추가되어야 하지만, 대부분의 개발이 AI에게만 맡겨도 성공적으로 진행됨을 시연함
- AI만으로 이 정도 수준의 애플리케이션, 특히 검증과 반복 개발까지 가능한 것에 큰 감탄을 표함
- 이에 한층 더 많은 반복과 기능 추가를 거친다면 거의 완벽에 가까운 애플리케이션 개발도 가능하리라 전망함

## 세부 요약 - 주제별 정리

### Claude가 claw.ai 복제본을 24시간 만에 주요 기능까지 자동 개발함을 실험함

- 실험 목표는 Claude(오퍼스 4.5 버전)를 24시간 쉬지 않고 활용하여 실제 서비스 복제 수준의 애플리케이션 구현이 가능한지 검증하는 것임
- 개발자는 ‘마치 claw.ai의 완전히 기능적인 클론’을 이미 구현한 듯한 느낌을 강조함
- 남은 절반의 테스트 목적조차 명확하지 않을 만큼 충분히 완성도가 높다고 밝힘
- Claude가 각 개발 루프에서 문제를 검증하고, 생성된 과거 대화를 통해 전 과정을 추적할 수 있음을 소개함

### 생성된 애플리케이션은 마크다운과 HTML 등 방대한 기능을 갖춤

- 결과물은 깔끔하고 체계적인 **마크다운 포맷팅** 지원 기능이 포함됨
- HTML 페이지 여러 유형을 자동으로 생성하고 관리할 수 있음
- 코드 작성, 실행 기능까지 모두 통합하여 사용자가 직접 확인 가능
- 모든 개발과정이 자동화된 채 여러 문서와 시각적 결과물이 생성된 점을 강조함

### 사용자는 상세 설정을 직접 변경할 수 있도록 인터페이스가 구축됨

- 설정(Settings) 메뉴에서 **테마(theme)** 변경 및 **기본 모델(default model)** 선택 기능이 제공됨
- **최대 토큰(max tokens)을 조절하는 슬라이더**가 인터페이스에 추가됨
- 인터페이스의 이러한 기능 풍부함(feature-rich)을 반복적으로 강조함

### “Human in the Loop”의 필요성은 남아있지만 AI 개발 성과에 큰 감탄을 표시함

- 인간의 개입을 통한 최종 검증 단계(Human in the Loop)는 아직 남아 있음
- 하지만 개발자는 직접 코드를 작성하지 않고도 이렇게 많은 기능을 구현한 결과에 놀람
- 완성도를 더욱 높이기 위해 추가적인 반복(iteration)과 기능 점검 의지를 내비침

### longer iterations과 반복 개발로 완벽에 가까운 애플리케이션도 실현 가능함을 시사함

- 만약 Claude가 24시간이 아닌, 추가적으로 하루만 더 주어진다면 정말 ‘완벽한’ 앱에 가까워질 수 있다고 전망
- Opus 4.5와 같은 최신 AI 도구와 이를 검증·관리할 수 있는 하니스(harness)가 결합되면, 반복적인 자동 검증과 고도화가 쉬워짐
- Claude가 다양한 피처를 통과하면서 자기검증(self-validation)과 여러 번의 반복 개선을 거치는 모습에 깊은 인상을 받았음을 밝힘

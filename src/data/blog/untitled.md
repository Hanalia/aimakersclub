## *Anthropic, Claude Sonnet 5를 공개하며 중견 모델의 강력한 성능과 가격 혁신을 보여줌* 핵심 요약

- Anthropic이 최근 Claude Sonnet 5를 출시했으며, 이번 버전의 핵심은 속도가 아니라 성능임
- Sonnet 5는 SweBench Verified 코딩 테스트에서 85.2점을 기록, 최상위 모델인 Opus 4.8에 거의 근접함
- 가격 면에서 Sonnet 5는 Opus 4.8과 비슷한 성능을 훨씬 저렴한 가격에 제공하는 것이 가장 큰 특징임
- Sonnet 5는 이전 세대에 비해 훨씬 뛰어난 ‘에이전트 능력’을 갖추었으며, 여러 단계의 작업 실행을 스스로 마무리할 수 있음
- 사용자는 저장소(리포지토리)와 목표만 지정하면 Sonnet 5가 계획을 세우고 터미널을 운용해 일련의 작업을 수행함
- 성능 지표에서는 Sonnet 4.6 대비 SweBench Verified 85, 터미널 벤치 67→80, Browse Comp 76→86으로 크게 도약함
- Frontier Code에서도 15에서 38로 2배 이상 상승했으며, 그 어떤 코드 벤치마크에서도 눈에 띄는 발전을 보임
- Opus 4.8이 아직은 소폭 앞서지만, 실제 이식 과정은 ‘모델 ID 한 줄’만 변경하면 될 만큼 간편함
- Sonnet 5의 가장 큰 혁신은 거의 Opus 수준의 코딩 성능을 가격 부담 없이 자유롭게 이용 가능하다는 것임
- 2024년 8월 31일까지 백만 토큰 입력 $2, 출력 $10의 프로모션이 제공됨
- 개인, 프로, 맥스, 팀, 엔터프라이즈, Claude Code, 플랫폼 등에 동일한 성능이 적용됨
- 영상 마지막에는 Sonnet 5가 ‘세대 교체’인지, ‘저렴한 Opus’인지 의견을 묻는 질문을 던짐

## 세부 요약 - 주제별 정리

### Claude Sonnet 5는 속도보다 성능에서 진정한 혁신을 보여줌

- 영상은 언론의 예상과 달리 Sonnet 5의 주요 이슈가 단순 속도 향상이 아니라 강조함
- Sonnet 5는 SweBench Verified 테스트에서 85.2점을 기록, 이는 매우 높은 점수로 Opus 4.8에 거의 근접한 성능임
- 이 성능은 중견 모델임에도 불구하고 플래그십 모델을 위협하는 수준임

### 더 강력해진 ‘에이전트’ 특성으로 멀티스텝 작업을 완수할 수 있음

- Sonnet 5는 지금까지의 Sonnet 모델 중 가장 ‘에이전트적(Agentic)’임을 강조
- 리포지토리를 지정하고 목표를 입력하면, Sonnet 5가 자동으로 계획을 세움
- 터미널 명령어 실행까지 수행하며, 기존 Sonnet이 중간에 포기하던 복잡한 멀티스텝 과업도 끝까지 완수함

### Sonnet 4.6 대비 성능 지표에서 대폭적인 상승을 기록함

- SweBench Verified 점수: Sonnet 4.6 75→Sonnet 5 85로 10점 상승
- Terminal Bench 점수: 67→80으로 13점 상승
- Browse Comp 점수: 76→86으로 10점 상승
- Frontier Code 점수: 15에서 38로 2배 이상 상승함 (코딩 영역에서 가장 큰 도약)

### 모델 교체는 간단하게 모델 ID 한 줄만 바꾸면 됨

- 기존 Sonnet 4.6에서 Sonnet 5로 마이그레이션 시 별도의 코드 수정이나 추가 작업이 필요 없음
- 단순히 모델 ID만 Sonnet 5로 변경하면 모든 이점을 누릴 수 있음

### Opus 4.8이 일부 벤치마크에서 여전히 앞서지만 실질적 차이는 작음

- Opus 4.8이 여전히 특정 지표에서는 Sonnet 5보다 약간 우위에 있음
- 특히 Frontier Code 점수에서는 Opus 4.8이 근소한 차이로 앞서 있음
- 그러나 가격 대비 실용성, 기존 코드와의 호환성을 감안하면 Sonnet 5의 메리트가 더욱 부각됨

### Sonnet 5는 다양한 Anthropic 서비스 구독자에게 동일한 성능을 제공함

- Sonnet 5의 높아진 성능은 무료(Free), 프로(Pro), 맥스(Max), 팀(Team), 엔터프라이즈(Enterprise), Claude Code, 플랫폼 등 모든 사용자 등급에 동일하게 적용됨
- 즉, 기업부터 개인 개발자까지 누구나 손쉽게 높은 성능을 활용할 수 있는 구조임

### 2024년 8월 31일까지 제한 특가 프로모션이 적용됨

- 입력 토큰 백만 개당 2달러, 출력 토큰 백만 개당 10달러로 가격 설정(기간 한정)
- 이로써 사용자들은 거의 Opus 수준의 코딩 성능을 매우 저렴하게 경험 가능
- 향후 가격 변동 가능성을 내포

### 에이전트 및 멀티스텝 작업 활용을 위한 실전 워크샵이 운영되고 있음

- 실제 ‘에이전트형 코딩’ 변화에 적응하고 싶은 개발자들을 위한 영상 내 언급 있음
- Dynamis 내부에서는 매주 이 주제를 다루는 실습 기반 워크샵이 열리고 있음

### 영상 마무리에서 Sonnet 5의 정체성에 대한 시청자 의견을 묻고 있음

- Sonnet 5가 ‘진정한 세대 교체 모델’인지, 단순히 ‘저렴해진 Opus’ 역할에 머무는지 검토를 촉구
- 시청자들에게 의견을 댓글로 남기라는 호출(Call to Action)으로 영상이 끝남

## *Claude에게 24시간 연속 코딩을 시켜본 실험 결과* 핵심 요약

- 영상은 Claude Opus 4.5 모델에게 24시간 동안 쉬지 않고 코딩을 시켰을 때 어떤 결과물이 나왔는지 구체적으로 소개함
- 실험 종료 시점에 이미 claw.ai의 완전히 작동하는 클론(복제 서비스)이 구축되어 있었음
- 인상적으로 많은 기능이 구현되어 있었으며, 실제로 실험자의 설명조차 필요 없을 정도로 완성도가 높았음
- Claude가 코드 생성과 각종 기능 검증 과정을 세부적으로 기록하고, 대화 이력을 통해 검증 과정을 추적할 수 있었음
- 마크다운 형식으로 깔끔하게 정리된 결과물, 다양한 HTML 페이지 생성, 코드 작성 및 실행 기능 등이 포함됨
- 테마 설정, 기본 모델 변경, 최대 토큰 슬라이더 등 세밀한 설정 기능까지 자동 구현됨
- 현재는 인간이 직접 개입하는 ‘human in the loop’ 기능만 추가하면 된다는 점이 남아 있음
- Claude와 같은 최신 AI가 애플리케이션 검증 및 반복 작업을 고도로 자동화하면서, 하루 만에도 기능이 풍부한 제품을 만들 수 있음을 입증함
- 실험자는 추가적인 24시간을 더 투자하면 정말 ‘완벽한’ 애플리케이션 완성이 가능할 것이라 언급함
- Claude Opus 4.5의 검증 자동화 능력과 반복적 개선 기능이 놀라운 결과를 이끌었음을 강조함

## 세부 요약 - 주제별 정리

### Claude가 24시간 만에 완전한 claw.ai 복제 서비스를 만들어냄을 확인함

- 실험자는 Claude Opus 4.5에게 24시간 동안 끊임없이 코딩을 시켰고, 결과를 점검함
- 실험이 끝나기도 전에 claw.ai와 유사한 복제 소프트웨어가 기능적으로 완성된 상태에 도달함
- 실험자는 “더 이상 테스트 절반이 뭔지 모르겠다”라고 말할 정도로 이미 모든 것이 동작함을 언급
- 복제된 서비스에 대한 전반적 만족도를 토로함

### Claude가 생성하는 코드와 검증 과정의 자동화가 인상적임

- Claude는 기능을 하나씩 생성할 때마다 자체적으로 코드 검증 루프를 반복 실행함
- 이를 통해 발생한 대화 이력(콘버세이션) 전체를 나중에 되짚어볼 수 있도록 저장함
- 각 루프마다 문제를 직접 해결하며, 실시간 피드백과 반복 개선 과정을 자체 관리함
- 이러한 자동화 작업은 실험자가 직접 수작업을 할 필요를 대폭 줄임

### 마크다운 포맷, HTML 페이지, 코드 실행 등 시각적·기능적 완성도가 높음

- 결과물은 마크다운 형식으로 깔끔하게 정리되었음
- 다양한 HTML 페이지를 자동 생성할 수 있도록 코드를 작성하고 실제로 실행함
- 코드 작성, 파일 저장 및 실행 과정에 있어서도 Claude가 직접 모든 단계를 처리함
- 실험자는 “완전히 작동하는 클론”임을 재차 강조

### 다양한 사용자 설정 기능과 편의성이 AI에 의해 자동 구축됨

- 서비스에는 테마(밝기/어둠 등) 변경 기능이 구현됨
- 사용자는 기본 AI 모델(엔진)을 직접 선택할 수 있음
- ‘최대 토큰 수’를 조절할 수 있는 슬라이더도 내장되어 있음
- 이런 고급 기능까지 처음부터 구현된 점에 실험자가 강한 인상을 받음
- 실험자는 “기능이 정말 풍부하다(so so feature-rich)”고 언급

### 인간 개입 요소(Human in the loop)만 남았음을 지적함

- 남은 과제로 ‘human in the loop’(AI가 자동화하는 과정에 사람이 개입할 수 있는 기능) 추가만을 남겨둠
- 현재 기술로 이미 상당 수준의 자동화와 기능 구현이 가능해졌음을 강조

### 하루 이상 추가 작업 시 ‘완벽한’ 애플리케이션 완성도 기대됨

- 실험자는 “이 상태에서 하루만 더 투자하면 실제로 완벽한 제품이 될 수 있을 것 같다”고 언급
- Claude Opus 4.5의 기능별 점검과 자동 반복 실행 능력이 빠른 완성도를 이끌어냄
- “crazy the kinds of things we can build”라는 표현으로 AI 개발 능력에 놀라움을 표함

### Claude Opus 4.5와 같은 AI 덕분에 검증·반복 작업이 혁신적으로 달라짐

- Claude를 harness(실행 환경)에 연결하여 실제 애플리케이션 상에서 검증 및 반복 작업을 자유롭게 수행함
- 기능별 테스트, 수정, 반복이 전자동으로 진행되어 개발 생산성을 극대화함
- 복잡한 피처와 설정도 AI가 알아서 통합하고, 사람이 손대지 않아도 작동함

### 전체 실험이 “손 하나 까딱하지 않고” 이뤄진 AI 자동 개발의 실증 사례임

- 실험자는 “정말로 손을 대지 않고 이 모든 것을 구현했다”며 AI 개발 자동화의 현실성을 보여줌
- 영상 전체에 걸쳐, 기존 수작업 자동화 한계를 Claude Opus 4.5가 뛰어넘었음을 강조

### AI가 애플리케이션 생성의 방식을 근본적으로 변화시키고 있음을 보여줌

- 하루 만에 다양하고 복잡한 서비스가 AI에 의해 자동 구축됨을 케이스로 제시
- AI 개발 능력의 현주소와 새로운 개발 생산성 표준을 직접적 사례로 보여줌
- 영상 내내 AI가 반복적으로 점검, 수정, 구현하는 과정을 실시간 대화 이력으로 추적할 수 있었음을 덧붙여 설명

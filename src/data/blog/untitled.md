## *Claude에게 24시간 동안 쉬지 않고 코딩을 시킨 결과* 핵심 요약

- 영상은 Claude Opus 4.5 인공지능에게 24시간 동안 쉬지 않고 코딩을 시킨 실험을 다룸
- 실험 결과, claw.ai의 기능을 거의 완벽하게 복제한 애플리케이션을 자동 생성함
- AI가 직접 생성한 모든 대화 기록과 코드 검증 과정을 확인할 수 있어 투명성이 높았음
- 마크다운 형식의 훌륭한 포매팅, 다양한 HTML 페이지 생성, 코드 작성 및 실행 기능이 포함됨
- 사용자 설정에서 테마 및 기본 모델 변경, 최대 토큰 조절 슬라이더 등 풍부한 설정 기능이 제공됨
- 아직 사람이 직접 개입(human in the loop)이 추가되지 않았지만, 거의 손대지 않고 다양한 기능을 구현함
- 추가적인 24시간 동작 시 완벽에 가까운 애플리케이션 생산도 기대할 수 있다고 언급
- Claude Opus 4.5와 자동 검증 하네스 도구가 합쳐졌을 때 응용 프로그램 제작 및 반복 검증 효율이 극대화됨

## 세부 요약 - 주제별 정리

### Claude Opus 4.5를 활용해 24시간 내 claw.ai 클론을 거의 완성함

- 영상 제작자는 Claude Opus 4.5라는 인공지능에 24시간 동안 쉬지 않고 애플리케이션 코딩을 맡김
- 그 결과 claw.ai의 기능을 매우 유사하게 구현한 클론이 완성됨
- 제작자 본인조차 일부 테스트의 용도를 알지 못할 만큼 기능 구현이 방대함
- “완전히 작동하는 클론”이라는 언급에서 기능 완성도에 대한 자신감을 드러냄

### AI가 생성한 대화와 검증 과정의 기록을 모두 확인할 수 있었음

- 개발 과정 중 Claude가 생성한 모든 대화(conversation) 기록이 남음
- 각 루프마다 기능을 검증하고 그 내용을 기록으로 남겼음
- 사용자는 이 대화 내역을 통해 빌드 및 검증의 전체 과정을 추적 가능함

### 마크다운 포매팅, 다양한 HTML 페이지 제작, 코드 실행 등 주요 기능이 구현됨

- 애플리케이션 내에서 마크다운(markdown) 형식의 세련된 포매팅이 가능함
- 여러 가지 HTML 페이지를 자동으로 생성하는 기능이 작동함
- 코드 작성 및 실행 기능도 포함되어 있음
- 위 기능들이 자동화된 방식으로 구축되어 사용 편의성이 높음

### 사용자 설정 메뉴에서 테마, 기본 모델, 최대 토큰 등 세부 조정이 가능함

- 설정(settings) 메뉴에서는 사용자 테마(theme) 변경이 가능함
- 기본 사용할 AI 모델(default model)도 교체할 수 있음
- 최대 토큰(max tokens) 사용량을 조절할 수 있는 슬라이더도 제공됨
- 다양한 맞춤화 옵션이 포함되어 기능성이 풍부함

### 실질적 인간 개입 없이 풍부한 기능을 자동으로 구현하는 과정이 돋보였음

- 제작자는 “직접 손을 대지 않고” 이 많은 기능을 구축하는 것이 상당히 놀랍다고 느낌
- 아직 ‘human in the loop(인간이 중간에 개입하는 기능)’이 추가되진 않았음
- 전 과정을 거의 AI가 처리했다는 점에서 자동화 정도가 매우 높음

### 추가 24시간 연장 시 완전무결한 애플리케이션 완성도 가능성을 언급함

- AI에게 더 많은 시간을 부여한다면 모든 기능이 추가되어 “완벽한 앱”이 나올 수 있다고 주장
- 24시간 만에 얻은 결과조차 이미 ‘crazy(말도 안 되는)’ 수준의 완성도임을 언급
- 기능 개발 속도 및 범위에서 Claude Opus 4.5의 잠재력을 강조

### 자동화·반복·검증 프로세스를 지원하는 하네스 도구가 완성도를 높였음

- ‘하네스(harness)’라는 별도의 자동화 검사 도구를 사용해 검증과 반복 작업을 자동 처리
- 개발 중 애플리케이션의 주요 기능들이 자동으로 테스트 및 검증되어 신뢰성 확보
- 검증-반복 구조 덕분에 휴먼 에러나 누락 없이 품질 높은 결과물이 탄생

### Claude Opus 4.5가 혁신적인 생산성 및 자동화 효율을 실증함

- 영상 전반에서 Claude Opus 4.5 AI의 실제 프로그래밍 응용사례를 체험적으로 확인함
- 사용자가 개입하지 않아도 복잡한 애플리케이션을 자동으로 설계·구현할 수 있음을 보여줌
- 반복 검증과 피드백 과정을 AI가 자체적으로 수행함에 따라 개발 생산성이 극대화됨

### claw.ai 클론 실험을 통해 향후 AI 기반 개발 환경의 가능성을 제시함

- claw.ai 같이 고기능 채팅형 인터페이스 앱도 AI의 자동화 능력만으로 복제 가능함을 시연
- 이런 자동화 및 검증 구조가 체계적으로 정립될 경우 소프트웨어 제작 패러다임에 큰 변화 예고
- ‘인공지능 + 검증 자동화 하네스’ 조합의 미래적 가능성을 실제 구축 사례로 설명

### 이번 실험 결과는 인공지능의 실질적 소프트웨어 개발력 향상을 방증함

- 사람이 거의 개입하지 않은 상태에서도 고품질 서비스가 신속하게 생성되는 결과를 보여줌
- 다양한 커스터마이징, 자동 검증, 코드 실행 등 개발생산성에서 기존 인간 작업을 뛰어넘는 수준 실현
- 향후 다양하고 복잡한 응용프로그램의 AI 기반 개발 확대 가능성을 실감할 수 있게 하는 성공 사례

## **Claude Code를 활용해 시각적으로 아름다운 다이어그램을 만드는 전체 워크플로우** 핵심 요약

- Claude Code와 같은 코딩 에이전트는 기본적으로 시각적 설명에 약하며, 시각적으로 설명하려면 추가 프롬프트와 기능 부여가 필요함
- 이를 극복하기 위해, 제작자는 자신의 전체 다이어그램 제작 워크플로우를 하나의 '스킬'로 패키징하여 자동화함
- 이 스킬을 통해 Claude Code 및 다른 코딩 에이전트가 아름답고 실용적인 Excal 다이어그램을 제작 가능하도록 함
- 핵심 지침 세트(Primary Instruction Set)를 제공, 시각적으로 논리적으로 주장하는 법을 Claude Code에 지도함
- Excal 다이어그램을 만드는 것뿐만 아니라, 생성된 다이어그램을 렌더링하고 스크린샷(PNG)으로 확인 후 불완전한 부분을 스스로 보완(자기 검사)하는 기능도 포함
- 이러한 자기 검증(Self-Validation) 프로세스 덕분에, 반복 개선을 거쳐 매우 우수한 결과물을 얻을 수 있었음
- Excalra Diagram Skill 설치는 코딩 에이전트에게 스킬 설정을 지시하면 자동화 과정으로 쉽게 가능, 사용 준비가 간단함
- 다이어그램의 브랜드 일치(예: 색상 스킴 등)나 스타일 커스텀도 가능함
- 단순히 블럭 나열이 아니라, 구조와 라벨로 전체 개념을 시각적으로 설명하는 방법을 코딩 에이전트에 학습시킴
- 제작자는 누구든지 해당 스킬을 레포(repo)에서 클론해 바로 사용해 볼 수 있으며, PDF나 코드베이스 등 어떤 대상을 시각화하는 데도 효과적임
- 이 자동화된 워크플로우 덕분에 매주 수 시간 이상의 작업 시간을 절감하고 있음

## 세부 요약 - 주제별 정리

### 코딩 에이전트는 시각적 설명에는 기본적으로 제약이 따름

- Claude Code 등 코딩 에이전트는 텍스트 중심이어서 직관적 시각 자료 생성이 약함
- 사용자가 시각적으로 특정 개념을 설명하려고 하거나, 다른 누군가에게 시각화해 보여주고 싶어도, 기본 상태에서는 만족스러운 결과물이 나오지 않음
- 좋은 다이어그램을 만들려면 다수의 프롬프트와 별도의 권한·기능을 수동으로 제공해야 함

### 전체 다이어그램 제작 프로세스를 단일 스킬로 패키징함

- 제작자는 자신의 종합적인 다이어그램 제작 워크플로우를 하나의 '스킬'로 통합 패키징함
- Claude Code 또는 기타 코딩 에이전트에 이 스킬을 추가하면, 자동화된 다이어그램 제작이 가능해짐
- 이 스킬은 실용적이면서도 아름다운 Excal Diagrams(엑스칼 다이어그램) 제작에 초점을 맞춤

### 핵심 지침(primary instruction set)으로 시각적 논증 방법을 에이전트에 지도함

- Claude Code에게 "시각적으로 주장(Argue Visually)"하는 방법을 가르치는 프롬프트 세트가 포함됨
- Excal 다이어그램 설계용 JSON 파일을 생성하는 방법을 단계별로 안내
- 기존 코딩 에이전트의 다이어그램 생성 결과물이 투박하게 블록만 나열하는 경우가 많음을 지적
- 단순한 레이아웃이 아니라, 의미 있는 구조와 명확한 라벨로 전체 주제를 시각적으로 설명하도록 설계

### Excal 다이어그램 생성 과정에 자기 검증 기능을 도입함

- 다이어그램을 만든 뒤, 곧바로 렌더링하여 실제 디자인 결과물(PNG)을 확인하도록 설정
- 생성된 이미지 파일을 직접 읽고, 시각적 불완전 항목을 반복(iterate)적으로 개선함
- 이러한 자기 검증 및 반복 개선 프로세스를 통해 완성도가 매우 높은 결과를 얻을 수 있음
- "자기 검사(self-validation)"는 코딩 에이전트의 시각적 한계를 보완하면서 퀄리티 높은 다이어그램을 생성하게 함

### Excalra Diagram Skill 설치와 사용법이 매우 간편하게 설계됨

- 코딩 에이전트에 Excalra 다이어그램 스킬 세팅을 요청하면, 에이전트가 readme 파일을 읽고 수동 설정 과정을 자동화함
- 사용자는 어렵지 않게 스킬 환경을 구축할 수 있음
- 셋업이 완료되면, 단순한 프롬프트 한 줄로 다양한 유형의 시각적 다이어그램을 바로 생성 가능함

### 시각화할 내용과 브랜드 스타일을 자유롭게 커스터마이징할 수 있음

- 사용자는 다이어그램의 색상, 브랜드 스타일, 선호하는 구조 등을 자유롭게 설정해 맞춤화 가능
- 다이어그램 생성 요청 시 자신의 브랜드 콘셉트 혹은 선호 색상 스킴 등을 적용할 수 있음

### 설계 핵심은 의미 중심의 시각 구조 설명을 적용함에 있음

- skill.mmd 파일의 프롬프트 설계 철학은 "비주얼 커뮤니케이션으로 설득력 있게 논증"하는 데 초점
- 그냥 사각형 블록만 여러 개 배치하는 것이 아니라, 전체 설명 구조와 라벨에서 해당 개념의 핵심을 시각적으로 명확히 표현하도록 유도
- 예시로, 단순한 블럭이 아닌, 구조적 의미가 드러나는 시각적 설계 추구

### 누구나 Github 레포에서 스킬을 가져와 바로 적용할 수 있음

- 사용자는 Excalra 다이어그램 스킬의 Github 저장소를 쉽게 클론하여 적용 가능함
- PDF 문서, 코드베이스 등 다양한 대상을 대상으로 시각화 작업을 진행할 수 있음
- 이 툴을 통해 단시간에 높은 퀄리티의 초안(Starting Point)을 확보할 수 있음

### 자동화로 매주 수 시간의 작업 시간이 절감됨

- 작업자가 스킬 기반 다이어그램 자동화에 적응한 뒤, 손수 하던 다이어그램 제작보다 훨씬 신속하게 작업 진행
- 매주 반복적으로 요구되는 다이어그램 업무를 수 시간씩 단축하는 효율성 확보
- 실질적으로 생산성 향상 및 작업 피로도 감소 효과를 누림

## *Claude 코드로 아름다운 다이어그램 그리기 (전체 워크플로우 공개)* 핵심 요약

- 영상은 코드 생성 에이전트(특히 Claude Code)가 시각적 작업에 약점을 보인다는 문제의식에서 출발함
- 창작자는 자신의 전체 다이어그램 생성 워크플로우를 하나의 '스킬'로 패키지화해 Claude Code 등 모든 코딩 에이전트에서 쉽게 활용 가능하도록 만듦
- 이 스킬은 Excalia(Excal) 다이어그램을 위한 JSON 파일 생성, 시각적 검증, 반복 개선을 자동화함
- 다이어그램을 생성한 뒤 PNG로 렌더링 및 스크린샷 촬영을 통해 실제 결과물을 코드 에이전트가 직접 평가하고 수정함
- 이 '자체 검증(self validation)' 루프를 통해 에이전트의 본래 약점(비시각적 성격)을 보완, 기존보다 훨씬 높은 품질의 다이어그램을 생성함
- 사용자는 단순히 Excalra 다이어그램 스킬 설치 요청만으로 자동으로 환경 설정 및 필수 매뉴얼 과정이 완료됨
- 개인화 및 브랜드 스타일 반영(예: 원하는 색상, 구조 등)도 가능함
- 다이어그램은 단순 블록 나열이 아니라, 구조와 라벨을 통해 전체 개념을 명확히 '시각적으로 주장'하도록 설계됨
- Excalra 다이어그램 스킬은 PDF, 코드베이스 등 어디든 활용 가능하여, 창작자에게 매주 수 시간의 작업 시간을 절약해줌
- 끝으로 영상은 해당 스킬의 레포지토리를 직접 클론해 사용해볼 것을 권장하며, 실제 예시 데모도 소개함

## 세부 요약 - 주제별 정리

### 코드 생성 에이전트의 시각적 한계가 업무 효율을 저해함

- Claude Code와 같은 코딩 에이전트는 뛰어난 텍스트 및 코드 생성 능력이 있지만, 시각적 설명이나 다이어그램 구성에는 명확한 한계를 보임
- 사용자 입장에서는 개념 설명이나 타인에게 전달 시 '그림'이 더욱 효과적이지만, 에이전트 기본 기능으로는 비직관적 결과물이 생성됨
- 원하는 수준의 시각적 출력물을 얻으려면 많은 프롬프트 수정 및 추가 능력(스킬) 부여가 필요함

### 시각적 주장(Arguing Visually)을 위한 전용 워크플로우가 필요함

- 제작자는 이 문제를 해결하기 위해 자신만의 전체 다이어그램 생성 프로세스를 하나의 '스킬'로 모듈화함
- 해당 프로세스를 통해 어떤 코드 생성 에이전트라도 쉽게 다이어그램 생성 업무를 수행할 수 있도록 설계됨
- 기본 아이디어는 코딩 에이전트가 "시각적으로 주장"할 수 있는 방법을 스스로 익히게 만드는 데 있음

### Excalia 다이어그램 생성 용 JSON 파일 자동화로 효율 극대화

- 다이어그램은 Excalia(약칭 Excal) 플랫폼의 규격을 따르는 JSON 파일로 생성됨
- 스킬은 일련의 명령어 집합(primary instruction set)을 에이전트에 내장해, 자동으로 해당 JSON 파일을 생성하도록 유도함
- 이 구조는 각 다이어그램 요소(블록, 라벨, 구조 등)를 일관되게 관리 가능하게 함

### 생성 다이어그램의 자동 시각적 검증 및 반복 개선 체계 구현

- 가장 핵심적인 장점은 생성된 다이어그램을 Excal로 바로 렌더링, PNG 이미지로 변환 후, 다시 코딩 에이전트가 이를 '시각적으로 점검'하는 기능에 있음
- 구체적 절차: 다이어그램 생성 → Excal에서 이미지로 렌더 → 스크린샷 → PNG 확인 및 평가 → 미흡 부분 수정 및 반복
- 인간이 직접 이미지 상태를 점검·수정하는 대신, 에이전트가 이 과정을 반복 수행하며 결과 품질을 자체적으로 높여감

### '자체 검증(Self Validation)' 루프를 통해 시각적 약점을 혁신적으로 극복함

- 일반적인 코딩 에이전트는 완성된 다이어그램의 시각적 완성도를 스스로 평가할 능력이 없음
- Excal 스킬을 통해 '자체 결과물 이미지를 직접 평가 후, 필요하면 코드를 재생성'하는 루프가 가능해짐
- 이로 인해 단순히 대충 만든 블록형 결과물이 아니라, 사용 목적에 부합하는 정교하고 아름다운 다이어그램이 자동 완성됨

### Excalra 다이어그램 스킬 설치 및 설정이 사용 경험을 크게 간소화함

- 사용자는 코딩 에이전트에게 'Excalra 다이어그램 스킬 설치'를 요청하면, 에이전트가 직접 readme를 읽고 필수 manual step까지 자동으로 진행함
- 그 결과 사용자는 별도 기술적 세팅 없이 즉시 다이어그램 생성 작업에 착수 가능
- 환경 설정에서 브랜드 컬러나 원하는 디자인 코드도 쉽게 반영 가능함

### 다이어그램의 브랜드화 및 개인적 스타일 맞춤 기능 제공

- 단순한 도형 나열이 아니라, 브랜드 전략이나 개인 취향(컬러, 구조)에 따라 다이어그램 결과물을 맞춤 제작 가능
- '블록 나열'이 아닌, 전체 개념 구조와 관계성 라벨링까지 강조함

### 핵심 프롬프트 설계로 시각적 주장의 논리 구조를 자동 반영함

- skill.mmd 내 프롬프트는 '시각적으로 주장하는' 방법을 코딩 에이전트에 체계적으로 전수함
- 예를 들어, 그냥 여러 블록을 나열하는 것을 지양하고, 전체 개념의 흐름과 관계성이 그림 구조에 드러나도록 설계됨

### 다양한 입력(코드베이스, PDF 등)에 즉시 활용 가능하여 생산성이 비약적으로 향상됨

- Excalra 다이어그램 스킬은 코드, 문서(PDF), 개념 설명 등 다양한 자료에 즉시 적용 가능함
- 복잡한 시스템, 데이터 흐름, 오브젝트 구조 등 다양한 시각화에 실질적으로 사용 가능함
- 매주 수 시간 이상의 작업 시간을 절약했다고 창작자가 직접 언급

### 누구나 직접 레포지토리 클론 후 간단히 활용 시작이 가능함

- 영상 마지막에는 해당 Excalra 다이어그램 스킬의 저장소(repository)를 클론(clone)해 테스트해볼 것을 적극 권장함
- 실제 간단한 데모(예시) 결과도 영상 내에서 소개하며, 실전 적용 난이도가 낮음을 강조

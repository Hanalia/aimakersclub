## *이 무료 툴로 Claude 코드 작업이 10배 더 쉬워짐* 핵심 요약

- 영상 제목은 "이 무료 툴로 Claude 코드 작업이 10배 더 쉬워짐"으로, Claude Code 개발 현장의 비효율을 극복하는 무료 보조 도구 ‘Kinugi’를 소개함
- Claude Code(클로드 코드)는 매우 강력하지만, 사용자 입장에서는 AI가 생성하는 코드 변경사항을 전부 검토(reivew)하는 과정이 상당히 번거로움
- 동시에 여러 개의 Claude Code 세션을 병렬로 운영하는 경우, 각 세션별 결과와 코드 변경 내역을 일일이 확인하기가 매우 불편함
- Kinugi(키누기)는 클로드 코드를 대체하는 것이 아니라 그 위에 시각적 레이어를 추가하는 방식으로, 기존 Claude Code의 작업 흐름을 그대로 활용함
- Kinugi는 ‘agentic development environment’(에이전트 중심 개발환경)로, 모든 클라우드 코드 에이전트의 병렬 실행 현황, 코드 변경, 리뷰 대상을 한 눈에 파악할 수 있음
- 각 AI 에이전트의 플랜 리뷰, 실시간 의견 코멘트, 구현 완료사항을 풀 리퀘스트 형태로 확인 가능해, 기존 로그/파일 기반 검토 방식보다 훨씬 직관적임
- Sonar팀이 개발한 만큼, AI가 생성한 코드에 대해 보안 리뷰(Security Review) 등 다양한 리뷰 방식까지 시도할 수 있음
- Kinugi는 현재 맥 OS에서만 사용 가능하며 매우 실험적인 단계(무료 제공)이나, Claude Code를 사용하는 개발자에게 큰 생산성 이득을 제공함

## 세부 요약 - 주제별 정리

### Claude Code는 매우 강력하지만 코드 리뷰 과정이 심각한 생산성 저하 요인임

- Claude Code(클로드 코드)는 실제로 사용해보면 개발 생산성에 큰 이점을 주는 매우 강력한 인공지능 코딩 도구임
- 영상 제작자는 개발의 거의 모든 영역에서 Claude Code를 사용하고 있다고 언급함
- 하지만 사용할수록 몇 가지 아쉬운 점, 특히 ‘AI가 생성한 코드를 하나하나 검토하는 과정’에서 불편함이 극대화됨
- Claude Code가 다양한 코드 변경을 빠르게 제안하는 것은 장점이나, 그 결과물 전체를 사람이 직접 검토하지 않을 수 없음
- 코드 리뷰가 반드시 필요한 이유는, AI의 출력에도 여전히 실수가 포함될 수 있기 때문임
- 여러 개의 클로드 코드 인스턴스를 동시에 실행시키는 경우가 많은데, 병렬 세션별로 결과와 변경사항을 추적하는 것이 큰 부담으로 작용함

### Kinugi는 기존 Claude Code의 비효율을 획기적으로 개선함

- Kinugi(키누기)는 ‘무료’로 제공되는 보조 툴로, Claude Code 위에 별도의 시각적 관리 계층을 제공함
- 전통적 코드 리뷰 과정을 혁신하며, 병렬 실행 중인 Claude Code 에이전트의 상태와 코드 변경 내역을 한 곳에서 직관적으로 확인 가능하게 만듦
- Kinugi는 기존 Claude Code를 대체하는 것이 아니라, 그 위에서 보완적으로 작동함(기존 워크플로우를 유지)
- 정식 명칭은 ‘agentic development environment’(에이전트 중심 개발환경)로 정의됨
- 이 플랫폼의 가장 큰 가치는 여러 병렬 세션을 한 화면에 집약해 관리할 수 있다는 점임
- 어떤 세션이 리뷰가 필요한지, 실행 중인 세션은 무엇인지, AI가 짜놓은 플랜 오브 어택(plan of attack)을 언제든 불러와 리뷰 및 코멘트 남길 수 있음

### Kinugi의 실질적인 사용 경험은 기존보다 월등히 낫다고 평가됨

- 기존 Claude Code에서는 로그를 뒤져 각 세션별로 생성된 파일을 하나하나 검토해야 했음
- Kinugi에서는 AI가 작업한 변경 결과물을 ‘풀 리퀘스트(pull request)’처럼 일목요연하게 제시해주므로, 검토 과정이 대폭 단순해짐
- AI가 생성한 계획(plan of attack)을 미리 보고 의견을 남길 수 있고, 실제 구현 완료 후 다시 한번 리뷰가 가능함
- 이를 통해 개발자는 실시간성과 효율성 모두를 확보할 수 있음

### Kinugi는 다양한 코드 리뷰와 보안 리뷰 기능을 제공함

- Kinugi에 포함된 새로운 기능 가운데 하나는 ‘AI 생성 코드에 대해 다양한 리얼타임 코드 리뷰 타입을 적용할 수 있다는 점’
- 특히 보안 리뷰(Security Review) 기능이 대표적이며, 이는 개발사 Sonar팀의 특화 분야와도 연관 있음
- Sonar팀이 제작한 만큼, 코드 품질과 보안성을 함께 점검할 수 있는 환경이 제공됨
- 영상제작자는 이 보안 리뷰 기능에 큰 만족감을 보임

### Kinugi의 현재 제공 환경과 실험적 성격

- Sonar팀은 Kinugi가 ‘맥OS에서만 동작하며 아직 매우 실험적인 단계’임을 명확하게 알리고 있음
- 제품이 무료로 제공되는 것도 이 실험적 성격과 연관됨
- 그럼에도 불구하고 Kinugi는 이미 충분한 기능성과 생산성 향상을 가져다줌
- 기존 Claude Code를 그대로 사용하면서도, Agentic 워크플로우를 유지할 수 있음

### Sonar팀이 직접 Kinugi 시연을 요청하며 영상 제작에 참여함

- 영상 제작자는 Sonar팀의 직접 제안에 따라 해당 툴을 시연해보고 이를 영상으로 공유
- Sonar가 개발한 툴 중 ‘Kinugi’가 지금까지 사용해본 AI 코딩 보조툴 중 가장 흥미롭고 쿨하다고 거듭 평가함
- 영상 하단 설명란에 Kinugi 다운로드 링크가 제공됨

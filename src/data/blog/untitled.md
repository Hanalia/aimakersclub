## *RAG 2.0: 에이전틱 RAG와 지식 그래프 결합(무료 템플릿 소개)* 핵심 요약

- 영상의 제목은 "RAG 2.0: 에이전틱 RAG와 지식 그래프 결합(무료 템플릿 소개)"로, 두 가지 최신 RAG(검색-확장-생성) 기술을 결합한 강력한 AI 에이전트를 소개함
- 제작자는 다양한 RAG 전략을 실험해 왔으며, 그중 에이전틱 RAG와 지식 그래프가 가장 유용한 것으로 판단
- 영상에서 직접 만든 에이전트는 두 방법을 결합하여, 질문의 유형에 따라 정보를 얻을 최적의 경로를 스스로 판단하고 사용함
- 벡터 데이터베이스(Postgres + PG Vector, Neon 서버리스 플랫폼)를 이용해 문서를 임베딩하여 저장함
- 문서에는 빅테크 기업들의 AI 이니셔티브와 상호 파트너십 정보가 포함되어 있음
- 동일한 문서를 지식 그래프(Graffiti, Neo4j 활용)에서도 관계 데이터로 저장하여, 관계성 질의가 가능함
- 에이전트는 단일 회사와 관련된 질문에는 벡터 DB만 참조하고, 관계형 질문(예: 기업 간 협력)에는 지식 그래프를 활용
- 사용자의 요청에 따라 두 데이터 소스를 동시에 활용할 수도 있음
- 이러한 에이전트의 구현 방법과 템플릿 설치법은 무료 영상 링크를 통해 확인할 수 있음

## 세부 요약 - 주제별 정리

### 제작자는 다양한 RAG 전략을 실험하며 에이전틱 RAG와 지식 그래프의 시너지가 최고임을 확인함

- 최근 몇 달 동안 거의 모든 RAG(검색-확장-생성) 전략을 집중적으로 테스트함
- 반복적으로 가장 효과적이라고 느낀 방법은 "에이전틱 RAG"와 "지식 그래프"임
- 이 두 가지 방식을 결합하면 각기 다른 정보 접근 방식의 장점을 최대한 활용 가능함

### 에이전트는 질문에 따라 최적의 정보 검색 경로를 스스로 판단하도록 설계됨

- 제작된 AI 에이전트는 주어진 질문에 대해 어떤 정보원(벡터 vs. 그래프)에 접근할지 합리적으로 결정함
- 정보 검색의 최적성을 자동으로 추론함으로써 사용자 경험 및 정답률이 크게 향상됨
- 질문의 맥락과 필요성에 맞추어 벡터 DB 또는 지식 그래프, 혹은 둘 다를 선택적으로 활용함

### 벡터 데이터베이스 구축은 Postgres + PG Vector, Neon 서버리스 플랫폼을 활용함

- 벡터 DB 구축을 위해 Postgres에 PG Vector 확장을 적용
- 서버리스 Postgres 플랫폼으로 Neon을 사용해 손쉽게 인프라를 관리함
- 문서를 여러 조각(chunk)으로 분할 후 임베딩(Embedding)을 생성하여 저장함

### 벡터 데이터베이스에는 빅테크 기업들의 AI 이니셔티브 및 협력 정보를 포함함

- 저장된 문서에는 대형 테크 기업들의 AI 관련 이니셔티브(전략·프로젝트 등)가 상세하게 담겨 있음
- 각 기업 간 AI 분야 내 협력 관계(파트너십) 등도 다수 수록됨
- 벡터 DB에 임베딩됨으로써, 특정 기업의 정보를 손쉽게 검색·활용 가능함

### 동일 정보를 Graffiti 및 Neo4j 기반 지식 그래프로도 저장하여 관계형 쿼리가 가능함

- 같은 문서를 지식 그래프 방식으로도 저장, 관계형 정보 구조를 가짐
- 지식 그래프는 Graffiti와 Neo4j를 사용하여 구축됨
- 예시: "Amazon은 Anthropic이 모델을 호스팅하는 곳"과 같은 기업 간 관계를 질의 가능함

### 에이전트는 단순 검색 질문에는 벡터 DB만 활용하고, 관계성 질문에는 지식 그래프를 활용함

- 예시1: "Google의 AI 이니셔티브가 무엇인가?"와 같은 단일 기업 질문시 벡터 DB만을 탐색함
- 해당 질의에서는 벡터 DB를 사용했다는 정보를 명확히 반환함
- 예시2: "Amazon과 Anthropic의 파트너십 관계"처럼 관계성 질문을 할 경우 지식 그래프를 사용함
- 어떤 도구가 활용되었는지(그래프 vs 벡터) 결과에 명시적으로 보여줌

### 사용자는 시스템 프롬프트나 직접 지시로 두 소스를 동시에 활용할 수 있음

- 시스템 프롬프트를 수정해 두 소스를 병행적으로 쿼리하도록 지정 가능
- 또는 명령을 통해 "벡터 DB와 지식 그래프 모두 사용"을 요청할 수 있음
- 이 경우 에이전트가 두 데이터 소스를 동시에 참조해 보다 탄탄한 답변 도출

### 프리 템플릿은 Cloud Code를 활용해 개발되었으며, 누구나 무료로 사용 가능함

- 영상 내 에이전트 구현은 Cloud Code를 사용하여 개발 생산성을 높임
- 전체 소스 및 구축 방법은 무료 템플릿으로 제공됨
- 전체 구현 방법과 설치 가이드는 본 영상의 전체 버전 링크에서 확인 가능함

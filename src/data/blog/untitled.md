## *Claude에게 24시간 동안 연속적으로 코딩을 시켜본 결과* 핵심 요약

- 영상에서는 Claude Opus 4.5 모델을 활용해 24시간 동안 쉬지 않고 인공지능이 자체적으로 애플리케이션을 개발하는 과정을 보여줌
- 현재까지 구현된 결과는 claw.ai(클로우 AI)와 매우 유사한 완성도 높은 클론임을 확인함
- 수많은 테스트를 거치며 대화 기록과 검증 과정을 모두 자동으로 생성, 추적할 수 있음
- 마크다운 포맷팅이 뛰어나며, 다양한 HTML 페이지 생성 및 코드 작성·실행도 자동화됨
- 테마, 기본 모델, 최대 토큰 수(슬라이더 형태) 등 세부 설정 기능도 자체 구현됨
- 전체적으로 기능이 매우 풍부하고 사용자 경험이 우수하게 구성됨
- 아직 완벽히 구현된 것은 아니며, 다음 목표로 ‘휴먼 인 더 루프(Human in the Loop)’ 기능 추가를 계획 중임
- 개발자는 본인이 직접 손으로 코드를 짜지 않고도 이 정도 수준이 가능하다는 점을 ‘놀라움’과 ‘가능성’으로 강조함
- 만약 인공지능이 24시간 더 개발을 이어간다면 사실상 완벽에 가까운 애플리케이션도 가능할 것이라 시사함
- Claude Opus 4.5와 같이 강력한 모델과 자동 검증·반복 시스템(하니스)를 활용하면, 최소한의 인간 개입으로 실질적 애플리케이션 완성에 근접할 수 있음을 시연함

## 세부 요약 - 주제별 정리

### Claude는 하루 만에 기능이 풍부한 claw.ai 클론을 거의 완성함

- 영상 제작자는 Claude Opus 4.5에게 24시간 동안 코딩 작업을 맡김
- 인공지능이 구현한 결과물은 기존 claw.ai 애플리케이션과 거의 구분이 어려울 정도로 완성도가 높았음
- 주요 기능, UI, 인터랙션 등이 이미 대부분 구현되어 있다는 평가를 직접 밝힘
- 실시간 테스트를 반복하며 기능의 정상 동작을 계속 검증할 수 있었음

### 자동화된 대화 기록과 검증 시스템이 개발 과정을 투명하게 만듦

- 애플리케이션은 개발 및 테스트 단계에서 거친 모든 대화를 기록함
- 각 단계별로 모델의 검증 루프(loop)가 명확히 남아 있어 개발 내역을 쉽게 추적 가능함
- 이전에 어떤 대화와 데이터가 오갔는지 손쉽게 돌아볼 수 있게 설계됨

### 마크다운 기반으로 문서화 및 가독성을 높임

- 생성된 페이지, 문서 및 테스트 결과들은 모두 마크다운 형식으로 잘 포맷팅되어 있음
- 코드, 대화 기록, 안내문 등이 보기 좋게 정돈되어 사용자와 개발자 모두에게 직관적으로 정보 전달이 가능함

### HTML 페이지 생성과 코드 실행까지 자동화되어 있음

- Claude가 직접 HTML 페이지 여러 개를 생성하고, 임의로 웹 디자인과 구조를 구축함
- 필요한 코드를 작성할 뿐 아니라, 실행 및 결과 확인도 인공지능이 자체적으로 수행함
- 개발자는 화면을 통해 결과만 확인하며 개입 없이 전체 프로세스를 감상함

### 사용자 설정 기능이 이미 세밀하게 탑재되어 있음

- 테마(예: 다크모드, 라이트모드 등) 변경 기능이 포함됨
- 사용자가 기본적으로 선택할 수 있는 모델을 변경하는 옵션도 구현됨
- 슬라이더를 통해 최대 토큰 수(모델의 응답 길이 제한)까지 세부적으로 조정할 수 있는 옵션이 있음

### 기능의 풍부함과 완성도 면에서 개발자의 기대를 뛰어넘음

- "so so featurerich"(기능이 너무 많을 정도로 풍부하다)는 표현으로, 단기간 내 구현된 기능 양에 감탄함
- 테스트 역시 대부분이 통과된 상태라, 남은 절반은 무엇을 위한 테스트인지 의구심이 들 정도라고 평가함
- 이처럼 빠르며 구석구석 디테일한 구현이 인공지능 만으로 가능함을 실제 과정으로 보여줌

### 앞으로의 과제로 ‘휴먼 인 더 루프’ 요소 도입을 언급함

- 현재의 시스템은 전적으로 인공지능이 자동 반복, 검증, 코드 작성 등을 담당
- 다만, 최종 단계에서는 인간 검수자 혹은 피드백 과정을 추가하는 ‘Human in the Loop’ 기능을 넣을 계획임을 언급
- 인공지능 중심의 개발에 인간 개입 방식을 더해, 실사용에 더욱 적합하도록 보완할 의도를 밝힘

### 개발자는 전혀 코드를 직접 작성하지 않고서도 애플리케이션 구축에 성공함

- 본인은 실제로 "손가락 하나 까딱하지 않았다"고 직설적으로 언급함
- Claude와 자동화된 하니스 시스템이 사람의 행동(코딩, 테스트, 동작 확인)을 완전히 대체한 셈임
- 이를 통해 완전 자동화된 개발이 이미 상당 부분 실현됐음이 드러남

### 더 많은 시간이 주어질 경우 완벽에 가까운 결과물을 기대할 수 있음을 강조함

- 만약 인공지능에게 24시간만 더 시간을 준다면, 남은 기능까지 모두 더해 ‘완전무결’에 가까운 애플리케이션을 만들 수 있을 것임을 암시
- Claude Opus 4.5의 강력함과 반복적 자동 검증 시스템의 궁합을 높이 평가함

### Claude Opus 4.5 및 자동 검증 하니스의 결합이 개발 효율을 극대화함

- Claude Opus 4.5와 같은 최신 대형 언어모델의 능력에, ‘하니스(harness)’로 불리는 자동화 프레임워크가 더해질 때의 시너지를 실제로 입증함
- 하니스 시스템은 반복적 테스트, 기능 검증, 코드 실행을 자율적으로 담당
- 덕분에 최소한의 인간 개입만으로 완성도 높은 소프트웨어 애플리케이션이 빠르게 완성될 수 있음을 시연함

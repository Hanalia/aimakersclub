**영상 링크:** [This Free Tool Makes Claude Code 10x Better](https://www.youtube.com/shorts/FDYhPXYDS_o)  
**채널명:** Cole Medin

## *이 무료 도구로 Claude 코드의 검토와 관리가 10배 더 쉬워짐* 핵심 요약

- 영상 제목은 “This Free Tool Makes Claude Code 10x Better”이며, Claude(Code) 활용자의 주요 고민과 이를 해결하는 신생 툴을 소개함
- 영상 제작자는 Claude 코드의 출력물을 리뷰하는 과정이 번거롭고, 특히 여러 인스턴스를 동시에 다룰 때 불편함을 강조
- 개발자가 겪는 가장 큰 문제는 Claude 코드의 여러 결과물을 일괄적으로 효율적으로 검토하는 것임
- 제작자는 최근 ‘Kinugi(기누기)’라는 무료 도구를 사용하며 이러한 불편함을 크게 해소했다고 설명
- Kinugi는 Cloud Code 위에서 동작하는 에이전틱(Agentic) 개발 환경으로, 기존 Cloud Code를 대체하지 않고 보완하는 구조임
- Kinugi를 통해 여러 개의 클라우드 코드 세션을 한 눈에 확인하고, 리뷰가 필요한 부분이나 진행 중인 작업을 빠르게 파악 가능
- AI가 생성한 변경 내역, 계획, Pull Request 등을 시각적으로 한 곳에서 확인 및 리뷰, 실시간 코멘트가 가능함
- 특히 Kinugi에서는 AI가 생성한 코드에 보안 리뷰(Security Review) 등 다양한 코드 리뷰 옵션을 추가할 수 있음
- 이 툴은 Sonar 팀이 제작했고, 현재 Mac OS 전용(무료, 실험적 단계)이지만 이미 강력한 기능을 제공함
- 영상 마지막에는 Sonar의 Kinugi 도구 제공에 감사 인사를 전하며, 시청자에게도 사용을 적극 권장

## 세부 요약 - 주제별 정리

### Claude 코드의 강력함과 함께 드러나는 치명적인 리뷰 번거로움

- Claude Code는 개발 작업에 있어 매우 강력한 도구로, 제작자는 현재 모든 개발에 이를 사용 중임
- 코드 생성 및 변경 작업을 Claude에 맡기면 업무 속도가 크게 향상됨
- 하지만, Claude가 생성하는 다양한 코드 변경 사항을 개발자가 반드시 검토해야 하는 단계에서 문제가 발생함
- 특히 Claude 코드 인스턴스를 동시에 여러 개 돌리는 경우, 각 인스턴스별 산출물을 모두 리뷰하는 것이 매우 번거롭고 시간이 많이 든다고 설명
- “Cloud code logs”를 일일이 뒤져 각각의 파일과 변경내역을 확인하는 과정에서 피로감이 커짐
- Claude 코드를 제대로 쓰려면 출력물 검토(Review)가 필수라며, 이 부분이 ‘가장 큰 불편함’이라고 강조함

### Kinugi는 Cloud Code를 대체하지 않고 보완하여 시각적으로 효율을 극대화함

- Kinugi(기누기)는 최근 출시된 무료 툴로, 기존 Claude Code 위에서 동작함
- Kinugi는 Cloud Code의 기능을 대체(replace)하지 않고 ‘위에 겹쳐진 시각적 레이어’로 작동한다고 설명
- “Agentic development environment(에이전틱 개발 환경)”라는 컨셉을 도입, 여러 Claude 코드 에이전트를 시각적으로 관리할 수 있음
- 기존 Claude Code 워크플로우를 그대로 유지하면서 Kinugi를 추가로 활용 가능함
- Kinugi의 핵심 목표는 “모든 병렬 세션을 한 곳에서 시각적으로 관리”하는 것임

### 여러 Claude 코드 인스턴스를 한 번에 관리하여 리뷰 환경을 혁신함

- Kinugi 도입 후 가장 만족스러운 점은 “모든 병렬 세션을 하나의 창에서 모두 볼 수 있다”는 점임
- 각 세션별로 어떤 것이 리뷰 대기 상태인지, 어떤 것이 아직 실행 중인지 즉시 파악이 가능해짐
- Claude의 에이전트가 “계획(plan of attack)”을 생성하면 이를 한눈에 보고, 실시간으로 코멘트 및 피드백이 가능함
- 구현(Implementation)이 끝나고 난 뒤 리뷰가 다시 개발자에게 넘어오면, 기존의 “log 파고들기” 대신, Pull Request와 비슷한 방식으로 전체 변경 내역을 시각적으로 검토함
- 코드 파일을 하나씩 따로 여는 수고를 덜 수 있어 업무 효율성이 대폭 상승함

### 다양한 코드 리뷰 옵션과 보안 점검 과정을 AI 코드에 바로 적용할 수 있음

- Kinugi는 AI가 생성한 코드에 대해 다양한 형태의 코드 리뷰를 추가로 적용할 수 있도록 지원함
- 특히 보안 리뷰(Security Review) 기능이 내장되어 있음
- 이는 Kinugi 제작사인 Sonar 팀이 “코드 품질 및 보안 툴”로 유명하기 때문에 자연스럽게 포함된 기능임
- 이 외에도 여러 종류의 리뷰 옵션을 직접 적용하거나 병렬 실행이 가능함
- 보안 및 품질 측면에서 AI 코드에 대한 신뢰성을 한 단계 높여주는 역할을 함

### Kinugi는 현재 Mac OS만 지원하며 실험적 단계지만, 무료로 이용 가능함

- Sonar 팀에서 만든 Kinugi는 현재 “Mac OS에서만 사용 가능”함
- 아직 베타/실험적(very experimental) 버전이라 공식 지원이 제한적임
- 이런 이유로 현 시점에서는 무료로 배포되고 있다고 명시
- 그럼에도 불구하고 Kinugi가 제공하는 기능과 경험은 이미 현존하는 대안 대비 매우 강력하다고 평가

### Sonar 팀과 제작자의 협업 체험, 그리고 시청자에게 도구 사용 적극 추천

- Sonar가 Cole Medin(제작자)에게 Kinugi 사용 제안 및 시연 요청을 했고, 제작자는 즉시 이를 받아들여 체험함
- 실제 체험 결과, Kinugi의 기능성과 혁신성을 크게 실감했음을 표현
- 영상 마지막에는 Sonar 및 Kinugi에 대한 감사를 전하며 직접 써볼 것을 시청자에게 권유
- 영상 설명란에 Kinugi 관련 링크를 남겼다고 언급함

## *Claude에게 24시간 동안 쉬지 않고 코딩을 시켰더니 벌어진 일* 핵심 요약

- 이 영상은 Cole Medin이 Claude Opus 4.5 인공지능에게 24시간 동안 쉬지 않고 코딩을 시켰을 때 어떤 결과물이 나오는지 직접 실험한 내용을 다룸
- Claude가 주어진 시간 내에 claw.ai라는 웹앱의 완전한 클론을 제작하는 과정을 보여줌
- 실험자의 말에 따르면, 테스트의 마지막 반도 가지 않았는데 이미 claw.ai의 모든 주요 기능이 구현되었다고 평가함
- 생성된 서비스는 과거 대화 기록을 추적하고, 검증 과정을 거치며 각각의 루프를 반복하는 작업 흐름을 가짐
- 마크다운 형식의 문서화와 다양한 HTML 페이지 생성, 코드 작성 및 실행까지 모두 가능
- 사용자 인터페이스에서는 테마, 기본 모델 변경, 최대 토큰 수를 조절하는 슬라이더 등 다양한 설정 메뉴를 제공함
- 매우 다양한 기능이 구현되어 있고, 실험 참여자는 코드에 손을 대지 않아도 될 정도로 자동화가 잘 되어 있음을 강조함
- 아직 ‘human in the loop’(사람의 검증/개입) 기능은 추가해야 하지만, 현재 결과만으로도 놀라운 생산성에 감명을 받음
- Claude Opus 4.5와 이 툴의 조합이 앱의 검증 및 반복 개발을 얼마나 효율적으로 해낼 수 있는지 보여줌
- 만약 하루만 더 이대로 굴린다면 완벽한 애플리케이션도 만들 수 있을 정도로 발전한 모습 소개

## 세부 요약 - 주제별 정리

### Claude가 claw.ai 클론을 24시간 만에 거의 완벽하게 제작함을 직접 확인함

- 실험자는 Claude에게 실제로 claw.ai라는 서비스를 똑같이 만드는 코드 작성을 일임함
- 전체 테스트의 절반도 가지 않았음에도, 주요 기능들이 모두 구현되었다고 밝힘
- “완전히 동작하는 claw.ai 클론을 이미 가진 느낌”이라고 직접 평가함

### 검증 과정과 대화 기록 관리 등 유지보수 기능이 자동으로 구현됨

- Claude가 주어진 목표에 따라 각 기능을 만들 때, 검증 단계마다 과거 대화와 작업 내역을 추적함
- 반복적인 구조(루프) 내에서 문제 검증과 피드백 과정을 계속 수행함
- 이전에 생성된 대화 기록을 사용자가 쉽게 볼 수 있도록 시스템이 구축되어 있음

### 마크다운 및 코드 실행, HTML 페이지 생성 등 다기능 지원이 확인됨

- 생성된 웹앱 내에서 마크다운(Markdown) 형식을 사용한 문서화 인터페이스 지원
- 다양한 HTML 페이지를 동적으로 만들 수 있도록 설계됨
- 코드까지 직접 작성하고 실행하는 기능을 갖추고 있음을 실험자가 시연함

### 사용자 설정 인터페이스가 풍부하게 구현되어 있음

- “Settings”(설정) 메뉴를 통해 여러 환경 및 기능을 직접 선택 가능함
- 테마(Theme) 변경, 기본 모델(Default Model) 전환 등 사용자 경험 향상을 위한 옵션이 추가됨
- 최대 토큰 수(Max tokens)를 조절할 수 있는 슬라이더 UI도 제공됨

### 기능의 다양성과 자동화 수준이 매우 높아 인간 개입 없이 구현이 가능함

- 실험자는 거의 아무런 수작업 없이 AI가 웹앱의 핵심 기능을 대부분 구현한 점에 놀라움을 표함
- “손 하나 까딱하지 않아도 될 정도”라는 언급에서 자동화 완성도가 강조됨
- 남은 과제로 ‘human in the loop’(사람의 직접 검수/피드백 기능)만 추가하면 된다고 밝힘

### Claude Opus 4.5가 검증과 반복 개발에 강점을 보였음을 직접 체험함

- Claude에게 harness(자동 테스트 및 검증 프레임워크)를 제공해 실시간으로 애플리케이션을 점검하고 개선하게 함
- 검증(validation)과 반복(iteration) 과정을 AI 스스로 처리하여 개발 주기의 효율성이 크게 증가했음
- “이런 harness가 있으면 앱의 검증 및 반복 개발이 매우 용이하다”는 소감 공유

### Claude와 harness 조합으로 완벽에 가까운 자동화 개발 사례를 보여줌

- 조금 더 시간이 주어진다면, 완전무결에 가까운 애플리케이션 제작이 가능할 것으로 기대함
- “하루만 더 돌리면 완벽한 앱이 나올 것 같다”는 결론으로 실험을 마무리함

### AI를 활용해 '코드 작성부터 검증 및 반복개발'까지 하나의 시스템에서 처리 가능함을 입증함

- 웹앱 기획, 코드 작성, 실행, 테스트, 사용자 설정 UI까지 전체 개발 프로세스를 Claude가 한 번에 해낼 수 있었음
- 복잡한 기능 추가나 수정에도 사람의 손길이 거의 필요 없었음을 강조함

### 실험 결과, Claude Opus 4.5가 AI 기반 자동화 개발의 잠재력을 강력히 시사함

- 코딩 자동화부터 복잡한 기능 통합, UI 구현 등 현존 AI 자동화의 현실적 한계를 넘어섬
- AI와 인간의 협력 필요성을 인정하면서도, 현재 상태만으로도 높은 완성도와 기능성을 증명함

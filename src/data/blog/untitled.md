## *Claude에게 24시간 동안 쉬지 않고 코딩을 시켰더니 벌어진 일* 핵심 요약

- 영상에서는 Claude Opus 4.5 AI에게 24시간 동안 계속해서 코딩과 애플리케이션 개발을 맡긴 실험 과정을 다룸
- Claude는 claw.ai의 완전한 복제 수준의 애플리케이션을 거의 자동으로 만들어냄
- 테스트 단계 절반이 무엇을 위한 것인지조차 모를 만큼 이미 완성도 높은 결과물을 생성함
- 자동 반복 루프를 통해 과거 대화 및 검증 과정을 차례로 확인 가능함
- 마크다운(Markdown) 포맷팅, HTML 페이지 생성, 코드 작성 및 실행 같은 다양한 기능이 구현됨
- 테마, 기본 모델 변경, 최대 토큰 수 조정 등 상세 설정이 가능한 설정 메뉴도 포함됨
- 계획하고 있는 추가 기능으로는 ‘human in the loop’(사람의 개입) 추가가 언급됨
- 개발자가 직접 손을 대지 않고도 상당히 풍부한 기능을 구축한 점을 강조함
- 하루 더 Claude에게 개발을 맡기면 ‘완벽한’ 애플리케이션이 될 수도 있다고 평가함
- Claude Opus 4.5와 같은 AI 모델이 검증(Validation) 및 반복 개발(Iteration)을 자동화하는 능력의 한계를 뛰어넘었다고 언급함

## 세부 요약 - 주제별 정리

### Claude에게 24시간 동안의 자동 코딩을 맡기니 claw.ai의 거의 완벽한 복제품이 탄생함

- 영상 제작자는 Claude Opus 4.5에게 하루 동안 쉬지 않고 코딩과 애플리케이션 제작을 맡기는 실험을 실시함
- Claude는 claw.ai 서비스와 거의 동일한 수준의 기능을 갖춘 복제 애플리케이션을 자동으로 구현함
- 개발자는 테스트 절차의 절반이 필요 없을 정도로 응용 프로그램이 이미 완전히 동작한다고 느낄 만큼 완성도가 높다고 평가함
- ‘진짜로 손 하나 까딱하지 않고 이만큼 완성되었다’고 언급하며 Claude의 생산성과 자율성에 놀라움을 표함

### 반복 루프와 검증 과정을 통해 대화 이력 및 개발 진척 상황을 체계적으로 관리함

- 애플리케이션 구현 과정에서 Claude는 여러 반복 루프를 사용해 기능을 점검, 검증하고 결과를 저장함
- 이전에 발생한 모든 대화와 검증 결과를 개발자가 추적·확인할 수 있음
- 각 기능 개발 시 어떻게 검증하고 피드백을 반영했는지 기록이 남아 있음

### 마크다운 포맷팅 구현, HTML 페이지 생성, 코드 작성 및 실행 등 다양한 기능이 무인으로 완성됨

- Claude는 마크다운(Markdown) 형식을 효과적으로 지원하여 가독성 높은 문서화 기능을 첨부함
- HTML 페이지를 동적으로 생성하고, 코드의 작성 및 실행까지 스스로 처리함
- 이러한 기능들이 AI에 의해 전자동으로 구현되어 실제로 화면에 적용됨

### 사용자 맞춤 설정 메뉴를 통한 세밀한 옵션 조절 기능도 포함함

- 애플리케이션 내에는 테마 변경 기능, 기본 사용 모델 변경, 최대 토큰 수 조정 슬라이더 등이 포함됨
- 이 설정 메뉴를 통해 사용자는 세부적인 운영 환경을 손쉽게 커스터마이즈할 수 있음
- ‘설정’ 기능의 디테일 역시 AI가 스스로 구현한 것임을 강조

### 남은 과제로 ‘human in the loop’ 구현 필요성을 인식함

- 영상 제작자는 현재까지 인간의 직접적 개입이 없었지만, 향후에는 사람이 판단하고 개입할 수 있는 루프(‘human in the loop’) 추가가 필요하다고 언급
- 인간의 개입을 통해 더욱 완성도 높고 안전한 애플리케이션 개발이 가능할 것으로 기대함

### Claude의 자동화 개발 능력에 대한 제작자의 놀라움과 만족감이 드러남

- 개발자는 ‘직접 손을 대지 않고 이렇게 다양한 기능을 구축했다’는 점에 극찬을 보냄
- Claude가 하루만 더 개발을 지속하면 사실상 ‘완벽한 애플리케이션’이 나올 수도 있다는 기대가 언급됨
- Claude Opus 4.5가 가진 자동 검증 및 반복 기능(harness)이 개발 생산성 급증을 이끌었다고 지적함

### Claude Opus 4.5의 검증 및 반복 개발 자동화 능력이 특히 돋보였음

- 애플리케이션 개발 과정에서 Claude의 검증 및 반복 자동화(harness) 기능이 매우 강력함이 드러남
- 반복적으로 기능을 만들고, 검증하고, 수정을 거치며 빠르게 완성도를 높임
- 제작자는 “정말 미친 것 같다(crazy the kinds of things we can build with Claude Opus 4.5)”라며 Claude의 가능성을 치켜세움

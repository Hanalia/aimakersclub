## *Claude 코드로 아름다운 다이어그램을 만드는 전체 워크플로우* 핵심 요약

- 이 영상은 Claude Code 같은 코드 에이전트가 시각적 설명에 약하다는 한계를 지적하며, 이를 극복하기 위한 실질적인 워크플로우를 제시함
- 제작자는 자신이 개발한 ‘Excal 다이어그램 스킬’을 코드 에이전트에 적용하여, 명확하고 실용적인 다이어그램을 만드는 과정을 소개함
- 이 스킬은 코드 에이전트가 다이어그램을 만드는 것뿐만 아니라, 결과물을 PNG로 렌더링하고 스스로 오류를 검증·수정하는 기능도 포함
- “self validation(자기 검증)”을 통해, 다이어그램 완성도를 높여 반복 작업 시간을 크게 절감했다고 강조
- Excal 다이어그램 스킬 설치 과정을 코드 에이전트가 스스로 README를 따라가며 자동화 가능
- 스킬 덕분에 사용자는 간단한 프롬프트만으로 자신만의 색상/브랜드로 구성된 커스텀 다이어그램을 쉽게 생성 가능
- 핵심 철학은 “시각적으로 주장(argue visually)”하는 것—즉 전체 개념을 구조와 레이블로 설명하는 다이어그램 생성법을 코드 에이전트에 학습시킴
- 과거에는 별도의 스킬 없이 단순한 블록 나열에 그쳤지만, Excal 스킬로 명확성과 디자인이 크게 향상됨
- 누구나 이 워크플로우(repo)를 클론해 PDF, 코드베이스 설명 등 다양한 시각적 작업에 즉시 활용 가능
- 결과적으로 매주 수 시간에 달하는 작업 시간을 단축하는 등 실질적인 업무 효율을 증명함

## 세부 요약 - 주제별 정리

### 코드 에이전트는 시각적 설명에 취약하다는 한계를 지님

- Claude Code와 같은 코드 에이전트는 주로 텍스트 기반 작업에 최적화되어 있음
- 사용자가 시각적 설명이나 그림을 요청하면, 자연스러운 시각화엔 한계가 있음
- 원하는 수준의 결과를 얻기 위해선 세밀한 프롬프트 작성, 추가 기능 부여 등 추가 작업이 필요함
- 많은 코드 에이전트가 기본적으로 다이어그램 생성에 약점을 보임

### Excal 다이어그램 스킬로 전체 워크플로우를 통합·자동화함

- 제작자는 기존의 복잡한 프롬프트 과정을 패키지화해 ‘스킬’로 제공함
- 이 스킬은 Claude Code·기타 코드 에이전트에 적용하여 전체 다이어그램 제작 워크플로우를 자동화함
- 코드를 통한 논리적 설명 및 시각적 자료 제작을 하나의 흐름으로 연결함
- 실용성과 미적 완성도를 동시에 추구하는 접근법임

### 코드 에이전트가 시각적으로 주장(Argue Visually)할 수 있도록 가르침

- 핵심 지침은 “시각적으로 주장하는 방법”을 코드 에이전트에 내재화하는 것임
- 구조 자체와 레이블·색상 등을 활용해 개념을 명확하게 전달하도록 설계됨
- 스킬의 주요 프롬프트는 Excal 다이어그램이 단순 힙스러운 블록만 놓이는 것을 피함
- 사용자 또는 청중이 한눈에 이해할 수 있는 시각적 사고 흐름을 생성하게 함

### Excal 다이어그램 스킬의 작동 방식—JSON 생성 및 자기 검증 프로세스 포함

- 스킬은 처음에 Excal 다이어그램용 JSON 파일을 자동 생성함
- 생성된 JSON은 Excal에서 다이어그램으로 렌더링되어 PNG 이미지로 변환됨
- 이 PNG 이미지를 코드 에이전트가 스스로 확인(스크린샷 분석)함으로써 다이어그램의 불완전한 부분을 스스로 찾음
- 이후 이상이나 오류가 있으면, 반복적으로 수정하여 완성도를 높임(자기 검증 루프)
- 자동화된 검수 과정을 통해 높은 품질의 결과물이 효율적으로 만들어짐

### 코드 에이전트가 스킬 설치 단계도 자동화할 수 있음

- 사용자는 스킬 설치를 코드 에이전트에 지시하는 것만으로 시작할 수 있음
- 에이전트가 repo의 README 문서를 인식, 직접 필요한 수동 단계를 차례로 진행
- 설치 이후 사용자는 ‘다이어그램을 만들어 달라’는 지시만 내리면 모든 준비가 완료됨
- 별도의 세세한 세팅 없이 바로 활용 가능함

### 브랜드 컬러나 커스텀 스타일도 손쉽게 적용 가능함

- 각 작업자의 브랜딩 컬러나 작업 선호 스타일을 손쉽게 반영할 수 있음
- 프롬프트 내에 원하는 색상, 스타일, 폰트 등을 명시할 수 있음
- 따라서 맥락에 맞는 일관성 있는 다이어그램을 지속적으로 생성 가능
- 예시에서는 사용자 선호 색채로 브랜드 지정 다이어그램 생성 가능성도 강조함

### 기존에는 ‘블록 나열’에 그치던 다이어그램 품질이 뚜렷하게 개선됨

- 스킬 없이 코드 에이전트가 만든 그림은 단순히 블록이 반복적으로 나열된 형태에 그침
- Excal 스킬 도입 시 부드러운 구조, 레이블, 컬러, 설명 요소 등 완성도가 높아짐
- 시각전달력 측면에서 월등한 향상을 보여줌
- 실제 데모(언급)에서 단순 작업 대비 큰 차이를 확인함

### 누구나 repo를 클론하여 다양한 용도에 즉시 활용 가능함

- 제작자는 “repo를 클론하고 skill을 가져가 바로 써보라” 권장
- 예시로는 PDF 문서 요약, 코드베이스 시각화 등 다양한 형태의 다이어그램 작업이 있음
- 텍스트·코드·논리 구조 등 복잡한 컨텐츠도 시각적으로 쉽게 설명 가능
- 활용 가능성에 사실상 제한이 없다고 언급함

### 반복적 자기 검증 및 자동화 덕분에 업무 효율과 시간을 대폭 단축함

- 자동화 워크플로우는 제작자 본인 기준 매주 수 시간 이상의 작업 시간을 절약함
- 실험 결과, 자기 검증 프로세스가 결과 완성도와 업무 스트레스를 동시에 개선함
- 다이어그램 생성의 진입장벽을 크게 낮춤

### 전체 철학은 “시각으로 주장하라”는 목표에 기반함

- 작업 본질은 “시각적으로 핵심을 전달하는 방법”을 코드 에이전트 수준에서 구현하는 것
- 기능적 장점(자동 세팅, 자기 검증, 시간 절약)은 이 본질적 목표(시각적 주장) 달성을 지원하는 수단임
- 사용자의 커스텀 요구, 다양한 협업 환경에도 쉽게 녹여낼 수 있음

## *클로드에게 24시간 동안 쉬지 않고 코드를 짜게 한 결과* 핵심 요약

- 영상의 실험 목표는 Claude Opus 4.5를 활용해 24시간 동안 인간의 개입 없이 웹 애플리케이션을 개발하도록 하는 것이었음
- 개발된 애플리케이션은 claw.ai와 같은 기능을 제공하는 완전히 동작하는 클론 수준이며, 기능 구현이 높은 수준에 도달했다고 평가됨
- 테스트 코드의 절반 이상이 실제로 어떤 용도인지 모를 정도로 이미 기능이 완벽에 가깝다고 언급
- Claude가 각 루프에서 검증을 수행하면서 생성된 이전 대화 기록을 모두 조회할 수 있도록 구현됨
- 뛰어난 마크다운 포맷팅 지원, 다양한 HTML 페이지 생성, 코드 작성 및 실행이 모두 가능함
- 테마와 기본 모델, 그리고 최대 토큰을 조절할 수 있는 슬라이더가 포함된 설정 기능을 추가함
- 직접 손을 대지 않고 이 정도까지 구현된 사실에 놀라움을 표시하며, 여전히 Human-in-the-loop 기능 추가 의도가 있음을 밝힘
- 추가로 하루 더 구현을 진행한다면 완벽에 가까운 애플리케이션도 가능하다고 평가
- Claude Opus 4.5와 앱의 반복적 검증(harness) 방식이 기능 개발 및 개선에 매우 강력하게 작동함을 강조
- 전반적으로 Claude 모델을 활용한 자동 개발 및 검증 방식의 혁신성과 효과를 극찬

## 세부 요약 - 주제별 정리

### Claude Opus 4.5로 24시간 자동 개발을 진행한 목적과 실험 방식

- 본 실험은 Claude Opus 4.5를 활용해 24시간 동안 사용자의 직접 개입 없이 웹 애플리케이션을 개발하는 데 초점을 맞추었음
- 인간 개발자가 전혀 코드에 손대지 않는 조건에서 Claude의 기능과 한계, 발전 가능성을 직접 검증
- 자동화 도구로서 Claude Opus 4.5가 어떤 식으로 애플리케이션 기능을 설계, 작성, 검증하는지 과정을 관찰함

### claw.ai의 완전한 클론 수준 기능 구현에 성공함

- 실험 결과, claw.ai와 거의 동일 수준의 완벽하게 동작하는 애플리케이션이 생성됨
- 기능의 완성도가 높아, 테스트 코드의 절반이 실제로 무슨 용도인지 의문이 들 정도로 완성도가 높다고 언급
- 사용자는 "완전히 기능이 구현됐다"고 느낄 만큼 실용적인 결과물 획득

### 자동 검증 및 반복 개선 과정에서 생성된 대화 히스토리 활용이 가능함

- Claude가 애플리케이션을 개발하면서 매 반복(loop)마다 검증과 피드백 과정을 거침
- 이 과정에서 생성된 모든 이전 대화 내용과 결과물을 한눈에 확인할 수 있도록 시스템이 구성됨
- 검증, 개선, 반복 프로세스가 명확히 기록되어 투명성 확보

### 애플리케이션의 주요 기능: 마크다운 포맷, HTML, 코드 실행 등이 구현됨

- 생성된 애플리케이션은 Markdown 포맷을 매우 잘 지원
- 다양한 HTML 페이지 생성이 가능하여, 웹 개발의 기술적 요구를 충족시킴
- 코드 작성뿐 아니라, 작성한 코드를 바로 실행할 수 있는 환경도 자동 구현됨
- 마크다운, HTML, 코드 실행 등 복합 기능을 Claude가 모두 소화함

### 사용자 설정 기능: 테마, 모델, 토큰 슬라이더 등 다양한 옵션 추가

- 사용자가 직접 바꿀 수 있는 테마 설정 옵션이 있음
- 기본적으로 사용할 AI 모델을 선택할 수 있도록 구성함
- 최대 토큰 수를 조절할 수 있는 슬라이더까지 설정 패널에 포함
- 복잡하고 다양한 사용자 지향 옵션 구현이 24시간 내에 이루어짐

### Human-in-the-loop 기능이 아직 남아 있고, 추가 발전도 기대됨

- 아직 애플리케이션에 human-in-the-loop(최종 의사결정자 혹은 관리자로서의 인간介입) 기능은 포함되지 않았으나, 향후 추가할 계획을 언급
- 현재 시점에서도 매우 인상적인 완성도를 보이나, 인간이 실시간으로 개입할 수 있는 구조도 중요하다고 암시

### 추가 개발 시간을 투입하면 사실상 완벽한 애플리케이션도 가능함을 언급함

- 만약 Claude가 하루 더 작업한다면, 더욱 완벽하고 모든 요구 기능을 갖춘 애플리케이션을 만들 수 있을 것이라고 언급
- 애플리케이션의 가능한 발전 범위를 직접 강조하며 Claude의 잠재력을 높이 평가

### Claude Opus 4.5와 반복 검증(harness) 조합의 혁신적 효과를 강조함

- Claude Opus 4.5 같은 고성능 AI와 자동 반복 검증(하네스) 체계가 결합될 때, 개발 효율성과 품질이 비약적으로 상승함을 실험 결과로 입증
- 자동화와 검증 루프를 통한 애플리케이션 개발·개선이 “crazy”하며 놀라운 결과를 가능케 한다고 표현

### 전 과정을 통해 CLI나 마우스로 아무것도 건드리지 않고도 높은 수준의 개발이 이루어짐

- 실제로 사용자는 코드를 “직접 손대지 않았다”고 강조
- 완전 자동화된 환경만으로 claw.ai 수준의 피쳐리치(feature-rich)한 결과물을 도출함
- AI 주도로 실질적이며 복합적인 애플리케이션 개발이 이미 현실임을 재확인

### 자동 개발 도구와 AI의 결합이 차세대 소프트웨어 개발방식임을 실감케 함

- Claude를 통한 자동 개발의 극적인 성과를 통해 차세대 웹/소프트웨어 개발 패러다임의 변화를 예감하게 됨
- AI+자동화 도구(harness)의 결합이 개발 효율 및 품질, 사용자 경험까지 크게 변화시키고 있음을 사례로 보여줌

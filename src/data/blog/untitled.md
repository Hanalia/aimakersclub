## *Claude Code로 아름다운 다이어그램을 만드는 전체 워크플로우* 핵심 요약

- Claude Code와 같은 코딩 에이전트는 시각적 설명 능력이 부족하여, 별도의 프롬프트와 기능 추가 없이 직관적인 시각 자료 생성이 어려움
- 제작자는 이러한 한계를 극복하기 위해 자신의 전체 워크플로우를 ‘스킬’로 패키징, 코딩 에이전트가 실용적이고 아름다운 다이어그램을 제작하도록 개선함
- Claude Code 혹은 다른 코딩 에이전트에 ‘Excal 다이어그램’을 제작·검증하는 시각적 논증(Arguing Visually) 능력을 부여하는 것이 핵심
- 이 워크플로우는 주 프라이머리 인스트럭션을 통해 Claude Code에게 Excal 다이어그램용 JSON 파일을 생성하고, 스스로 시각적으로 검증하는 기능을 추가함
- 다이어그램 렌더링 후 PNG로 스크린샷을 찍고, 이미지를 확인한 뒤 원하는 형태가 될 때까지 반복적으로 수정하는 ‘자가 검증’ 시스템이 특징
- Excal 다이어그램 스킬이 적용된 워크플로우를 활용하면, 코딩 에이전트가 직접 설치 및 세팅 절차를 자동화하며 사용자의 개입이 최소화됨
- 사용자 브랜딩 및 선호 색상 등 커스터마이징이 주요 절차에 포함, 다양한 스타일 및 목적의 다이어그램 생성이 가능
- ‘skill.mmd’ 내 프롬프트 설계의 핵심은 ‘단순 블록 나열’을 지양하고, 구조와 레이블을 통해 전체 개념을 시각적으로 설명하도록 유도하는 데 있음
- 해당 스킬은 PDF, 코드베이스 등 다양한 소스를 기반으로 다이어그램을 생성할 수 있어 활용 범위가 넓음
- 제작자는 이 스킬이 반복 작업을 자동화하여 주당 수 시간 이상을 절감한다며, 누구나 레포를 클론해 즉시 적용할 것을 권장함

## 세부 요약 - 주제별 정리

### 코딩 에이전트는 시각적 설명 능력이 한계에 부딪힘을 지적함

- Claude Code와 같은 코딩 에이전트들은 본질적으로 텍스트 위주 도구로, 시각화 작업에는 능숙하지 않음
- 구조나 개념을 다른 사람에게 설명할 때 시각적 자료가 필요하지만, 별도의 프롬팅과 기능 보강 없이 활용 시 이에 적합하지 않음
- 즉각적인 다이어그램 생성이나 설명은 물론, 시각적 커뮤니케이션의 품질이 낮음

### 맞춤형 워크플로우를 스킬로 패키징하여 시각화 성능을 극적으로 강화함

- 제작자는 ‘전체 워크플로우’를 Skill(스킬) 형태로 설계하여 코딩 에이전트에 적용함
- 해당 스킬을 적용하면 Claude Code나 기타 코딩 에이전트가 실용적이고 아름다운 다이어그램을 자동으로 만들어냄
- 실질적인 다이어그램 설계 능력이 코딩 에이전트에 내장됨

### 시각적으로 논증(Argue Visually)하는 능력이 코딩 에이전트의 핵심 스킬로 추가됨

- ‘Arguing Visually’(시각적 논증) 철학이 프라이머리 인스트럭션 세트에 주입됨
- Claude Code는 Excal 다이어그램용 JSON 파일을 제작하며, 논리적 구조와 시각 전개 방식을 이해하도록 최적화됨
- 단순 블록 나열이 아니라, 구조화와 레이블링을 통해 복잡한 개념을 명확히 전달함을 목표로 함

### Excal 다이어그램은 JSON 파일 생성 및 시각적 검증을 통해 완성도를 높임

- Claude Code는 Excal 다이어그램 제작 시 자동으로 JSON 파일을 생성함
- 다이어그램 완성 후 자체적으로 렌더링 및 PNG 파일로 스크린샷을 촬영함
- 제작된 이미지를 분석한 뒤, 불충분하거나 미려하지 않은 부분이 있으면 반복적으로 수정함
- 자가 검증(self validation) 절차가 포함된 작업 구조 덕분에 결과물의 품질이 대폭 향상됨

### 자가 검증(Self Validation) 시스템이 반복 작업과 시간 소모를 크게 줄여줌

- 기존에는 시각적 결과물의 오류 확인 및 수정이 수작업으로 이루어져 번거로웠음
- 스킬 적용 후, 코딩 에이전트가 직접 작업물의 완성도를 평가하고 개선함
- 결과적으로 수작업 교정 시간과 반복 작업 시간이 절감되어 생산성이 상승함

### Excalra 다이어그램 스킬의 설치 및 세팅 과정이 자동화됨

- 사용자는 코딩 에이전트에게 ‘Excalra 다이어그램 스킬’을 설치 및 세팅하도록 요청함으로써 초기 환경 구성이 가능함
- 스킬에 내장된 프롬프트와 코드가 Readme 파일을 읽고, 매뉴얼 설치 과정 전체를 자동으로 수행함
- 사용자는 세팅 완료 후 즉시 임의의 다이어그램 생성을 요청할 수 있음

### 사용자 브랜딩 및 색상 커스터마이징 등이 손쉽게 적용됨

- 다이어그램의 브랜드 컬러나 개인 취향의 색상을 지정하여 일관된 시각 자료를 구현할 수 있음
- 사용 목적에 따라 다양한 스타일로 다이어그램을 만들 수 있어 맞춤형 활용도가 높음

### skill.mmd 내 프롬프트 설계가 시각적 설명을 명확히 유도함

- 프롬프트의 주된 원칙은 ‘단순 블럭 생성’ 대신 ‘구조 및 레이블’을 활용해 개념의 흐름과 연결을 명확하게 시각화하는 것임
- Excal 다이어그램 스킬 미적용 시, 생성 결과가 ‘블로키(덩어리식 표현)’로만 나오기 쉬움
- 스킬 적용 시 개념 연결, 정보 흐름, 핵심 메시지가 한눈에 드러나도록 다이어그램을 생성함

### 다양한 입력(예: PDF, 코드베이스)에서 출발해 무한한 활용이 가능함

- Excal 다이어그램 스킬은 PDF 파일, 코드베이스 등 여러 데이터 소스를 입력값으로 사용할 수 있음
- 단순 설명 외에 복잡한 프로젝트 구조나 동작 흐름도, 알고리즘 도식화 등도 빠르게 구현할 수 있음
- 사용자의 필요에 따라 실무적부터 창의적 용도까지 다양하게 확장 가능함

### 레포 클론 및 즉시 적용을 권장하며 업무 효율이 현저히 향상됨

- 제작자는 본인이 직접 ‘수 시간 단위’의 업무 절감 효과를 경험했다고 밝힘
- 특정 저장소(repo)를 클론하고 스킬을 가져온 뒤 곧바로 활용할 수 있도록 안내함
- 설치 및 적용 과정이 간단하며 처음 시도하는 사용자도 쉽게 접근 가능함

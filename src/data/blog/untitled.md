## *클로드에게 24시간 동안 쉬지 않고 코딩을 시켰더니 생긴 일* 핵심 요약

- 영상에서는 AI 모델 Claude Opus 4.5를 활용해 기능이 풍부한 claw.ai 클론 애플리케이션을 개발하는 과정을 소개함
- 제작자는 테스트의 절반이 어떤 용도인지 모를 정도로 이미 완전한 수준의 클론을 구현했다고 밝힘
- AI가 각 반복 루프마다 검증한 대화 내용을 모두 열람 가능함
- 마크다운 형식 지원 및 다양한 HTML 페이지 생성, 코드 작성 및 실행 기능까지 포함됨
- 테마, 기본 사용 모델, 최대 토큰 설정 슬라이더 등 사용자의 환경설정 기능이 추가됨
- 아직 Human in the loop(사람의 검토, 개입 절차)는 적용하지 않았으나, 이미 충분히 놀라운 결과임을 강조
- 제작자는 직접 코드를 작성하지 않고도 방대한 기능을 구축한 점에 감탄함
- 하루만 더 두고 모든 기능 검증을 완료한다면 완벽에 가까운 애플리케이션이 될 수 있다고 자신함
- Claude Opus 4.5와 검증/반복이 자유로운 ‘하네스(harness)’ 조합으로 놀라운 생산성과 품질 확보가 가능함을 설명함
- 영상의 전체 흐름은 AI의 자율적 반복·검증 프로세스와 제작자의 최소 개입으로 완성도 높은 소프트웨어를 만드는 데 중점을 둠

## 세부 요약 - 주제별 정리

### 클로드 Opus 4.5가 claw.ai 클론을 사실상 완벽하게 구현함

- 영상은 AI 모델 Claude Opus 4.5를 사용해 claw.ai(클로.ai)와 유사한 애플리케이션을 개발하는 과정을 다룸
- 제작자는 "이미 완전히 기능하는 claw.ai 클론을 만든 느낌"이라고 밝힘
- 테스트의 절반이 무엇을 위한 것인지 알 수 없을 만큼 높은 완성도를 자랑함
- 기능의 충실도와 범위가 기대를 뛰어넘는 수준임

### AI의 반복적 검증 덕분에 대화 및 개발 진행 상황을 투명하게 확인 가능함

- AI가 각 개발 단계별로 반복 루프를 돌며 기능을 검증함
- 그 과정에서 생성된 모든 과거 대화와 검증 내역을 사용자가 직접 열람할 수 있음
- 투명하고 신뢰성 높은 개발 진행 상황 파악이 가능함

### 마크다운 포맷팅, HTML 페이지 생성, 코드 작성/실행 등 강력한 기능이 구현됨

- 개발된 애플리케이션은 읽기 쉽고 보기 좋은 마크다운(Markdown) 형식 지원이 뛰어남
- 다양한 HTML 페이지를 생성할 수 있음
- 애플리케이션 내에서 코드 작성과 실행이 모두 가능하도록 구현됨
- 기존 AI 챗봇 영역을 넘어선 확장성과 응용력을 보여줌

### 사용자 설정 기능(테마, 모델 선택, 토큰 슬라이더)까지 탑재됨

- 사용자는 테마(색상 스키마 등) 변경 기능을 활용할 수 있음
- 기본 사용할 AI 모델을 직접 선택 가능하도록 설계됨
- 최대 토큰 수를 조절하는 슬라이더 UI가 구현되어 있음
- 사용자 개개인의 취향과 실정에 맞추어 서비스를 세밀하게 설정할 수 있음

### Human in the Loop(사람의 검토 단계)는 아직 적용되지 않음

- 제작자는 "Human in the loop" 즉, 사람이 검수하거나 개입하는 절차를 추가할 예정임을 밝힘
- 현재 단계에서는 AI가 모든 설계 및 개발 과정을 주도하였기에, 사람의 관리나 중재가 없었음
- 그럼에도 불구하고 생성물의 완성도가 매우 높다는 점을 강조함

### 코드에 직접 손대지 않고 AI 자율 개발만으로 다채로운 기능을 구축함

- 영상 내내 제작자는 자신이 "직접 건드린 것이 없다"는 점에 놀람을 표함
- 실제로 본인은 아무 작업도 하지 않은 상태에서 이만큼의 결과물이 나왔음을 반복적으로 강조
- AI 모델이 명령에 따라 스스로 기획·개발·검증을 순환하며 전체 애플리케이션을 완성해감

### 하루만 더 개발과 검증 시간을 주면 완벽에 가까운 앱 제작이 가능할 것으로 전망함

- 제작자는 "만약 AI에 하루만 더 맡겼다면 모든 기능 검증까지 끝낼 수 있을지도 모른다"며 확신을 표함
- 현재 상태에서도 이미 대부분 요구 사항을 충족했으나, 추가 반복과 테스트로 완벽한 결과 가능함을 시사
- AI의 자율적 개발 역량과 속도에 대한 신뢰를 드러냄

### Claude Opus 4.5와 반복·검증 ‘하네스’ 결합이 압도적 생산성을 제공함

- 실제 코드 실행, 자동 검증, 다회 반복이 가능한 개발 환경(하네스)을 구축해 Claude를 운용함
- AI가 애플리케이션을 개발하면서 스스로 검증, 제품 개선, 기능 추가까지 시행함
- 이러한 조합 덕분에 전통적인 개발 프로세스 대비 혁신적 생산성과 품질 향상이 실현됨
- AI + 하네스라는 체계적 조합을 통해 반복 검증, 자동화된 에러 수정, 효과적인 피드백 루프가 가능했음

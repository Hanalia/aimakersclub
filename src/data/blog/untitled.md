## *Claude에게 24시간 동안 비 nonstop으로 코딩을 시킨 결과* 핵심 요약

- 실험자는 Claude Opus 4.5에게 24시간 동안 멈추지 않고 코딩을 시키는 실험을 실시함
- Claude가 비슷한 서비스인 claw.ai의 완전한 복제판을 거의 완성함
- 생성된 여러 대화 기록을 통해 각 단계별 검증 과정까지 추적이 가능함
- 마크다운 형식의 우수한 포맷팅 지원 기능이 포함됨
- 다양한 HTML 페이지 생성, 코드 작성 및 실행이 자동으로 가능해짐
- 테마와 기본 모델을 설정하고, 최대 토큰 양을 조절하는 슬라이더 등 풍부한 설정 기능 탑재
- 인간의 개입이 필요한 부분이 남아 있지만, 대부분의 작업을 AI가 스스로 처리함
- Feature의 추가와 테스트를 반복할 수 있는 구조적 ‘루프’가 구현됨
- 실험자는 하루만 더 맡긴다면 “완벽한 앱이 나올 것 같다”고 평가함
- Claude Opus 4.5와 같은 최신 AI와 자동 검증 도구(harness)가 결합될 때, 애플리케이션 생성과 반복 검증이 혁신적으로 빨라진다는 점이 강조됨

## 세부 요약 - 주제별 정리

### Claude는 24시간 만에 claw.ai와 유사한 완성형 애플리케이션을 자동 구축함

- Claude Opus 4.5에게 24시간 동안 논스톱으로 코딩과 반복 검증을 시키는 실험이 진행됨
- 실험 후 “나는 마지막 테스트들 중 절반이 무엇을 위한 것인지조차 모르겠다”며 실제로 이미 완전한 claw.ai 복제판이 만들어졌음을 언급
- 기능적 완성도가 상당히 높아, ‘기능 풍부하다(featurerich)’고 강조

### 생성된 대화 기록과 검증 루프가 작업 과정을 투명하게 기록함

- Claude가 애플리케이션 각 기능을 만들고 검증하는 모든 과정을 대화 형태로 저장
- 이전 대화 내역을 통해 어떤 검증 루프를 거쳤는지 추적할 수 있음
- 개발 자동화 과정의 ‘투명성’과 ‘재현성’까지 확보

### 자동 생성된 마크다운 형식 포맷이 높은 가독성과 편의성을 제공함

- 애플리케이션 내에서 마크다운 문서 포맷팅이 깔끔하게 적용됨
- 주요 정보와 코드, 설명 등이 마크다운 스타일로 손쉽게 확인 가능

### HTML 페이지의 자동 생성 및 코드 작성, 실행까지 Claude가 담당함

- Claude가 다양한 HTML 페이지를 직접 생성해냄
- 코드 작성뿐 아니라, 코드 실행까지 자동으로 처리함
- 실질적으로 인간이 손대지 않아도 주요 기능 구현이 가능해짐

### 세부 설정 기능과 사용자 맞춤형 환경이 완비됨

- 애플리케이션 내에 다양한 설정(Settings) 기능이 구현됨
- 테마(Theme) 선택 및 기본 사용 모델(Default model) 지정이 가능함
- 최대 토큰(Max tokens) 설정을 조절할 수 있는 슬라이더 인터페이스가 탑재됨

### 인간의 개입이 필요한 부분이 일부 남아 있지만, 자동화 수준이 인상적임

- 아직 ‘human in the loop(인간의 확인/개입)’ 단계가 추가될 예정
- 그럼에도 불구하고, 대부분의 코딩과 검증 작업을 AI 단독으로 수행함

### 자동화된 반복 루프와 검증 기능이 앱 개발 과정을 혁신적으로 단축함

- ‘루프’를 통해 각기 다른 기능과 테스트를 생성, 검증, 반복함
- 테스트 케이스가 꼼꼼하게 생성되어 애플리케이션의 버그 가능성을 줄임

### Claude Opus 4.5와 자동화 테스트 도구(harness)의 결합이 생산성을 극대화함

- 최신 언어모델(Claude Opus 4.5)의 강점을 극대화할 수 있는 자동 검증 환경(harness)이 구축됨
- Harness는 AI가 자체적으로 개발-테스트-수정의 사이클을 반복하게 해줌
- 이 결합 덕분에 기존의 개발-테스트-반복보다 훨씬 빠른 애플리케이션 생성이 가능해짐

### 단 하루의 시간으로 기능 완성도가 높은 앱 개발이 현실적으로 가능함

- 실험자는 “하루만 더 계속되면 정말 완벽한 앱이 될 것”이라고 평가
- 최신 AI와 자동화 도구의 결합 덕분에 개발자의 직접적 개입 없이도 복잡한 프로젝트가 완성 가능함

### Claude를 활용한 실험이 보여주는 미래 개발환경의 가능성이 크다고 시사함

- “우리가 Claude Opus 4.5로 어떤 것까지 만들 수 있는지 정말 놀랍다”고 강조
- 인공지능이 개발 자동화의 핵심 도구로 자리잡을 수 있음을 입증하는 사례로 제시됨

## *Anthropic, Claude Sonnet 5를 출시하며 Opus급 코딩 성능을 저렴하게 제공* 핵심 요약

- Anthropic이 Claude Sonnet 5 언어모델을 전격 출시했고, 이번 버전의 핵심은 속도가 아닌 “중간 가격대 모델로 Opus 4.8급 코딩 점수”임
- Claude Sonnet 5는 Sweetbench 검증 코딩 테스트에서 85.2점 획득, 최상위 모델인 Opus 4.8에 거의 근접
- 가격은 Opus 대비 훨씬 저렴한데, 특히 “가격이 핵심 반전”이라고 소개됨
- Sonnet 5는 지금까지 나온 Sonnet 중에서 에이전트적 기능(자동화·멀티스텝 작업 처리)이 가장 강력함
- 리포지터리(코드 저장소)를 지정하면, 계획 실행, 터미널 작업, 다단계 프로세스까지 이전 Sonnet보다 확실히 마무리 가능
- 기존 Sonnet 4.6 대비 주요 벤치마크 점수가 대폭 상승: bench verified 85점(4.6은 그 이하), terminal bench 80점(4.6은 67점), browse comp 86점(4.6은 76점)
- Frontier Code 테스트에서 Sonnet 5는 38점으로, 구버전 Sonnet(15점)에 비해 두 배 이상 성능 향상
- 모델 업그레이드는 매우 간단해서, 모델 ID만 ‘sonnet-5’로 바꾸면 마이그레이션이 끝남(코드 재작성 불필요)
- Opus 4.8이 여전히 소폭 앞서긴 하나, Sonnet 5는 거의 근접한 수준에 가격 경쟁력까지 가짐
- Sonnet 5 성능은 무료, Pro, Max, Team, Enterprise, Claude Code, 플랫폼 등 다양한 상품군에서 동일하게 제공
- 8월 31일까지 100만 토큰당 2달러라는 파격적인 요금으로 Opus급 코드 생성이 가능
- 영상 마지막에서, “Sonnet 5가 진정한 세대 교체(Generational Leap)인가, 아니면 단지 저렴한 Opus인가?”라는 질문을 던지며 마무리함

## 세부 요약 - 주제별 정리

### Sonnet 5 출시는 기존과 달리 속도가 아닌 코딩력 향상에 중점이 두어짐

- Anthropic이 최근 Claude Sonnet 5를 출시
- 영상 초반부터 “이번 Sonnet 5의 헤드라인은 속도가 아니다”라고 강조
- 주요 혁신은 ‘중간 가격대 모델이 Opus 상위 모델에 거의 필적하는 코딩 능력’을 보인다는 점에 있음
- 코딩 관련 벤치마크 성능에서 극적인 도약을 기록

### Sweetbench 검증 코딩 점수에서 Opus 4.8에 거의 맞먹는 성과를 거둠

- Sonnet 5가 Sweetbench verified coding 점수 85.2를 획득
- 최상위 모델인 Claude Opus 4.8(섬세한 설명 없이도 인공지능 분야에서 첨단임)에 거의 근접하는 수치
- 이 수치를 중간 가격대의 Sonnet 모델이 기록한 점이 강한 인상으로 언급됨

### Sonnet 5의 가격 경쟁력은 이번 모델의 핵심 반전 요소로 부각됨

- 영상에서 “사실 가격이 진짜 반전”이라고 언급
- Opus 대비 낮은 가격에 거의 같은 수준의 성능
- 특히 8월 31일까지 Sonnet 5는 100만 토큰당 2달러라는 낮은 요금으로 제공됨

### Sonnet 5는 에이전트(Agentic)로서 자동화 작업 및 멀티스텝 수행력이 크게 진보함

- “가장 agentic(자동화·에이전트 지향) Sonnet”이라고 소개
- 리포지터리를 지정해주면, 모델이 직접 계획 세우고, 터미널 명령 실행 및 멀티스텝 태스크를 자동으로 처리
- 이전 Sonnet(4.6 등)은 절반만 처리하다가 중단하는 멀티스텝 작업도 Sonnet 5는 끝까지 수행

### 주요 벤치마크 수치에서 Sonnet 4.6 대비 Sonnet 5가 큰 폭으로 개선됨

- bench verified 점수: Sonnet 4.6 대비 Sonnet 5가 85점으로 대폭 향상
- terminal bench: Sonnet 4.6의 67점에서 Sonnet 5는 80점 달성
- browse comp: Sonnet 4.6의 76점보다 Sonnet 5가 86점으로 상승
- 세 수치 모두 명확한 업그레이드임

### Frontier Code 테스트에서 Sonnet 5는 구버전 대비 두 배가 넘는 코딩 능력을 입증함

- Frontier Code 벤치마크에서 Sonnet 5가 38점 기록 (Sonnet 4.6은 15점)
- “Sonnet 사상 가장 큰 단일 성능 도약”으로서 주목
- 단순 점수 차가 아닌, 실제 코드 생성·문제 해결 능력에서 대폭 상승

### Sonnet 5 업그레이드는 매우 단순하며 기존 코드의 호환성이 뛰어남

- 사용자는 기존 Sonnet 4.6 모델 ID를 ‘sonnet-5’로 한 줄만 수정하면 즉시 업그레이드 적용
- 코드 자체를 다시 작성할 필요가 없음
- 간단한 마이그레이션 과정이 장점으로 부각

### Opus 4.8이 아직은 소폭 앞서지만, 거의 근접한 Sonnet 5 성능은 폭넓게 제공됨

- Opus 4.8이 여전히 특정 벤치마크(아마도 frontier code 등)에서 근소하게 우위
- 그러나 그 차이 자체에 집중하기보다는, Opus에 필적하는 소넷 코딩을 폭넓게 제공한다는 점이 중요
- Sonnet 5의 코딩 성능을 Claude 무료, Pro, Max, Team, Enterprise, Claude Code, 플랫폼 등 전 상품군에서 동일하게 이용 가능

### ‘Sonnet 머니’로 Opus급 코딩을 저렴하게 접할 수 있는 파격적 요금 정책이 시행됨

- 2024년 8월 31일까지 Sonnet 5는 100만 토큰당 2달러라는 요금으로 제공
- Opus에 가까운 코딩력이 ‘Sonnet 머니’(비용)로 가능해짐
- 중소 개발팀부터 기업까지 모두에게 가격 측면에서 이점을 안겨줌

### Sonnet 5는 세대교체(Geneational Leap)인지, 저렴한 Opus인지에 대한 논의로 영상이 마무리됨

- 영상 클로징에서, “Sonnet 5가 진정 세대 교체 모델이냐, 아니면 저렴한 Opus냐?”라는 질문 제시
- 사용자의 실제 경험과 워크샵(‘Dynamus’의 주간 핸즈온 교육) 안내로 마무리
- Sonnet 5의 미래적 의미와 실질적 활용점을 동시에 암시하는 엔딩 구성

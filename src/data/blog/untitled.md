## *AI 전쟁이 다시 시작됐다! Sonnet 4.5, DeepSeek V3.2, GLM-4.6 출시* 핵심 요약

- 여름 동안 대단히 인상적인 새로운 대형 언어모델(LLM)은 등장하지 않았으나, 최근 다양한 신모델이 동시에 출시됨
- Anthropic에서 어제 공개한 Claude Sonnet 4.5가 출시되며 현존 ‘AI 코딩 최강자’로 떠오름
- Sonnet 4.5는 Opus 4.1 및 GPT-5 코드계 모델보다 약간 더 우수하며, 속도 역시 매우 빠름(직접 테스트 결과 언급)
- DeepSeek에서는 새 버전인 DeepSeek V3.2를 출시했으며, 속도가 더 빨라지고 경량화되었지만 강력함을 유지함
- DeepSeek V3.2는 Sonnet 4.5에 비해 가격이 수십 배 저렴함
- 중국에서도 GLM 4.6이라는 새로운 LLM이 발표됨
- Anthropic의 Sonnet 4.5 발표 직후 바로 GLM 4.6이 공개되며, 중국의 빠른 대응이 돋보임
- GLM 4.6은 다양한 벤치마크에서 Sonnet 4.5와 비교되고 있으며 대부분의 성능에서 승리함
- 벤치마크 결과가 전부를 말해주진 않지만, 현재 LLM 분야에서 매우 흥미롭고 격동적인 시기임을 알림

## 세부 요약 - 주제별 정리

### 여름 동안 눈에 띄는 새로운 LLM 출시가 거의 없었으나 최근 동시에 다수의 신모델이 등장함

- 여름철에 기대만큼 혁신적이거나 주목받는 새로운 대형 언어모델(LLM)의 등장이 드물었음
- 그러나 최근에는 "고전적인 패턴처럼" 여러 모델이 한꺼번에 쏟아지는 양상을 보임
- 이 변화는 시장의 경쟁이 다시 가열되고 있음을 시사함

### Anthropic의 Claude Sonnet 4.5가 출시되며 AI 코딩 분야에서 새로운 최강자로 자리매김함

- Claude Sonnet 4.5는  어제(영상 촬영 시점 기준) Anthropic에 의해 공개됨
- 발표와 동시에 AI 코딩 분야의 새로운 강자로 소개됨
- Opus 4.1, GPT-5 코덱스 모델과 직접 비교에서 약간 더 우수하다고 평가됨
- 속도 측면에서는 본인 테스트 결과 Sonnet 4.5가 매우 빠름을 강조
- 성능 우위와 스피드를 통해 개발자·코더 중심의 활용도가 높아질 것으로 전망

### DeepSeek V3.2는 이전보다 혁신적으로 빨라졌으며 가격 경쟁력이 매우 높음

- DeepSeek V3.2가 새롭게 공개됨
- 새 버전은 속도가 더 빨라지고, 모델 자체가 경량화되었으나 기존과 같은 강한 성능을 유지함
- DeepSeek V3.2는 Claude Sonnet 4.5와 같은 최첨단 모델과 비교할 때 성능은 유사하지만 훨씬 저렴함
- 구체적으로 Sonnet 4.5 대비 ‘수십 배’ 저렴한 비용 구조를 갖춤
- 개발 및 테스트 예산에 민감한 이용자에게 큰 매력 요소가 됨

### 중국의 GLM 4.6이 Sonnet 4.5 출시 직후 빠르게 공개되어 글로벌 경쟁 구도가 증폭됨

- 중국에서 새로운 LLM 모델인 GLM 4.6이 출사표를 던짐
- Anthropic의 Sonnet 4.5가 발표된 바로 다음 날 GLM 4.6이 출시됨(즉각적 대응)
- 중국의 AI 생태계도 글로벌 LLM 경쟁에서 빠르게 움직임을 보임
- "물론 Anthropic의 발표 직후"라는 표현에서 중국 업체들의 신속함과 전략성이 엿보임

### GLM 4.6은 다양한 벤치마크에서 Sonnet 4.5보다 우위를 점하고 있음

- GLM 4.6은 공식 벤치마크 결과에서 Claude Sonnet 4.5와 항목별 직접 비교됨
- 다수의 주요 벤치마크에서 GLM 4.6이 우수한 결과를 기록함(“winning most of the benchmarks”)
- 이러한 성능 경쟁 결과는 중국 LLM의 기술력이 글로벌 최상위권임을 다시 한번 입증함

### 벤치마크만으로 모델의 모든 가치를 평가할 수는 없음

- 영상 내에서 “벤치마크 결과가 모든 것을 말해주지는 않는다”는 단서가 붙음
- LLM의 실사용 경험, 활용도 등의 다양한 요소가 모델 평가에 영향을 미침을 시사함

### 최근 LLM 개발 경쟁은 매우 흥미진진하고 격동적인 시대로 접어듦

- 여러 LLM 등장과 각국(특히 미국, 중국)의 신속한 대응으로 시장이 다시 뜨거워짐
- 기술 성능·속도·비용·글로벌 벤치마크 등에서 치열한 경쟁이 이루어짐
- 현재는 LLM 전문 AI 분야에서 “재미있고 흥미로운 시기”로 평가됨

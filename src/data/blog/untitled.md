## *Claude에게 24시간 동안 논스톱으로 코딩을 시킨 결과* 핵심 요약

- 영상 제목: *Claude에게 24시간 동안 논스톱으로 코딩을 시킨 결과*
- 영상 제작자는 Claude Opus 4.5를 이용해 실제로 손을 대지 않고 대화형 웹앱(실제 claw.ai 기능을 복제한 것)을 24시간 동안 자동 생성함
- 총 기능 면에서 이미 claw.ai와 유사한 거의 완전한 복제본이 완성됐다고 평가
- 각 개발 단계 및 코드 검증 작업이 반복적으로 이루어지고, 과거 모든 대화 내역과 작업 이력을 확인할 수 있음
- 생성된 애플리케이션은 마크다운 포맷팅이 잘 구현되어 있음
- HTML 페이지 생성, 코드 작성 및 실행 등 다양한 기능이 자동으로 탑재됨
- 테마 및 디폴트 모델 변경, 최대 토큰 수 조절 등이 가능한 설정 메뉴도 구현됨
- 인간의 개입(Human in the loop) 기능 추가만이 미완성 상태이며, 그 외에는 매우 높은 완성도를 자랑함
- 인공지능이 스스로 애플리케이션을 검증·반복 개선하는 자동화 하네스(harness)가 큰 역할을 함
- AI의 반복적 자기 검증 및 기능 추가를 하루 더 지속하면 '완벽한 애플리케이션'에 근접할 것으로 기대

## 세부 요약 - 주제별 정리

### Claude Opus 4.5의 코드 자동 생성 능력은 기대 이상으로 뛰어나게 확인됨

- 영상 제작자는 Claude Opus 4.5를 활용해 어떠한 직접적 코딩 작업 없이 하루 동안 실제 웹앱을 자동 생성함
- 애플리케이션의 완성도가 높아, 본래 목표였던 claw.ai의 복제본을 거의 완벽하게 구현했다고 언급함
- 자동화 작업에서 제작자는 실제로 “손 하나 까딱하지 않았다”고 강조
- Claude는 기능 구현 및 문제 검증, 코드 반복 개선 루프 등을 자체적으로 수행함

### 대화 내역 및 작업 이력 관리가 투명하게 제공됨

- Claude가 각 루프(반복 과정)에서 어떤 대화를 나누고, 어떤 검증 절차를 거쳤는지 모든 이력이 기록됨
- 사용자는 과거의 모든 대화 내역을 언제든지 확인 가능하여 디버깅이나 기능 추적이 용이함
- 이 같은 투명성은 개발 진행 상황의 신뢰성을 높임

### 고도화된 마크다운 포맷팅과 HTML 페이지 생산력이 자동 구현됨

- Claude가 제작한 애플리케이션은 마크다운 기반의 가독성 높은 문서 포맷팅을 포함함
- 별도의 수작업 없이 다양한 HTML 페이지가 자동으로 생성됨
- 이 과정에서 Claude는 코드 작성, HTML 변환, 실행까지 일괄적으로 처리함

### 실제 코드 작성 및 실행 기능이 내장되어 있음

- Claude는 단순히 인터페이스만 만드는 것이 아니라 실질적인 코드 작성과 실행까지 지원함
- 코드의 실행 여부와 결과까지 자동으로 확인할 수 있으므로, 개발자가 수동으로 코드를 테스트할 필요가 없음
- 이로 인해 개발 생산성이 극적으로 개선됨

### 설정(Menu) 기능이 다양한 사용자 맞춤 옵션을 제공함

- 애플리케이션 내에는 테마(예: 라이트/다크 모드 등) 변경 기능이 탑재됨
- 디폴트 AI 모델을 사용자가 선택할 수 있는 옵션이 있음
- 최대 토큰 수(max tokens) 조절을 위한 슬라이더 기능도 제공됨
- 이러한 설정 기능들은 완성도 높은 SaaS형 소프트웨어의 모습을 보여줌

### Human-in-the-loop 기능만이 미완성인 채로 남아 있음

- 현재 생성된 애플리케이션은 거의 모든 주요 기능을 구현했으나, 인간 중재(human-in-the-loop) 기능은 추가 예정임
- 인간의 개입을 통해 더욱 신뢰도 높은 애플리케이션을 만들려는 계획이 언급됨
- 그 외의 기능들은 매우 만족스럽게 동작하여 실사용이 가능한 수준임

### 자동화 하네스(harness)의 도입으로 반복 검증·개선이 이루어짐

- Claude는 자체 검증 루프와 자동화 코드를 바탕으로 지속적인 검증(validation)과 반복(iteration)을 실행함
- AI가 맡은 소프트웨어가 여러 번의 테스트 루프를 통해 발전하며, 각 반복마다 품질이 일정 수준 이상으로 향상됨
- 이러한 자동화 하네스 구조는 기존 소프트웨어 개발보다 개발 속도와 완성도를 비약적으로 높임

### 추가 개발 시간이 주어진다면 '완벽'에 더욱 가까워질 것으로 기대함

- 영상 제작자는 하루 더 돌리면 애플리케이션이 거의 완벽해질 것이라고 언급함: “maybe this would be an absolutely perfect application”
- Claude Opus 4.5와 자동화 검증 시스템의 조합으로, 제품 전체가 거의 손을 대지 않고도 완성 가능
- AI의 자기 개선 능력이 개발 패러다임의 변화를 시사함

### AI 모델의 코드 실행 및 반복적 자기 검증 과정이 혁신적임을 강조

- 단순한 코드 생성에서 끝나는 것이 아니라, 실행 및 검증을 반복함으로써 문제점을 자체적으로 보완
- 검증 과정에서 생성된 모든 대화를 기록하고, 각 검증 루프가 해당 기능 구현의 신뢰도를 높임
- AI에게 맞긴 반복 작업과 분석이 실제 코딩 환경에서도 바로 활용될 수 있음을 시연함

### Claude 4.5와 하네스 시스템의 결합이 개발 환경을 혁신적으로 변화시킴

- 영상은 Claude 4.5와 이를 지원하는 자동검증 하네스가 개발자 없이도 복잡한 앱을 빠르고 완성도 높게 구현할 수 있음을 보여줌
- 이 방식으로 사람의 작업 부담 없이 주요 기능이 배치·개선됨
- 인공지능을 통한 코드 자동화, 자기 검증 구조의 궁극적인 개발 생산성 도약 가능성을 실험적으로 입증함

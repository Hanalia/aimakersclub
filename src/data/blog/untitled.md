## *Claude Code로 아름답고 실용적인 다이어그램을 만드는 전체 워크플로우* 핵심 요약

- 영상은 Claude Code 같은 코딩 에이전트가 시각적 설명에는 취약하다는 점에서 출발함
- 이를 개선하기 위해 제작자는 자신만의 다이어그램 제작 워크플로우를 하나의 ‘스킬’로 패키징함
- 이 스킬을 통해 Claude Code 같은 코딩 에이전트가 Excal 다이어그램을 시각적으로 논리적으로 만들도록 지시할 수 있음
- 주된 방식은 코딩 에이전트에게 시각적으로 주장(설명)하는 방법을 익히도록 프라이머리 인스트럭션 세트를 제공하는 것임
- Excal 다이어그램 제작 과정에서 JSON 파일 생성 뿐 아니라, 결과물을 실제로 렌더링 후 PNG로 스크린샷을 찍고 부족한 점을 반복적으로 개선함
- 이런 ‘셀프 유효성 검증’이 실제로 엄청난 효과를 내며, 시각작업이 약한 코딩 에이전트의 단점을 보완함
- Excalra diagram skill 세팅은 코딩 에이전트에게 직접 요청하면 readme를 읽고 스킬 설치까지 자동으로 수행할 수 있음
- 사용자는 이 스킬로 자신만의 브랜드나 컬러 스킴 등 원하는 스타일의 다이어그램을 쉽게 제작할 수 있음
- skill.mmd 프롬프트 구조의 핵심은 ‘블록 나열’ 대신 ‘시각적 주장’이므로, 구조와 라벨링에 신경쓴 다이어그램을 생성함
- 이 워크플로우는 PDF, 코드베이스 등 다양한 맥락에서 수시간의 작업을 절약해주며, 누구나 레포를 클론하고 손쉽게 활용할 수 있음

## 세부 요약 - 주제별 정리

### 코딩 에이전트는 시각적 설명에 본질적으로 약하며 추가 기능이 필요함

- Claude Code와 같은 AI 기반 코딩 에이전트들은 원래 코드 생성과 언어 기반 설명에는 뛰어나지만, 시각적으로 개념을 전달할 때는 한계가 있음
- 사용자가 무언가를 시각적으로 설명받거나 직접 설명하고 싶을 경우, 단순한 프롬프트만으로는 만족스러운 결과에 도달하기 어려움
- 기본적인 설정만으로는 다이어그램 제작 요청에 응답하지만, 세부적이거나 직관적인 시각적 해석이 부족하게 나타남

### 다이어그램 제작 워크플로우를 하나의 스킬로 패키징하여 자동화함

- 영상 제작자는 시각적으로도 뛰어난 다이어그램을 효율적으로 만들 수 있도록, 자신만의 워크플로우를 ‘skill’ 형태로 묶어서 자동화함
- 이 스킬은 Claude Code뿐만 아니라, 다양한 코딩 에이전트(코드 생성 AI)와 연동하여 쓸 수 있도록 설계됨
- 사용자는 이 스킬만 임포트하면 복잡한 설정 없이 다이어그램 생성이 가능

### 프라이머리 인스트럭션 세트로 시각적 주장(설명) 방식을 학습시킴

- 스킬에는 Claude Code에게 ‘시각적으로 설명하는 방법’을 단계별로 가르치는 인스트럭션 세트가 핵심적으로 포함됨
- 단순한 블록 배치가 아니라, 개념/관계 중심의 다이어그램 구조, 직접적인 라벨링 등을 통해 논리적 구조가 한눈에 들어오도록 설계
- 기존의 Excal 다이어그램 생성은 각 요소가 독립적으로 블록화되어 직관성이 떨어졌으나, 인스트럭션 세트로 개념적 쟁점을 시각적으로 드러냄
- 구조 및 라벨링 방식이 ‘시각적 논증’에 중점을 두도록 제작

### Excal 다이어그램 파일(JSON) 자동 생성 및 자체 시각 검증 반복

- 코딩 에이전트는 인스트럭션에 따라 Excal 다이어그램용 JSON 파일을 자동으로 생성
- 중요한 차별점은 단순히 생성만 하는 것이 아니라, 렌더링(Excal로 변환) 후 PNG 스크린샷을 찍어 실제 결과를 확인함
- PNG 결과물을 직접 시각적으로 검토 후, 부족하거나 어색한 부분을 자동으로 반복 개선(Iterate)하는 방식 적용
- 시각적 셀프 유효성 검증(XeX Excal)이 결과의 퀄리티를 비약적으로 올려줌
- 영상 제작자는 이 과정을 “insane results”라 표현할 만큼 생산성과 품질에서 극적인 차이를 체감함

### Excalra diagram skill 세팅이 코딩 에이전트의 자동화 능력을 극대화함

- 사용자는 “Excalra diagram skill” 설치 및 세팅을 복잡하게 할 필요 없이, 코딩 에이전트 자체에 요청만 하면 됨
- 코딩 에이전트는 레포지토리의 README 파일을 자동으로 탐독하고, 수동 설치가 필요한 부분까지 모두 알아서 진행
- 설정이 끝나면 사용자는 단순한 프롬프트를 통해 원하는 다이어그램을 바로 요청 가능

### 브랜드 및 컬러 맞춤형 다이어그램 생성이 편리해짐

- 사용자의 브랜드 색상이나 선호하는 컬러 스킴 등, 개별 맞춤형 다이어그램 제작이 손쉽게 구현됨
- 영상에서는 실제 간단한 데모에서 이런 맞춤형 옵션 적용이 인상적으로 드러남

### skill.mmd의 기본 철학은 ‘시각적 논증’이며, 단순 블록생성과 분명한 차이가 있음

- skill.mmd(스킬 프롬프트 구조)는 “단순한 상자 나열이 아닌, 구조적이고 메시지가 분명한 시각 자료”를 만드는 데 초점
- 다이어그램의 노드, 흐름, 라벨이 왜 그처럼 배치되었는지 설명까지 포함해 실제 사용 목적에 맞춤
- 기존 AI 다이어그램 자동생성과 달리, 논리적 흐름과 개념, 관계가 한눈에 들어오게 지도함

### 다양한 문서와 코드베이스에 적용하여 수많은 시간을 절약함

- PDF 문서 요약, 코드베이스 구조화 등 작업에 쉽게 응용
- 이 스킬을 이용하면 매주 수시간 이상의 작업 시간이 줄어드는 실제 효과 체감
- 영상 끝에서는 “클론해서 바로 써보라”고 권장하는데, 그만큼 범용성 및 실용성이 뛰어남

### 누구나 레포를 클론하고 즉시 적용 가능함

- Excalra diagram skill의 레포(repository)는 일반 사용자도 쉽게 복제(clone) 후 자신의 워크플로우에 통합 가능
- 단순 설치만으로 Excal 다이어그램 생성 AI에 즉시 활용할 수 있음
- 적용 대상(문서, 코드, 아이디어 등)이 매우 다양하며, 사용자가 직접 적용하면서 숙련도를 높일 수 있음

## *Claude에게 24시간 동안 코드 작성을 시키자 완전한 claw.ai 클론이 완성됨* 핵심 요약

- 영상은 Claude Opus 4.5 AI 모델이 24시간 동안 멈추지 않고 코드 작업을 수행하는 과정을 보여줌
- 결과적으로 claw.ai와 거의 동일하게 동작하는 매우 완성도 높은 클론 웹앱이 자동으로 완성됨
- 수많은 테스트 루프를 거치며 대화 생성, 검증, 반복 등 AI의 작업 내역을 과거 이력까지 확인 가능
- 마크다운(Markdown) 형식의 문서화 기능이 매우 깔끔하게 구현됨
- HTML 페이지를 다양하게 자동 생성하며, 코드 작성 및 실행 기능도 내장되어 있음
- 테마 변경, 기본 모델 설정, 최대 토큰 값을 조절 가능하게 하는 슬라이더 등 다양한 설정 기능 구현
- 기능이 매우 풍부하여 실제 사용 가능한 수준에 근접했다고 평가됨
- 아직 "human-in-the-loop" 기능(사람의 검토)이 남아 있지만, 거의 손대지 않고 이 정도 시스템 구축이 가능함에 놀라움을 드러냄
- 애플리케이션의 검증 및 반복 최적화를 위해 Claude를 “하네스(harness)”로 활용할 수 있다는 점을 강조
- 만약 추가적인 하루가 주어진다면 이 앱이 완벽해질 수 있을 정도로 Claude의 자동화와 반복 개발 역량에 큰 감명을 받음

## 세부 요약 - 주제별 정리

### Claude의 자동 코딩으로 claw.ai 클론이 거의 완벽하게 제작되어 놀라움을 안겨줌

- 24시간 동안 Claude Opus 4.5에게 연속적으로 코딩을 맡긴 결과를 소개함
- 제작자는 이미 결과물이 충분히 완성된 claw.ai 클론임을 실감한다고 언급
- 테스트 루프의 일부가 어떤 용도인지 모를 정도로, 기대치를 뛰어넘는 강력한 기능 구현에 감탄함
- “이 정도로 많은 기능이 자동화되어 구현된다는 것이 대단하다”라고 평가

### 과거 대화 기록과 GPT 내부 검증 결과를 손쉽게 확인할 수 있게 됨

- 챗봇 앱에서 지난 대화 내역을 모두 신속하게 확인할 수 있음
- 이력에는 각 반복 루프에서 Claude가 생성하고 검증한 대답들이 저장되어 있음
- 검증 및 반복 개선과정 전체를 추적할 수 있어, 개발 및 디버깅에 도움을 준다고 언급

### 마크다운 및 HTML 자동화로 강력한 문서화와 코드 실행 환경이 구축됨

- 마크다운(Markdown) 포맷을 매우 깔끔하게 자동 생성함
- 다양한 HTML 페이지를 개발자가 개입하지 않고도 Claude가 자체적으로 제작
- 코드 작성 및 실행이 가능한 통합 개발 환경도 자동으로 구현됨

### 사용자 인터페이스 설정 기능까지 완벽하게 구현됨

- 테마(빛/어둠 등) 변경 옵션이 생김
- 기본 AI 모델 설정 기능 도입 (사용자가 선호하는 모델을 선택 가능)
- 최대 토큰(token) 수를 조절할 수 있는 슬라이더도 포함
- 이런 UI/UX 설정 기능이 자동 생성된 점을 특히 강조

### 개발자가 “거의 손대지 않고도” 실사용 가능한 수준의 채팅앱이 완성됨

- 주도적 사람의 작업 개입 없이, Claude가 거의 모든 코드를 스스로 작성
- 일부 인간 검토(human-in-the-loop) 단계만 추가로 필요하다고 언급
- 제작 과정에서 본인은 손을 거의 대지 않았다고 직접적으로 설명
- 추가적인 하루만 Claude에 맡기면 “완전히 완벽한 애플리케이션이 될 것”이라고 말함

### Claude Opus 4.5와 하네스(harness) 시스템의 반복, 검증 기능이 혁신적으로 작동함

- Claude Opus 4.5가 새로운 애플리케이션 기능을 반복적으로 테스트하고 검증함
- 하네스(harness) 시스템 덕분에 앱의 검증 및 지속적 개선이 자동으로 이루어짐
- 반복적 검증 루프를 통해 버그 및 오류를 잡고, 기능 완성도를 끌어올릴 수 있음을 시연

### Claude를 통한 자동화가 앞으로의 소프트웨어 개발 방식에 큰 영향을 줄 것임을 시사

- AI가 며칠 만에 복잡한 앱을 “사람 손대지 않고” 만들 수 있다는 사실에 놀라움을 표함
- Claude와 같은 AI의 개발 능력이 실제 소프트웨어 개발 환경에 어떤 변화를 줄지 기대한다는 뉘앙스 발언
- 반복, 검증, 기능 확장 등 소프트웨어 개발의 전체 과정을 AI에 맡길 수 있음을 실례로 증명

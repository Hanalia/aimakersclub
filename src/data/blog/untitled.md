## *Claude에게 24시간 내내 코딩을 시킨 결과* 핵심 요약

- 영상은 Claude Opus 4.5라는 AI 코딩 모델을 활용해 24시간 동안 자동 코딩 실험을 진행한 결과를 소개함
- 현재까지 구현된 결과물은 claw.ai의 완벽한 복제판 수준이라고 칭하며, 실제 사용 가능한 다양한 기능이 포함됨
- 과거의 모든 대화 기록을 저장하고 검증 과정별로 확인 가능하도록 구현됨
- 마크다운 포맷팅이 깔끔하게 지원되어 사용성 및 가독성이 우수함
- 다양한 HTML 페이지를 생성하고 AI가 직접 코드를 작성해 실행할 수 있음
- 설정 화면에서는 테마 변경, 기본 모델 변경, 토큰 최대치 조절 슬라이더 등 세밀한 커스터마이징이 가능함
- 인간의 개입(human in the loop)을 아직 추가하지 않았으나, 완성도 높은 기능 구현에 놀라움을 표함
- 진행자가 별도의 수작업 없이 Claude만으로 복잡한 어플리케이션을 구축한 것에 대해 깊은 인상을 밝힘
- 하루 더 AI에게 시간을 주면 완벽에 가까운 어플리케이션이 탄생할 수도 있음을 시사함
- Claude Opus 4.5와 같은 하니스(자동 반복 검증 프레임워크)의 결합이 강력한 자동화와 발전 가능성을 보여줌

## 세부 요약 - 주제별 정리

### Claude Opus 4.5는 claw.ai와 동일한 수준의 복제판을 거의 자동으로 완성함

- 24시간 동안 Claude를 사용해 코딩하면서 claw.ai의 주요 기능을 대부분 구현했다고 언급
- 수작업 없이 AI 주도로 진행했으며, 결과물의 완성도에 대해 놀라움을 표현
- 아직 테스트의 후반부가 남았지만 이미 거의 완전한 대체품이라고 평가

### AI가 생성한 이전 대화와 검증 과정을 모두 투명하게 확인할 수 있음

- Claude가 작업을 수행하면서 생성한 과거 대화(conversations)를 모두 기록 및 보관
- 각 반복 루프(loops)별 진행 상황을 사용자가 추적 가능하게설계됨
- 검증(verification) 단계마다 어떤 대화와 결과가 있었는지 쉽게 확인이 가능

### 효율적이고 보기 좋은 마크다운 포맷팅이 높은 사용성을 제공함

- 최종 결과물이 마크다운(markdown) 포맷으로 깔끔하게 정리되고 출력됨
- 코드, 설명, 구조가 명확하게 정리되어 가독성이 뛰어남
- 사용자는 개발 과정과 결과물을 직관적으로 이해할 수 있음

### 다양한 HTML 페이지 생성과 코드 작성 및 실행이 모두 자동화됨

- Claude가 HTML 페이지 여러 종류를 직접 생성하는 기능을 구현
- AI가 스스로 코드를 작성하는 것은 물론, 즉시 그 코드를 실행하여 결과를 검증함
- 개발 자동화 프로세스가 한 단계 진전됐음을 실험적으로 증명

### 상세한 설정 페이지에서 커스터마이징이 자유롭게 가능함

- 테마(Theme) 변경 인터페이스가 있어, 사용자가 시각적 선호에 맞춰 조정 가능
- 기본 모형(default model) 선택 기능을 제공
- 토큰 최대치(max tokens)를 조절할 수 있는 슬라이더 인터페이스도 구현됨

### 아직 인간의 개입이 일부 필요하지만, 완성도 높은 수준임을 강조함

- ‘human in the loop’, 즉 인간의 감시·중재 기능이 추후 보완 예정임을 밝힘
- 그럼에도 불구하고 이미 현존 많은 서비스에 견줄 만한 완성도를 자랑함
- 자동화 코드 생성의 잠재력에 대해 감탄을 표함

### 별도 작업 없이 Claude 하나로 복잡한 프로젝트가 구현될 수 있었음

- 진행자는 “손 하나 까딱하지 않고” 상당한 수준의 어플리케이션 빌드가 가능했다고 밝힘
- AI 코딩툴의 생산성이 실제로 급격히 향상되고 있음을 실증함
- 개발자의 수동 작업 최소화가 현실화되는 과정을 보여줌

### 추가 테스트와 시간이 있다면 완벽한 애플리케이션 구현도 기대할 수 있음

- 하루만 더 Claude에게 작업을 맡길 경우, 더욱 완벽한 앱이 가능할 것이라고 언급
- 아직 완성되지 않은 세부 기능이나 테스트도 향후 반복 과정을 통해 보완될 수 있음

### Claude Opus 4.5의 반복 검증(하니스) 능력이 전례 없는 자동화를 실현함

- Claude가 자체적으로 애플리케이션의 기능을 반복적으로 검증, 개선함
- 이른바 “하니스”(harness) 환경에서 AI가 지속적으로 품질을 높이는 구조
- 애플리케이션 품질과 기능을 끊임없이 증진하는 자동화 모델의 잠재력을 실험적으로 입증

### AI 코딩 파이프라인과 자동 검증 체계의 결합이 미래 소프트웨어 개발의 한 방향을 제시함

- AI 코딩과 자동 반복 검증 인프라가 결합하면, 소프트웨어 개발의 패러다임 변화가 예상됨을 시사
- 반복 검증과 하니스 구조를 통해, 빠르고 정확한 애플리케이션 프로토타입 완성이 가능함
- Claude와 같은 최신 AI 모델을 활용한 개발 방식이 실제 서비스 수준의 복제도 실현 가능함을 실제 구현 사례로 제시

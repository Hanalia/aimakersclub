## *Claude에게 24시간 동안 멈추지 않고 코딩을 시키자 벌어진 일* 핵심 요약

- 이 영상은 생성형 AI인 Claude(Claude Opus 4.5)에게 24시간 동안 쉬지 않고 웹 애플리케이션을 개발하게 한 실험 결과를 다룸
- 실험을 통해 기존 claw.ai와 거의 동일한 기능의 복제 앱(클론 앱)을 자동 생성하는 데 성공했다고 언급
- AI가 생성한 결과물에는 대화 내역, 코드 검증 기록, 마크다운 문서화, HTML 페이지 생성 등 다양한 기능이 포함됨
- 사용자는 테마, 기본 모델, 최대 토큰 수 등의 설정을 직접 조정할 수 있는 인터페이스도 생성됨
- “Human in the loop(사람의 개입)” 요소를 추가하고 싶다고 언급하지만, 이미 AI만으로 상당히 완성도 높은 결과를 얻었다고 평가
- AI가 각 반복(루프)마다 기능을 확인하고 기록에 남기는 등, 자체 검증과 검토 단계가 자동으로 이루어짐
- 실험자는 자신이 직접 코드를 작성하지 않고도 다양한 기능을 가진 앱이 만들어진 점을 “매우 인상적”이라고 표현
- Claude Opus 4.5와 같은 최신 AI 모델과 자동 테스트 환경의 결합으로, 사람이 거의 개입하지 않아도 정교한 소프트웨어가 만들어질 수 있음을 강조
- 추가적인 시간이 주어진다면 더욱 완벽한 애플리케이션 개발이 가능할 것 같다고 전망하며, 기술 발전에 감탄함
- 본 실험을 통해 인공지능을 활용한 소프트웨어 자동화 개발의 새로운 가능성을 보여줌

## 세부 요약 - 주제별 정리

### Claude Opus 4.5는 하루 만에 claw.ai 복제 앱을 거의 완벽하게 자동 생성함

- 실험자는 Claude Opus 4.5에게 24시간 동안 지속적으로 코딩을 시키는 실험을 진행함
- 그 결과, 대화형 AI 서비스 claw.ai의 핵심 기능을 모두 복제한 앱이 완성됨
- 실험자가 각 테스트의 절반이 어떤 용도인지조차 모를 정도로, 이미 완벽에 가까운 결과물이 생성됨
- 인공지능이 단독으로 각 기능을 순서대로 검증하고, 결과를 쌓아가며 반복적으로 개선함

### AI가 생성한 웹 앱은 풍부한 기능(대화 로그, 마크다운, HTML 생성, 실행 등)을 스스로 갖추게 됨

- 앱 내에서 사용자가 기존 대화 기록을 자유롭게 탐색할 수 있는 구조로 설계됨
- 모든 기능 검증 과정을 자동으로 기록하며, 이 과정에서 생성된 대화도 보존됨
- 마크다운(markdown) 포맷으로 결과가 깔끔하게 정리되어 가독성을 높임
- 다양한 HTML 페이지가 자동으로 만들어지며, 코드 작성 및 실행까지 지원함

### 사용자 맞춤 설정(테마, AI 모델, 토큰 제한 등) 인터페이스도 자동으로 구현됨

- 사용자는 설정 패널에서 전체 테마(밝기/어둠 등) 및 기본 적용 AI 모델을 직접 선택 가능
- 슬라이더 기능을 통해 한 번에 처리 가능한 최대 토큰 수(max tokens)도 세밀하게 조정할 수 있음
- 별도의 추가 입력이나 코딩 없이 이런 사용자 맞춤 환경이 제공된 점이 강조됨

### 자동 검증과 반복 개선 루프가 AI 코드 생산 품질을 크게 높임

- Claude는 각 기능을 추가하거나 변경할 때마다 자동으로 결과를 검증하여 기록을 남김
- 내부 루프 구조를 통해 지속적인 테스트와 검토가 반복적으로 이루어짐
- 코드 품질 및 기능 신뢰성을 별도의 사람 개입 없이 기계적으로 확보함

### 실험자는 직접 손을 대지 않고도 “현실적으로 쓸만한” 앱이 완성된 결과에 깊은 인상을 받음

- “아무것도 손대지 않고 이렇게까지 앱을 만들 수 있다는 것이 놀랍다”고 직접 언급
- 개발자가 일일이 손을 보지 않아도 다양한 고급 기능과 유저 편의 장치가 탑재된 점을 중요하게 평가
- 사람 개입 없이 진행된 자동 생성·검증 프로세스의 효율성과 혁신성에 감탄함

### 남은 과제는 향후 ‘휴먼 인 더 루프’(Human in the Loop) 도입 등 추가 고도화임을 인정함

- 현재 결과도 매우 우수하지만, 사람의 감독과 수동 개입 요소가 보완될 경우 완성도가 더욱 높아질 것으로 언급
- 사람의 피드백이나 승인 과정을 삽입하면 실제 서비스 수준으로도 손색없을 것으로 기대함

### Claude와 같은 최신 생성형 AI와 자동화 툴의 결합이 개발 패러다임을 크게 변화시킴

- 최신 AI의 진화와 자동화 하니스(harness) 시스템을 결합하면, 기존 수작업을 거의 대체할 수 있음을 실증함
- 애플리케이션 개발에 소요되는 인력과 비용, 시간이 대폭 감축될 가능성을 시사

### 실험자는 추가 개발 시간만 주어진다면 완벽에 가까운 앱 도출도 가능할 것으로 본다며 기대감을 표명함

- 24시간 만에 이 정도면, 하루나 이틀만 더 주어도 ‘완전한’ 클론 앱이 나올 수도 있다고 전망
- AI의 개발 생산성과 자동화 반복 실험의 효율성에 놀람을 감추지 않음

### 전체적으로, AI 기반 자동화는 소프트웨어 개발 방식에 혁신적인 변화를 가져다 줄 가능성이 크다고 결론지음

- 이번 실험이 AI의 한계와 가능성을 동시에 보여주는 사례임
- “정말 미친 수준이다”라고 마무리하며 인공지능이 만들어내는 혁신에 감탄함

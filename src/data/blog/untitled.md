## *RAG 2.0: 에이전틱 RAG와 지식 그래프의 결합 (무료 템플릿 소개)* 핵심 요약

- 영상 제목: **RAG 2.0: 에이전틱 RAG와 지식 그래프의 결합 (무료 템플릿 소개)**
- 최근 몇 달간 다양한 RAG(검색 기반 생성) 전략을 실험한 끝에 에이전틱 RAG와 지식 그래프가 가장 효율적임을 강조
- 두 방식을 결합한 강력한 AI 에이전트를 직접 개발하여 시연
- 이 에이전트는 질문의 특성에 맞춰 최적의 정보원(벡터 데이터베이스 또는 지식 그래프)을 자동으로 선택하여 답변
- 벡터 데이터베이스는 PostgreSQL+PG Vector 및 서버리스 플랫폼 Neon을 활용하며, 문서 임베딩 및 청킹 구조로 구성
- 동일 문서는 Graffiti와 Neo4j를 통한 지식 그래프에도 저장, 회사 간의 관계성 탐색이 가능
- 사용 예시: 단일 기업의 AI 이니셔티브 질문시 벡터 DB를, 두 기업 간 관계성 질문시 지식 그래프를 자동 활용
- 시스템 프롬프트나 명시적 지시를 통해 두 데이터 소스 모두를 동시에 질의해 더 완성도 높은 답변 생성 가능
- 개발 전체 과정을 담은 본 영상의 전체 버전 및 무료 템플릿/설치 방법은 링크를 통해 제공

## 세부 요약 - 주제별 정리

### 다양한 RAG 전략을 실험한 결과, 에이전틱 RAG와 지식 그래프가 가장 강력함을 확인함

- 최근 몇 달간 다양한 RAG 전략(검색 기반 생성, Retrieval-Augmented Generation)을 심층 실험
- 실험을 거듭한 결과 꾸준히 돌아오게 되는 두 가지 핵심 방식으로 에이전틱 RAG와 지식 그래프를 선정
- 이 두 가지 전략의 장점을 살려 결합할 때 가장 큰 효과가 있음을 발견

### 에이전틱 RAG와 지식 그래프를 결합한 새로운 AI 에이전트를 직접 구축함

- 에이전틱 RAG(Agentic RAG)와 지식 그래프(Knowledge Graph)의 혼합 모델을 구현
- 해당 모델은 질문에 대해 어떤 정보 출처가 최적의 답을 줄지 자체적으로 추론, 유연히 검색 방향을 결정
- 영상 내 시연에서는 이 에이전트가 다양한 방법으로 자동 판단 및 정보를 추출하는 과정을 보여줌

### 벡터 데이터베이스에는 PG Vector와 Neon을 활용하고 문서 임베딩 구조를 적용함

- 벡터 데이터베이스(Database)로 PostgreSQL에 PG Vector 확장 플러그인을 적용, 서버리스 플랫포옴 Neon에서 사용
- 문서는 여러 청크(chunk)로 분할하여 임베딩을 생성, 빠른 벡터 검색 및 추론에 활용
- 임베딩 데이터는 주로 빅테크 기업들(예: Google, Amazon 등)의 AI 이니셔티브 및 상호 파트너십 정보를 포함

### 동일 문서를 Graffiti와 Neo4j 기반 지식 그래프에도 저장하여 관계성 탐색을 용이하게 함

- 동일한 정보를 지식 그래프 시스템(Graffiti, Neo4j)에 저장하여 관계형 탐색 가능
- 지식 그래프에서는 기업들 간의 상호 작용, 협력 관계(예: Amazon이 Anthropic의 모델 호스팅 위치) 등의 관계형 질의가 가능함

### 질문 유형에 따라 에이전트가 벡터 DB와 지식 그래프 중 적절한 도구를 선택하여 답변함

- 예시 1: Google의 AI 이니셔티브 등, 단일 회사 정보 탐색은 벡터 데이터베이스만을 자동 활용
- 예시 2: 두 회사 간의 관계성(Anthropic과 Amazon의 협업 등) 탐색시 지식 그래프 검색을 선택
- 에이전트는 답변과 함께 실제 사용한 툴(벡터 DB 또는 그래프 DB)이 무엇인지 명확히 출력

### 시스템 프롬프트나 명령을 통해 두 데이터 소스 동시 활용도 가능함

- 사용자가 시스템 프롬프트를 바꾸거나 명확히 “두 소스를 다 써라”고 요구할 수 있음
- 이 경우, 에이전트는 벡터 DB와 지식 그래프 양 방면에서 정보를 추출해 결합된 답변을 제공
- 복합적인 질문 및 고품질 답변 생성에 용이함

### 무료 템플릿이 제공되며 전체 개발 과정은 본 영상 링크에서 확인 가능함

- 해당 AI 에이전트의 구현 템플릿이 무료로 제공됨을 명시
- 전체 구축 방식, 시스템 환경, 사용 방법, Cloud Code 등 상세 개발 과정은 전체 영상 링크에서 확인 가능
- 시청자가 직접 따라할 수 있도록 안내

### Cloud Code 활용으로 에이전트 개발 및 설치가 용이해짐

- 본 영상의 전체 버전에서는 Cloud Code 이용법도 구체적으로 다룸
- 개발 과정 최적화, 배포 효율 향상 등 실제 설치 및 운영 팁도 포함

### 다양한 실사례 시연으로 이해도와 신뢰도를 높임

- 실제 에이전트 질의 결과(어떤 데이터베이스가 사용되었는지, 답변 과정) 시각적으로 직접 시연
- 빅테크 기업들의 구체적 사례, 기업 간 협업 정보, AI 이니셔티브 등 실제 데이터를 활용
- 각 단계마다 화면과 설명을 맞춰 보여줘 구체적인 작동 원리와 장점을 시각적으로 전달
